Tranzicije i animacije


Tranzicije (prelazi) omogućavaju da dati element postepeno promeni stil u neki drugi prelaskom kursora miša preko njega. Potrebno je obratiti pažnju na sledeća svojstva:
  • transition-property – određuje na koje CSS svojstvo ćemo primeniti prelaz (inicijalna vrednost je all)
  • transition-duration – određuje trajanje prelaza
  • transition-delay – kašnjenje tj. određuje vremenski početak vršenja tranzicije
  • transition-timing-function - opisuje kako će se vrednosti menjati tj. dopušta promenu brizne tokom trajanja prelaza (ease, linear, ease-in, ease-out, ease-in-out...)

Sledeće linije koda opisuju tranziciju, gde prelaskom kursora miša preko elementa on promeni svoju širinu u trajanju od 4s. div { width: 150px; height: 150px; -webkit-transition: width 4s; /* Safari */ transition: width 4s; } div:hover { width: 400px; }
Napomena: U kodu sa desne strane vidimo kako prvi element menja svoju širinu i dužinu po zadatim parametrima. Naredna četiri elementa prikazuju pomenute timing-function, gde u prvom od njih primećujemo kašnjenje tranzicije za 2s.
Zadatak 1: Izmeniti linije koda gde će pravougaonik, prelaskom kursora preko njega, postati kvadrat u trajanju od 5s i kašnjenjem od 1s.


Animacije


Animacije omogućavaju da dati element promeni stil u neki drugi. Možemo promeniti koliko god svojstava želimo neograničen broj puta.
    Potrebno je obratiti pažnju na sledeće:
  • animation-name – određuje na koje CSS svojstvo ćemo primeniti prelaz (inicijalna vrednost je all)
  • @keyframes pravilo – opisuje putanju promena koje vršimo na elementu
  • animation-duration - trajanje animacije; ako ga ne navedemo animacija se neće izvršiti, jer je inicijalna vrednost 0s
  • animation-iteration-count - određuje koliko puta želimo da se animacija ponovi; vrednost infinitive će animaciju izvršavati beskonačan broj

Naredne linije koda prikazuju animaciju u kojoj element zelene boje prelazi u plavu u trajanju od 3s. @keyframes primer { from {background-color: green;} to {background-color: blue;} } div { width: 200px; height: 200px; animation-name: primer; animation-duration: 3s; } <br> Možemo definisati @keyframes koji ima više promena/prelaza uz pomoć procenta na sledeći način: <xmp class = "primer_ta" rows=10 disabled> @keyframes example { 0% {background-color: blue;} 25% {background-color: yellow;} 50% {background-color: red;} 100% {background-color: green;} } Svojstvo animation-delay opisuje kašnjenje, odnosno kada će prelaz početi da se izvršava.
Napomena: U kodu sa desne strane u trećem primeru je korišćena funkcija animation-iteration-count sa vrednošću beskonačno (infinitive), pa se animacija stalno ponavlja, dok se drugi primer izvršava samo jednom. Primetiti da se nakon izvršenja animacije, objekat vraća na početno stanje.
Zadatak 2:
Izmeniti kod u drugom primeru tako da menja pozicije kada menja i boju, tako da:
Kada je boja plava, pozicija 0px,0px.
Kada je boja zuta, pozicija 100px,100px.
Kada je boja crvena, pozicija 400px,0px.
Kada je boja zelena, pozicija 0px,0px.


Animacije imaju još nekoliko zanimljivih svojstava:
  • animation-direction - opisuje kako će se animacija kretati
    • reverse - animacija se izvršava u obrnutom smeru
    • alternative - animacija se izvršava u ciklusima
    • Napomena: U kodu sa desne strane prva dva elementa opisuju dve animacije od kojih se jedna odvija u smeru sleva na desno,a druga u obrnutom smeru.
  • animation-timing-function - opisuje kako će se vrednosti elementa menjati tj. dopušta promenu brizne promene tokom trajanja prelaza
    • easy - ovo je inicijalna vrednost, animacija kreće sporo, pa brzo, zatim ponovo usporava
    • linear - brzina animacije je konstantna
    • easy-in - animacija ima spor početak
    • easy-out - animacija ima spor kraj
    • easy-in-out - animacija ima spor i početak i kraj
    • Napomena: U kodu sa desne strane su prikazani elementi koji opisuju ove funkcije (različite brzine izvršenja animacija).

Pogledajmo kako bi izgledao kod gde bismo upotrebili sva svojstva koja opisuju jednu animaciju: div { animation-name: primer; animation-duration: 4s; animation-timing-function: easy; animation-delay: 1s; animation-iteration-count: infinite; animation-direction: alternate; } Postoji skraćeni oblik zapisa ovog koda koji obuhvata sva pomenuta svojstva animacije: div { animation: primer 4s easy 1s infinite alternate; }
Zadatak 3:
Na kraju ovog poglavlja pokušajte da uradite zadatak u kome se istovremeno javljaju animacije, tranzicije i transformacije:
Napisati kod u kome pravougaonik menja boje i kreće se beskonačno mnogo puta po određenoj putanji (desno, dole levo, dole desno, gore levo). Kada prisloniš kursor miša na element, on postaje kvadrat.
