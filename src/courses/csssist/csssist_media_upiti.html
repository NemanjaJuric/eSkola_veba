@media upiti

@media pravilo se koristi za definisanje stilova za različite tipove uređaja kao što su tableti, mobilni telefoni, laptopovi, desktop računari i medija kao što je štampanje. Tip medija, dakle, može biti screen i print, a vrednosti @media upita mogu biti: screen, print, projection, handheld i all, a mogu se koristiti i više vrednosti odjednom odvojene zarezom.
Primer prikaza screen medija:
@media screen { body { font: 12px arial, sans-serif } section { display: block; } }

Primer prikaza print medija:
@media print { body { font-size: 10pt; font-family: times, serif; } section { display: none; } }

Ukoliko uzmemo naš originalni raspored koji je u dve ili više kolona i probamo da ga umanjimo na nivo mobilnog telefona, videćemo da je to loše vidljivo posmatraču. Pametni telefoni se tipično koriste u portret režimu, što znači da je ekran duži nego što je širi. Na ovaj način možemo vertikalno skrolovati neki sajt, ali ovaj pristup nije dobar za one sajtove koji imaju širi prikaz sa nekoliko kolona. Tu onda koristimo media upite koji su CSS tehnologija i dostupni su u Web pregledačima već nekoliko godina i suštinska su komponenta prilagodljivog dizajna. Media upiti omogućavaju to da se CSS može primeniti samo ako se ispune specifični uslovi.
Na primer, možemo napisati media upit koji će CSS primeniti tek kada Web pregledač dostigne specifičnu širinu. Ovim putem media upit detektuje širinu strane i na osnovu toga može da reorganizuje način prikaza te strane.

Specifikacija CSS-a u odnosu na veličinu veb pregledača

Sledeći primer definiše do koje širine ekrana će se primenjivati deklarisani CSS stil:
@media screen and (max-width: 1000px) { #content { width: 100% } } @media screen and (max-width: 800px) { div { float: none } } @media screen and (max-width: 600px) { #content { float: none; display: block; } }
Napomena: U primeru vidimo da ako je širina ekrana manja od 600px primenjuju se sve tri deklaracije.
Možete isto koristiti "min-width" umesto "max-width" koja definiše od koje širine ekrana će se primenjivati deklarisani CSS stil.

Specifikacija CSS-a u odnosu na orijentaciju

Ako želite da primenite CSS u zavisnosti od orijentacije veb pregledača, koristite neki od sledećih vrednosti:
@media screen and (orientation: landscape) { div { float: left } } @media screen and (orientation: portrait) { div { float: none } }
Napomena: Specifikacija CSS-a u odnosu na orijentaciju može biti najkorisnija za mobilne uređaje.

Specifikacija CSS-a u odnosu na uređaj koji koristite

Širina i visina (width, height)

device-width, device-height, min-device-width, max-device-width, min-device-height i max-device-height mogu se koristiti za računanje rezolucije ekrana uređaja:
@media screen and (min-device-height: 768px) and (max-device-width: 1024px) { /* Deklaracija CSS-a za uređaje čija je minimalna visina ekrana 768px i maksimalna širina 1024px */ }

Odnos piksela (pixel ratio)

Bitno je, takođe, imati na umu da piksel u CSS-u ne mora biti isti kao fizički piksel. Na primer, može se desiti da je ekran fizički 720px širok, a da veb pregledač može prikazati preko stilova da ima širinu 480px. U ovom primeru odnos piksela je 1.5:1 (na 1.5 fizičkih piksela prikazuje 1 CSS piksel). Desktop monitor ima odnos piksela 1:1.
Ako želite da prikažete stilove samo na uređajima sa odnosom piksela 2:1 (takvi su, na primer, iPhone 4 i niže verzije), možete koristiti na sledeći način:
@media (device-pixel-ratio: 2) { body { background-color: #ececec; } }
Napomena: Možete isto koristiti min-device-pixel-ratio i max-device-pixel-ratio.
