CSS selektori, boje i osobine teksta


Selektori

Razlika između klasnih i ID selektora je u znaku koji im prethodi, naime ukoliko je znak # (taraba) u pitanju je ID selektor, u suprotnom .(tačka) označava klasni selektor.
#top{ background-color: #ccc; padding: 20px } .intro { color: red; font-weight: bold; }
Ako želimo da primenimo navedeni CSS na HTML stranicu, koristimo atribute id i class. To izgleda ovako:
<div id="top"> <h1>Zaokruživanje decimalnih brojeva</h1> <p class="intro">Kada se decimalni broj zaokrugljuje, gleda se prethodna cifra. Ako je prethodna cifra veca od 5, zaokrugljuje se na veći broj, ako je manja, zaokrugljuje se na manji.</p> <p class="intro">Kada je prethodna cifra 5, gleda se da li postoji jos neka cifra pre nje. Ako postoji, broj se zaokrugljuje na veći, ako ne, broj se zaokrugljuje na manji. </p> </div>
Jos jedna razlika između klasnih i ID selektora je što se ID koristi za prepoznavanje samo jednog elementa, dok klasni za više. Možete primeniti selektor na određeni HTML element jednostavno navodeći HTML selektor prvi, tako da npr. p.naziv_klase { ... } će biti primenjen samo na elemente paragrafa koji sadrže klasu " naziv_klase" .

Boje teksta i pozadine

CSS donosi 16 777 216 boja. One mogu biti u formi:
  • imena - CSS prepoznaje 147 definisanih naziva boja. Ove boje se zadaju jednostavnim navodjenjem naziva boje.
  • RGB (red/green/blue) vrednosti - zadaju se tri cela broja, od 0 do 255, koji predstavljaju tri kolorne komponente - crvenu, zelenu i plavu, čime se mogu postići sve nijanse.
  • heksadekadnog zapisa - boju zadajemo u obliku #RRGGBB. Heksadekadni kodovi se zadaju sa vodećom "tarabom".
Sve ove vrednosti, za zadavanje crvene boje, imaju isti rezultat:
  • red
  • rgb(255,0,0)
  • #ff0000
Imena nekih boja koje možemo da koristimo: aqua, black, blue, blanchedalmond, blueviolet, chocolate, coral, darkblue, grey, green, deeppink, gold, maroon...

CSS pravilo za podešavanje boje teksta zadaje se u obliku deklaracije koja ima sledeći oblik:
color:red
Ovaj atribut ima samo tu jednu svrhu, odnosno može se koristiti isključivo za postavljanje boje slova unutar nekog HTML dokumenta.

Boje se mogu primeniti korišćenjem color i background-color (boja pozadine) na sledeći način:

Žuta pozadina i plavi tekst bi izgledali ovako:
h1{ color: blue; background-color: yellow; }

Možemo primeniti color i background-color svojstva na većinu HTML elemenata, uključujući i body, što će promeniti boju stranice i svega na njoj.
To ćemo uraditi na sledeći način:

body{ color: green; }
Zadatak 1: Napisati rečenicu čiji će tekst biti crvene boje,a pozadina žute.


Osobine teksta

Atributi fonta

Najosnovniji atributi koji određuju izgled slova su: font-family, font-size, font-weight, font-style, font-variant, font.
  • font-family - Izbor fonta za ispis teksta.
  • font-size - Veličina slova.
  • font-weight - Podebljani ispis slova.
  • font-style - Zakošen ispis slova.
  • font - Atribut za postavljanje više osobina odjednom.

font-family

Ovaj atribut odredjuje izgled slova kojim se ispisuje tekst. Moguće je navesti bilo koji font koji postoji na računaru, a neke od mogućnosti su:
  • serif - Font sa postoljima, podrazumevani je Times new roman.
  • sans-serif - Font bez postolja, podrazumevani je Arial.
  • monospace - Slova fiksne širine, podrazumevani je Courier New.
  • cursive - Kurzivni (iskošeni) font.
Ako želimo da zadamo sam naziv fonta koji u sebi ima razmak, pišemo ga unutar navodnika.
.primer1{ font-family: "Times New Roman", serif; }

font-size

Ovaj atribut određuje veličinu slova. Zadaje se kao:
  • numerička vrednost - Postavlja veličinu slova na određenu veličinu u pikselima.
  • procentualna vrednost - Postavlja veličinu slova na procenat u odnosu na matične elemente.
  • jedna od sledećih mogućnosti - xx-small, x-small, small, medium, large, x-large, xx-large.
.primer2{ font-size: 14px; font-size: 40%; }

font-weight

Ovaj atribut odredjuje debljinu slova kojim se tekst ispisuje. Unapred definisane vrednosti su:
  • normal - Obična slova.
  • bold - Podebljana slova.
  • bolder - Ukoliko je moguće, jače podebljana slova.
  • lighter - Slova tanja od normalnih.
.primer3{ font-weight: bold; }

font-style

Atribut koji odredjuje da li su slova iskošena.
  • normal - Obična slova.
  • italic - Iskošena slova.
.primer4{ font-style: italic; }

font

Atributom font možemo istovremeno da postavimo više podešavanja za font slova. Redosled zadatih vrednosti jeste bitan.
  • Prvo se zadaju vrednosti za font-style i za font-weight.
  • Zatim se zadaje font-size.
  • I uvek se na kraju zadaje font-family.
.primer5{ font: bold, 14pt, "Times New Roman",serif; }
Primer
Sada ćemo na dva načina zadati potpuno iste parametre:
Prvo ćemo ih zadati pomoću pojedinačnih atributa:
.primer6{ font-family: "Times New Roman",serif; font-size: 11pt; font-weight: bold; font-style: italic; }
A zatim ćemo iste parametre zadati preko kompleksnog atributa font.
.primer7{ font: italic, bold, 11pt, "Times New Roman",serif; }
U ovom primeru slova su ispisana "Times New Roman" fontom,iskošena su,boldirana i njihova veličina je 11pt.

Atributi teksta

Atributi teksta obuhvataju podešavanja vezana za font i pasus i to su:
  • text-decoration - Vrsta podvlačenja teksta.
  • text-transform - Transformacija u velika ili mala slova.
  • text-align - Poravnanje teksta.

text-decoration

Načini podvlačenja teksta su:
  • none - Slova bez linije.
  • underline - Linija se nalazi ispod teksta.
  • overline - Linija se nalazi iznad teksta.
  • line-through - Linija se iscrtava preko testa.
.primer8{ text-decoration: underline; }

text-transform

  • none - Slova bez promene.
  • capitalize - Svako prvo slovo reči postaje veliko.
  • uppercase - Sva slova pretvara u velika.
  • lowercase - Sva slova pretvara u mala.
.primer9{ text-transform: lowercase; }

text-align

Zadaje se za blok unutar kog se nalazi tekst. Poravnanje se vrši horizontalno prema ivicama bloka.
  • right - Tekst poravnat po desnoj strani.
  • left - Tekst poravnat po levoj strani.
  • center - Tekst je centriran.
  • justify - Tekst poravnat po obe ivice bloka.
.primer10{ text-align: center; }

Zadatak 2: Napisati pasus čiji će tekst biti centriran, veličina slova 12px, a slova da budu iskošena i boldirana.
