Grupisanje, ugnježdavanje, pseudo klase, skraćeni zapis osobina i slike u pozadini


Postoje dva načina za uprošćavanje koda koji sadrži i html i css i korisćenje istih.

Grupisanje


Možemo dati iste osobine velikom broju selektora bez potrebe za ponavljanjem.
h2 { color: red; } .thisOtherClass { color: red; } .yetAnotherClass { color: red; }
Možemo jednostavno radvojiti selektore zarezom (,) u jednoj liniji i dodeliti iste osobine za sve.
h2, .thisOtherClass, .yetAnotherClass { color: red; }

Ugnježdavanje


Ako je CSS dobro postavljen, neće biti potrebe za korišćenjem mnogo klasnih i ID selektora, jer se osobine (novih) selektora mogu navesti kroz postojeće selektore.
#top { background-color: #ccc; padding: 1em } #top h1 { color: #ff0; } #top p { color: red; font-weight: bold; }
Ovim se uklanja potreba za korišćenjem klasnih i ID selektora na p, h1 etiketama. Primenjeno na HTML to izgleda ovako:
<div id="top"> <h1>Čokoladni kolač</h1> <p>Ovo je recept za pravljenje kolača!</p> <p>Prijatno!</p> </div>
Ovo ugnježdavanje može da bude komplikovano ukoliko postoji više od dva nivoa, pa bi valjalo provežbati!

Pseudo klase


Pseudo klase su usko vezane za selektore i određuju stanje ili vezu prema selektoru. Imaju sledeću formu: selektor:pseudo_klasa{osobina:vrednost}

Linkovi


Linkovi, bilo posećeni (viseted) ili neposećeni link su najjednostavnije pseudo klase. Naime, menjaće se boje svih linkova na stranice u zavisnosti da li je korisnik posetio tu stranicu ili ne.

a:link { color: blue; } a:visited { color: purple; }

Dinamičke pseudo klase


Najčešće korišćene za linkove, dinamičke pseudo klase mogu se koristiti za primenu stilova u navedenim situacijama:
  1. active kada je nešto aktivirano od strane korisnika (recimo kada se klikne na link)
  2. hover kada još uvek nije neka akcija preduzeta, već nam se pruža kao mogućnost (recimo prelazak preko linka)
  3. focus kada je nešto u fokusu, kada se selektuje ili se sprema za unos sa tastature

a:active { color: red; } a:hover { text-decoration: none; color: blue; background-color: yellow; } input:focus, textarea:focus { background: #eee; }
focus nije podržan od strane starijih pretraživača (browsers), stoga pažljivo!

First children


Konačno, tu je i pseudo klasa first child (prvo dete). Ovo se odnosi samo na prve potomke elemenata (glave, tela...)
<body> <p>I am the body&#8217;s first paragraph child. I rule. Obey me.</p> <p>I resent you.</p> ...
...ako želimo da stilizujemo sledeći paragraf, to radimo uz pomoć CSS koda:
p:first-child { font-weight: bold; font-size: 40px; }

Margine i uokvirenja


margin omogućava da se spoje margin-top, margin-right, margin-bottom, i margin-left u sledećoj formi osobina: top right bottom left;
p { margin-top: 1px; margin-right: 5px; margin-bottom: 10px; margin-left: 20px; }
Možemo zapisati:
p { margin: 1px 5px 10px 20px; }
padding funkcioniše po istom principu.
Ako se navedu samo dve vrednosti, prva će se odnositi na top i botton, a druga na left i right. Naime, na primeru to izgleda ovako : padding: 1em 10em;

Granice


border-width se koristi na isti način kao i prethodne dve osobine. Ukoliko želimo da spojimo border-width, border-color, i border-style u jednu celinu, to ćemo moći na sledeći način.
p { border-width: 1px; border-color: red; border-style: solid; }
Jednostavnije, to izgleda ovako:
p { border: 1px red solid; }
border-top i border-right takođe imaju skraćeni zapis.

Stilovi


Svojstva vezana za stilove se mogu spojiti pomoću jednog svojstva font. Kombinacija je sledeća:[style] [variant] [weight] [size]/[line-height] [family]
p { font: italic bold 12px/2 courier; }
Sve zajedno:
p { font: 14px/1.5 "Times New Roman", times, serif; padding: 30px 10px; border: 1px black solid; border-width: 1px 5px 5px 1px; border-color: red green blue yellow; margin: 10px 50px; }

Slike u pozadini


CSS svojstvo background-image omogućava postavljanje slike na pozadinu bilo kog bloka u CSS-u.
Ovo svojstvo proširuje mogućnosti dizajna, izuzetno je korisno i daje karakterističan izgled stranici.
Vrednost ovog svojstva počinje slovima URL koja su praćena putanjom do željene slike. Putanja je zapisana u zagradi pod navodnicima.
background-image:url("slike/proleće/cveće.jpg");
Ovo svojstvo ima prednost u odnosu na svojstvo background-color (svojstvo koje određuje boju elementu), ukoliko se koriste istovremeno.
Napomena: Sliku koja se koristi kao pozadina treba sačuvati u dimenzijma u kojima želite da se pojavi.
Sliku u pozadini ne treba koristiti za prenos važnih informacija.
Često se slike učitaju nakon ostalih elemenata pa mogu produžiti vreme učitavanja.
Neki stariji pregledači mogu imati problema sa prikazom slike u pozadini tabele.

Ukoliko su dimenzije slike manje od dimenzija bloka slika se ponavlja dok ne popuni ceo blok. Ukoliko to ne želite, koristite svojstvo background-repeat, koje ima četiri vrednosti:
repeat (podrazumevano, ponavljanje slike dok ne popuni blok)
repeat-x (ponavljanje slike horizontalno)
repeat-y (ponavljanje slike vertikalno)
no-repeat (bez ponavljanja)

Poziciju slike u pozadini je moguće menjati korišćenjem svojstva background-position. Vrednosti ovog svojstva su sledeće:
x% y% (procenti duž x i y ose)
x y (apsolutne dužine u pikselima duž x i y ose)
left (prikazuje sliku levo)
right (prikazuje sliku desno)
center (prikazuje sliku u centru)
top (prikazuje sliku na vrhu bloka)
bottom (prikazuje sliku na dnu bloka)

Napomena: Moguće su i kombinacije, top left, top center, top right, bottom left, bottom center, bottom right.

Za fiksiranje slike u pozadini koristi se svojstvo background-attachment, sa sledećim vrednostima:
fixed (prilikom skrolovanja slika se ne pomera)
scroll (prilikom skrolovanja slika se pomera)
Moguće je koristiti više svojstava odjednom, redosled nije bitan, ako se izostavi neka vrednost primeniće se podrazumevana. Na taj način skraćujemo zapis, pa na primer, umesto navođenja svih svojstava:
background–color:purple; background–image:url("slike/proleće/cveće"); background–repeat:repeat-x; background–attachment:fixed; background–position:bottom;
dovoljno je napisati:
background: purple url("slike/proleće/cveće") repeat-x fixed bottom;

Zadatak 1: U primeru je na različite načine prikazana slika. Isprobaj i ti, malim izmenama u kodu, da prikažeš sliku u pozadini primenom preostalih svojstava.
