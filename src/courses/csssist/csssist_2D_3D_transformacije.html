2D i 3D transformacije


Transformacija je efekat koji utiče na element tako što menja njegov oblik, veličinu ili poziciju. CSS3 transformacije omogućavaju pomeranje, okretanje, iskošenje i rastezanje elemenata. Postoje 2D i 3D transformacije. 2D se više koriste u praksi, pa ćemo se njima više baviti.
CSS model opisuje koordinatni sistem u kojem je svaki element pozicioniran. Pozicije i veličine elemenata izražavaju se u pikselima počevši od gornjeg levog ugla.

Translacija


Translacija je pomeranje elementa u odnosu na početnu poziciju. Razlikujemo sledeće metode:
  • translateX(n) - pomeranje elementa duž X-ose za n piksela
  • translateY(n) - pomeranje elementa duž Y-ose za n piksela
  • translate(n,m) - pomeranje elementa duž X-ose za n piksela i m piksela duž Y-ose

div { -ms-transform: translate(20px); /* IE 9 */ -webkit-transform: translate(20px); /* Safari */ transform: translate(20px); }

Rotacija


Rotacija podrazumeva okretanje (rotiranje) elementa za dati ugao. Koristimo metod:
rotate(angle) - pozitivne vrednosti parametra rotiraju u smeru kazaljke na satu, a negativne obrnuto.
div { -ms-transform: rotate(20deg); /* IE 9 */ -webkit-transform: rotate(20deg); /* Safari */ transform: rotate(20deg); }


Zadatak 1:
a) Izmeniti deo koda tako da se pravougaonik pomeri za 10px duž X-ose i 50px duž Y-ose.
b) Izmeniti deo koda tako da se pravougaonik rotira za 100 stepeni u smeru suprotnom kazaljke na satu.


3D transformacije


Za razliku od 2D transformacija sada se nalazimo u trodimenzionom koordinatnom sistemu.
Metode su slične kao i u 2D:
    Translacija:
  • translate3d(x,y,z) - 3D translacija
  • translateX(x) - 3D translacija po X-osi
  • translateY(y) - 3D translacija po Y-osi
  • translateZ(z) - 3D translacija po Z-osi
    Rotacija:
  • rotate3d(angleX,angleY,angleZ) - 3D rotacija
  • rotateX(angle) - 3D rotacija oko X-ose
  • rotateY(angle) - 3D rotacija oko Y-ose
  • rotateZ(angle) - 3D rotacija oko Z-ose
div { -webkit-transform: rotateZ(60deg); /* Safari */ transform: rotateZ(60deg); }
    Skaliranje:
  • scale3d(x,y,z) - 3D skaliranje
  • scaleX(x) - 3D skaliranje po X-osi
  • scaleY(y) - 3D skaliranje po Y-osi
  • scaleZ(z) - 3D skaliranje po Z-osi
    Kombinacija 3D metoda:
  • matrix3d (n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n) - definiše 3D transformaciju pomoću 16 parametara


Zadatak 2:
a) Izmeniti deo koda tako da se pravougaonik rotira oko Y-ose za 250 stepeni u trodimenzionom koordinatnom sistemu.
b) Izmeniti deo koda tako da se pravougaonik rotira oko Z-ose za 120 stepeni u trodimenzionom koordinatnom sistemu.


Razlike između 2D i 3D rotiranja


Kako smo već napomenuli 2D rotiranje se odvija u dvodimenzionom koordinatnom sistemu, dok 3D u trodimenzionom.
U kodu sa desne strane je prikazana razlika između ova dva rotiranja i to uz pomoć animacija o kojima će biti reči u narednim lekcijama.
Rotacije će se izvršiti kada pređemo kursorem miša preko elemenata.
