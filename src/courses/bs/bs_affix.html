Affix

Affix je plagin (eng.plugin) koji se koristi kada je potrebno fiksirati određeni element na veb strani. Fiksiranje se najčešće primenjuje nad menijem ili nad trakom za navigaciju i obezbeđuje im konstantan prikaz na veb strani bez obzira na skrolovanje.

Element se može fiksirati na dva načina:
  • upotrebom data-spy="affix" atributa
  • upotrebom jQuery biblioteke .


Upotreba data-spy="affix" atributa

Jedan od načina da se fiksira element je upotrebom
data-spy="affix" atributa na nivou tog elementa. Pored toga, poželjno je navođenje opcionih atributi: data-offset-top i
data-offset-bottom.

Vrednost atributa data-offset-top predstavlja broj piksela koji je potrebno skrolovati od početka strane da bi se dati element fiksirao na određenoj poziciji. Slično, vrednost atibuta
data-offset-bottom predstavlja broj piksela koji je potrebno skrolovati od kraja strane da bi se dati element fiksirao. Pre nego što bude objašnjeno kako se zadaje pozicija na kojoj će se fiksirati, razmotrimo kako affix plagin funkcioniše u pozadini.

U pozadini, affix plagin radi tako što smenjuje sledeće Bootstrap-ove klase:
  • .affix
  • .affix-top i
  • .affix-bottom .
Ove klase se automatski smenjuju nad fiksnim elementom, u zavisnosti od njegove pozicije tokom skrolovanja.
Ako je naveden atribut data-offset-top, elementu se prvo pridružuje klasa .affix-top, a kada skrolovanjem dostigne vrednost zadatu tim atributom, pridružuje mu se klasa .affix.
Sa druge strane, ako je na nivou elementa zadat atribut
data-offset-bottom, pridružuje mu se prvo klasa .affix. U trenutku kada dostigne vrednost zadatu atributom, pridružuje mu se .affix-bottom.

Klasu .affix odlikuje CSS svojstvo position:fixed; dok preostale dve nemaju te vrednosti. Kao i svaku drugu Bootstrapovu klasu, klasu .affix je moguće nadograditi drugim svojstvima. Ono što je obavezno je zadavanje CSS svojstva top, kao u primeru: Ovo svojstvo, kao i podrazumevano svojstvo position:fixed, će se primeniti na elementu onog trenutka kada nastupi klasa .affix , odnosno kada atributi data-offset-top|bottom dostignu zadate vrednosti.

Sada smo u mogućnosti da odgovorimo na pitanje: "Kako se zadaje pozicija fiksnog elementa, nakon što atributi data-offset-top|bottom dostignu zadate vrednosti ?"
Odgovor je: CSS svojstvo top klase .affix određuje poziciju fiksnog elementa i predstavlja njegovu udaljenost od vrha strane.

Upotreba jQuery biblioteke

Drugi način da se element fiksira je da se korišćenjem jQuery biblioteke pozove metod affix() nad istim.
Pozicija na kojoj se element fiksira je isto određena CSS svojstvom top klase .affix.
Broj piksela koji je potrebno skrolovati od početka, odnosno od kraja strane, da bi se element fiksirao, zadaje se kao parametar metoda opcijom offset.

Opcija offset

U slučaju da je element fiksiran pomoću data-spy="affix" atributa, opcija offset se koristi preko atributa data-offset-top i data-offset-bottom.

Sa druge strane, u slučaju da se element fiksira pomoću jQuery biblioteke, ova opcija će se koristiti kao parametar affix() metoda. Ekvivalent atributu data-offset-top="100", ovde će biti naredba:
.affix({offset: {top: 100} });.
Slično, ekvivalent atributu data-offset-bottom="100" biće:
.affix({offset: {bottom: 100} });.
