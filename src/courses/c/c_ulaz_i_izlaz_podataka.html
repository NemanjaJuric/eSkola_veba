Ulaz i izlaz podataka

  • U C-u ne postoje posebne naredbe za ulaz/izlaz podataka,
  • postoji skup bibliotetskih funkcija za učitavanje podataka u program i za upis rezultata na standardni izlaz ili u datoteku. Funkcije scanf i printf su manje-više standardizovane i postoje (sa istim imenom i načinom funkcionisanja) u svim C prevodiocima,
  • deklaracije funkcija za upravljanje ulaznim/izlaznim resursima u C-u nalaze se u fajlu stdio.h. Pre korišćenja funkcija za ulaz/izlaz podataka u fajl sa izvornim kodom treba uključiti fajl stdio.h. Za uključivanje jednog fajla u drugi koristi se pretprocesorska direktiva include. Format include direktive: #include <ime_fajla> ili #include “ime_fajla” Konkretno, kad god se u programu koriste funkcije scanf ili printf, treba navesti: #include <stdio.h>

Izlaz

Jedna od osnovnih izlaznih funkcija u C-u. Funkcija ispisuje na standardnom izlazu (ekranu, displeju) formatiran tekstualni niz definisan od strane korisnika. Na primer:

Primer 1: C izlaz #include <stdio.h> int main() { /* prikazi tekstulni niz unutar navodnika */ printf("C programiranje"); return 0; }

Navedeni program na standardan izlaz ispisuje:

C programiranje

Kako funkcioniše navedeni program?

  • Svaki validan C program mora da sadrži funkciju main (od engleskog main-glavna, glavno) i izrvršavanje programa uvek počinje od te funkcije. Prazne zagrade iza njenog imena ukazuju na to da se eventualni argumenti ove funkcije ne koriste. Reč int pre imena funkcije označava da ova funkcija, kao rezultat, vraća celobrojnu vrednost (engl. integer), tj. vrednost tipa int. Naredbe funkcije main su grupisane između simbola { i }(koji označavaju početak i kraj tela funkcije). Obe naredbe funkcije završavaju se simbolom ;. Programi obično imaju više funckija i funkcija main poziva te druge funkcije za obavljanje raznovrsnih podzadataka.
  • Prva naredba u navedenoj funkciji main je poziv standardne bibliotečke funkcije printf koja ispisuje tekst na takozvani standardni izlaz (obično ekran). Tekst koji treba ispisati se zadaje između para znakova ", koji se ne ispisuju. U ovom slučaju, tekst koji se ispisuje je C programiranje.
  • Da bi C prevodilac umeo da generiše kod poziva funkcije printf,potrebno je da zna tip njene povratne vrednosti i tipove njenih argumenata. Ovi podaci o funkciji printf, njen svojevrsni opis, takozvana deklaracija, nalaze se u datoteci zaglavlja stdio.h(ova datoteka čini deo standardne biblioteke zadužen za ulazno-izlaznu komunikaciju i deo je instalacije prevodioca za C). Prva linija programa, kojom se prevodiocu saopštava da je neophodno da uključi sadržaj datoteke stdio.h je pretprocesorska direktiva. To nije naredba C jezika, već instrukcija C pretprocesoru koji predstavlja nultu fazu kompilacije i koji pre kompilacije program priprema tako što umesto prve linije upisuje celokupan sadržaj datoteke stdio.h.
  • Naredba return 0; prekida izvršavanje funkcije main i, kao njen rezultat, vraća vrednost 0. Vrednost funkcije vraća se u okruženje iz kojeg je ona pozvana, u ovom slučaju u okruženje iz kojeg je pozvan sâm program. Uobičajena konvencija je da se iz funkcije main vraća vrednost 0 da ukaže na to da je izvršavanje proteklo bez problema, a ne-nula vrednost da ukaže na problem ili grešku koja se javila tokom izvršavanja programa.
  • Tekst naveden između simbola /* i */ predstavlja komentar. Komentari su korisni samo programerima, doprinose čitljivosti i razumljivosti samog programa. Njih C prevodilac ignoriše i oni ne utiču na izvršivu verziju programa.

Primer 2: Int izlaz #include <stdio.h> int main() { int ceoBroj=5; printf("Ceo broj= %d\n",ceoBroj); return 0; } Navedeni program na standardnom izlazu ispisuje:
Ceo broj=5
  • Prva linija funkcije main je deklarizacija promenljive ceoBroj sa inicijalizacijom ceoBroj=5 njom se promenljivoj ceoBroj dodeljuje inicijalna vrednost 5.
  • Korišćenjem funkcije printf, pored fiksnog teksta, može se ispisati i vrednost neke promenljive ili izraza. U formatu ispisa, na mestu u tekstu gde treba ispisati vrednost izraza zapisuje se format tog ispisa — u ovom slučaju %d, jer će biti ispisana celobrojna vrednost. Nakon niske koja opisuje format, navodi se izraz koji treba ispisati. Dakle, koristimo format nisku %d za ispisivanje int tipa. U ovom primeru, format niska %d unutar navodnika biće zameljena vrednošću promenljive ceoBroj.
  • Znakovi \n se ne ispisuju, nego obezbeđuju prelazak u novi red.

Primer 3: float i double izlaz #include <stdio.h> int main() { float broj1=13.5; double broj2=12.4; printf("broj1=%f\n",broj1); printf("broj2=%lf\n",broj2); return 0; }

Navedeni program na standardnom izlazu ispisuje:

broj1=13.500000 broj2=12.400000

Osnovna razlika u odnosu na prethodne programe je što se koriste razlomljeni brojevi, tj. brojevi u pokretnom zarezu. Dakle, u ovom programu, umesto promenljivih tipa int koriste se promenljive tipa double i float dok se prilikom učitavanja i ispisa ovakvih vrednost za format koriste niska %lf i niska %f umesto niske %i.


Primer 4: Char izlaz #include <stdio.h> int main() { char k='a'; printf("karakter = %c\n", k); return 0; }

Navedeni program na standardnom izlazu ispisuje:

karakter = a

U ovom programu koristi se promenljiva tipa char dok se prilikom učitavanja i ispisa ovakvih vrednost za format koriste niska %c.


Ulaz

U C programskom jeziku, scanf() je najčešće korišćena funkcija za unos podataka od strane korisnika. Funkcija scanf() čita ulaz sa standardnog ulaza, poput tastature. #include <stdio.h> int main() { int ceoBroj; printf("Unesi jedan ceo broj: "); scanf("%d", &ceoBroj); printf("Ceo broj = %d\n", ceoBroj); return 0; }

Navedeni program na standardnom izlazu ispisuje:

Unesi jedan ceo broj: 7 Ceo broj = 7 Kako funkcioniše navedeni program?
  • Prva linija funkcije main je deklaracija promenljive. Ovom deklaracijom se uvodi promenljiva ceoBroj celobrojnog tipa - tipa int.
  • Naredna naredba (poziv funkcije printf) na standardnom izlazu ispisuje tekst Unesi jedan ceo broj: , a naredba nakon nje (pozov funkcije scanf) omogućava učitavanje vrednosti neke promenljive sa standardnog ulaza (obično tastature).
  • U okviru poziva funkcije scanf zadaje se format u kojem će podatak biti sačuvan - u ovom slučaju treba pročitati podatak tipa int i format se u tom slučaju zapisuje kao %d. Nakon formata, zapisuje se ime promenljive u koju treba upisati pročitanu vrednost. Simbol & govori da će promenljiva ceoBroj biti promenjena u funkciji scanf, tj. da će pročitani broj biti smešten na adresu promenljive ceoBroj.

Primer 5: Ulaz/izlaz - više vrednosti
U ovom primeru ćemo videti na koji način je omogućeno korisniku da unese više različitih unosa i prikazivanje istih. #include <stdio.h> int main() { int a; float b; printf("Unesi jedan ceo i jedan decimalan broj: "); /* skladistenje vise ulaza */ scanf("%d%f", &a, &b); printf("Uneo si %d i %f", a, b); return 0; } Navedeni program na standardnom izlazu ispisuje: Unesi jedan ceo i jedan decimalan broj: -3 3.4 Uneo si -3 i 3.400000
Format niske za ulaz/izlaz
  • %d za int
  • %f za float
  • %lf za double
  • %c za char
