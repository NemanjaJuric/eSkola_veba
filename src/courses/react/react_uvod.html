Uvod u React (Zdravo svete)


React je JavaScript biblioteka za kreiranje korisničkih interfejsa.

Jednostavan React primer:
ReactDOM.render( <h1>Zdravo svete!</h1>, document.getElementById('root') ); Na stranici ispisuje naslov "Zdravo svete!".

Ovde ćemo se upoznati sa osnovama React aplikacija: elementima i komponentama. Kada njih savladate, moći će te da napravite složene aplikacije od manjih komponenti za višestruku upotrebu.

Ovo je prvo poglavlje u korak-po-korak upoznavanju osnovinih React koncepata.

Svako od narednih poglavlja nadograđuje se na znanja stečena u prethodnim poglavljima.
Kao što smo spomenuli, React je JavaScript biblioteka, tako da se pretpostavlja da posedujete osnovno razumevanje JavaScript jezika.Ukoliko sumnjate u vaše poznavanje, savetujemo vam da odvojite malo vremena i proverite vaše JavaScript znanje.
