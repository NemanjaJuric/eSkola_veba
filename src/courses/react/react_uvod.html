Uvod u React (Zdravo svete)


React je JavaScript biblioteka za kreiranje korisničkih interfejsa.

Jednostavan React primer:
ReactDOM.render( <h1>Zdravo svete!</h1>, document.getElementById('root') ); Na stranici ispisuje naslov "Zdravo svete!".

Ovde ćemo se upoznati sa osnovama React aplikacija: elementima i komponentama. Kada njih savladate, moći ćete da napravite složene aplikacije od manjih komponenti za višestruku upotrebu.

Ovo je prvo poglavlje u korak-po-korak upoznavanju osnovnih React koncepata. Naučićemo i šta su to atributi komponenti, zatim da komponente mogu da imaju svoje stanje i čemu ono služi. Takođe, videćemo i kako se rukuje događajima u React-u.

Lekcije su osmišljene tako da vas na jednostavan i zanimljiv način uvedu u svet React-a. Svaka lekcija je prikazana i na konkretnom primeru u okviru nje.

Svako od narednih poglavlja nadograđuje se na znanja stečena u prethodnim poglavljima.
Kao što smo spomenuli, React je JavaScript biblioteka, tako da se pretpostavlja da posedujete osnovno razumevanje JavaScript jezika.Ukoliko sumnjate u svoje poznavanje, savetujemo vam da odvojite malo vremena i proverite svoje JavaScript znanje. Takođe, preporučujemo da se pre React-a posvetite HTML-u i CSS-u.
