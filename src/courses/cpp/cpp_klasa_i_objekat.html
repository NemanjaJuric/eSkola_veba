Klasa i objekat

Klasa

Klasa predstavlja strukturirani korisnički definisan tip podataka koji obuhvata podatke i metode namenjene definisanju operacija nad podacima klase.
Podaci koji su deo klase nazivaju se podaci članovi klase (engl. data members), polja ili atributi.
Metode koje su deo klase nazivaju se metode članice klase (engl. member methods).

Primer iz svakodnevnog života

Zamislimo da pišemo program koji modeluje ljudsko biće. Ovo ljudsko biće treba da ima identitet: ime, datum rođenja, mesto rođenja i pol - informacije koje ga čine jedinstvenim.
Pored toga, ono može da izvrši određene metode, kao što su razgovor i predstavljanje sebe drugima.

Za modelovanje ljudskog bića u programu potrebna je struktura koja omogućava da unutar nje grupišemo atribute koji definišu ljudsko biće (podaci) i aktivnosti koje to biće može da izvrši (metode), koristeći dostupne atribute.

Primer: Osoba


Deklarisanje klase u programskom jeziku C++

Klasa se deklariše pomoću ključne reči class, praćenu nazivom klase i blokom iskaza koji obuhvata niz atributa, metoda, ali i specifikatora pristupa unutar velikih zagrada. Na kraju se nalazi znak tačka-zarez.

class imeKlase {
//atributi i metode
};

Klasa koja modeluje osobu class Osoba { string imePrezime; string pol; string datumRodjenja; string mestoRodjenja; void predstaviSe(); void razgovor(string tekst); ... }; Ključnom rečju class obezbeđujemo mogućnost kreiranja sopstvenog tipa podataka, koji dalje omogućava kapsuliranje atributa i metoda koje koriste te atribute.
Kapsuliranje predstavlja logičko grupisanje podataka i funkcija koje ih koriste. Ono je važno svojstvo objektno orijentisanog programiranja.

Objekat

Da bismo koristili metode klase, potrebno je kreirati primerak (instancu, pojavu) konkretne klase.
Jedan primerak (instanca, pojava) klase naziva se objektom te klase.
Kada se definiše klasa, memorija se ne dodeljuje, dok kod kreiranja objekta dolazi do dodeljivanja određene količine memorije.

Kreiranje objekta u programskom jeziku C++

imeKlase imeObjekta;

Objekte klase Osoba možemo kreirati na sledeći način: Osoba covek; Alternativno, možemo dinamički kreirati instancu određene klase, koristeći ključnu reč new Osoba* covek = new Osoba();
