Zajednički članovi klase (static)


Kada se veći broj objekata kreira iz iste klase, svaki od njih ima svoje zasebne kopije, tj. instance promenljive.
Na primer, u slučaju klase Bicikl, instance promenljive su kadenca, zupčanik i brzina. Svaki objekat ove klase ima svoje vrednosti za ove promenljive, smeštene na različitim memorijskim lokacijama.

Ukoliko želimo promenljive koje su zajedničke za sve objekte, to možemo postići pomoću statičkog modifikatora, koristeći ključnu reč static. Polja koja imaju statički modifikator u svojoj deklaraciji se nazivaju statička polja ili klasne promenljive. One su povezane sa klasom, a ne sa objektom.

Svaka instanca klase deli jednu klasnu promenljivu koja se nalazi na fiksnom mestu u memoriji.
Statička (zajednička) polja klase su dostupna svim objektima klase. Statičko polje ima samo jednu instancu bez obzira na broj kreiranih objekata klase.

Deklaracija i upotreba statičkih polja

Deklaracija statičkog polja se nalazi u telu klase, dok se njegova definicja (inicijalizacija) uvek vrši van nje (pre poziva main()). Razlog je što statičko polje treba da postoji i pre nego što se deklarišu objekti tipa klase.

Upotreba statičkih polja omogućava da objekti iste klase dele neki podatak i/ili da preko njega međusobno “komuniciraju”. Ona se ponašaju isto kao i globalne statičke promenljive sem što su enkapsulirane u klasu.
Bilo koji objekat može promeniti vrednost klasne promenjive, ali klasnim promenljivim se takođe može manipulisati bez stvaranja instance klase.
Statičke metode ne smeju pristupati poljima koja nisu statička. Statičkim članovima klase se može pristupiti tako što se navede ime klase, :: i ime statičkog člana.

Primer

Pretpostavimo da želimo da kreiramo određeni broj objekata klase Bicikl i svakom objektu želimo da dodelimo serijski broj koji kreće od 1 za prvi objekat.
Ovaj ID broj je jedinstven za svaki objekat i zato je promenljiva instance. Istovremeno, potrebno nam je polje koje će pratiti koliko je objekata klase Bicikl stvoreno, da bismo znali koji je ID sledeći.
Ovaj podatak nije povezan ni sa jednim pojedinačnim objektom, već sa klasom u celini. Da bismo to postigli, možemo koristiti klasnu promenljivu brojBicikala: //primer napisan u programskom jeziku include <iostream> using namespace std; class Bicikl { public: int kadenca; int zupcanik; int brzina; int id; static int brojBicikala; //konstruktor koji je zadužen za inicijalizaciju objekata (o kome će biti više reči u nastavku) Bicikl(int k, int z, int b) { kadenca = k; zupcanik = z; brzina = b; id = ++brojBicikala; //povecamo brojBicikala svaki put kada kreiramo novi objekat klase Bicikl } }; //inicijalizacija statickog polja int Bicikl::brojBicikala = 0; int main() { Bicikl b1(5,2,15); Bicikl b2(6,3,17); cout << "Ukupno objekata: " << Bicikl::brojBicikala << endl; return 0; } Izlaz Ukupno objekata: 2
