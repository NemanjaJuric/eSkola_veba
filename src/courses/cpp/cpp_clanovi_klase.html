Članovi klase


Klase sadrže članove koji predstavljaju njihove podatke i ponašanje. Polja i metode (funkcije) unutar klasa se nazivaju članovima klasa.
Polja predstavljaju podatke koji opisuju stanje objekta klase, dok su metode namenjene definisanju operacija nad poljima klase.
Na primer, klasa Auto ima tri člana klase, dva polja i jednu metodu: class Auto { public: //polja string boja="crvena"; int maxBrzina=200; //metoda void punGas(){ cout << "Auto se krece najbrze moguce." << endl; } };

Polja

Polja (eng. fields) služe za čuvanje nekih podataka (vrednosti) tj. polja predstavljaju stanja objekata. U prethodnom primeru smo mogli videti da su polja promenljive unutar klase.
U sledećem primeru ćemo pokazati pristupanje poljima, o čemu će u narednim lekcijama biti više reči.

Primer //primer napisan u programskom jeziku C++ #include <iostream> #include <string> using namespace std; class Auto { public: string boja="crvena"; int maxBrzina=200; }; int main(){ //kreiranje objekata tipa Auto Auto mojObjekat1; Auto *mojObjekat2=new Auto(); cout << mojObjekat1.boja << ", " << mojObjekat1.maxBrzina << endl; cout << mojObjekat2->boja << ", " << mojObjekat2->maxBrzina << endl; return 0; } Izlaz: crvena, 200 crvena, 200 Polja možemo ostavljati praznim i modifikovati ih pri kreiranju objekta. Na primer: //primer napisan u programskom jeziku C++ #include <iostream> #include <string> using namespace std; class Auto { public: string boja; int maxBrzina; }; int main() { Auto *mojObjekat=new Auto(); mojObjekat->boja="crvena"; mojObjekat->maxBrzina=200; cout << mojObjekat->boja << ", " << mojObjekat->maxBrzina << endl; return 0; } Izlaz: crvena, 200 Ovo je zapravo vrlo korisno pri kreiranju više objekata jedne klase: //primer napisan u programskom jeziku C++ #include <iostream> #include <string> using namespace std; class Auto { public: string boja; string model; int godina; }; int main() { Auto *ford=new Auto(); ford->boja="crvena"; ford->model="Mustang"; ford->godina=1969; Auto opel; opel.boja="bela"; opel.model="Astra"; opel.godina=2005; cout << ford->boja << ", " << ford->model << ", " << ford->godina << endl; cout << opel.boja << ", " << opel.model << ", " << opel.godina << endl; return 0; } Izlaz: crvena, Mustang, 1969 bela, Astra, 2005

Metode

Metode koristimo kako bismo omogućili našem objektu određene aktivnosti. Metode izvršavaju aktivnosti nad internim stanjem objekta i osnovni su mehanizam za komunikaciju između objekata. One uglavnom pripadaju klasi i definišu kako se objekat ponaša.

Primer //primer napisan u programskom jeziku C++ #include <iostream> #include <string> using namespace std; class Auto { public: string boja; int maxBrzina; void punGas(){ cout << "Auto se krece najbrze moguce." << endl; } }; int main() { Auto *mojObjekat=new Auto(); //pozivanje metode mojObjekat->punGas(); return 0; } Izlaz: Auto se krece najbrze moguce.
