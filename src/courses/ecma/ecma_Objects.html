Objekti

Objekti u Javascript-u nam omogućavaju da definišemo nove tipove podataka. Za razliku od promenljivih koje mogu da imaju samo jednu vrednost, objekti mogu imati više. Vrednosti mogu biti niz drugih objekata ili primitivnih tipova podataka, te vrednosti zovemo još i polja.
Postoje dva načinda da se kreira objekat:
  • Koristeći literale objekta
  • Koristeći konstruktor

var user = {}; var name = new Object();

Klase

Klase su neizostavni deo objektno orijentisanog programiranja (OOP). Klase se koriste za definisanje nacrta za modeliranje objekata u stvarnom svetu i organizovanje koda u delove koji se mogu ponovo koristiti. Pre ES6, bilo je teško napraviti klasu u JavaScriptu.
U ES6 možemo da kreiramo klasu pomoću ključne reči class. class Student{ constructor(name, age){ this.name = name; this.age = age; } } Definicija klase može da sadrži samo konstruktore i funkcije. Ove komponente se zajedno nazivaju članovima podataka klase. Klase sadrže konstruktore koji memoriju dodeljuju objektima klase. Klase sadrže funkcije koje su odgovorne za izvršavanje radnji sa objektima.
Kao i drugi programski orijentisani objekti, pomoću nove ključne reči možemo da instanciramo objekat klase. var stu = new Student('Peter', 22)

Nasleđivanje

Nasleđivanje je mogućnost kreiranja novih entiteta od postojećih. Klasa koja je proširena za kreiranje nove klase naziva se nadređena klasa, dok se novostvorena klasa naizva potklasa.
Pre ES6, implementacija nasleđivanja zahtevala je nekoliko koraka. Ali ES6 je pojednostavlila implementaciju nasleđivanja koristeći extends i super. class Parent { show() { document.write("show() metod klase Perent <br>"); } } class Child extends Parent { show() { super.show(); document.write("i show() metod klase Child <br>"); } } var obj = new Child(); obj.show();
