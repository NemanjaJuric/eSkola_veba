ECMAScript funkcije

U sledećem odeljku podsetićemo se javascrypt funkcija i videti koja su to poboljšanja vezana za funkcije koja dolaze sa ECMAScript6 verzijom.

Prednost java skript funkcija

Postoje dve glavne prednosti funkcija, a to su:

  • Možemo da pozivamo jednu funkciju vise puta
  • Program postaje manji i pregledniji jer ne moramo da pišemo novi kod za često obavljene zadatke

Sintaksa funkcije

Je izgled linije koda koji sadrži funkciju, uglavnom izgleda ovako:
function functionName(argument1,argument2,..argumentN) { kod koji funkcija izvršava; }

Ove funkcije mogu da imaju bilo koji broj argumenata

Primeri java skript funkcija

Evo ga primer funkcije bez argumenata

function msg() { alert("Ovo je poruka"); }

Funkcije sa povratnom vrednošću

Funkcija koju smo pozvali će vratiti vrednost na osnovu dobijenih argumenata: function add( a, b ) { return a+b; } var sum = add(10,20); console.log(sum);

Default parametri funkcija

U ES6, funkcije nam omogućavaju inicijalizaciju parametara sa zadatim vrednostima ako parametar nije definisan ili mu se ne prenese vrednost.
function show (num1, num2=200) { console.log("num1 = " +num1); console.log("num2 = " +num2); } show(100);

Arrow functions

Arrow functions ili lambda funkcije uvedene su u ES6, što nam omogućava tačniji način pisanja funkcija u JavaScript. Lambda funkcije čine naš kod čitljivijim i strukturiraniji.
Lambda funkcije su anonimne funkcije (funkcije bez imena i nisu vezane za identifikator). Ne vraćaju nikakvu vrednost i mogu se deklarisati bez ključne reči function. Lambda funkcije se ne mogu koristiti kao konstruktori. Kontekst unutar lamda funkcije je leksički ili statički definisan.

Postoje tri dela lambda funkcije:
  • Parametri: Svaka funkcija može ,ali ne mora da ima parametre.
  • Oznaka strelice / lambda notacija: to je oznaka za strelicu (=>).
  • Iskaz: Predstavlja skup instrukcija funkcije.

const add = (n1,n2) => n1+n2 console.log(add(10,20)) const isEven = (n1) => { if(n1%2 == 0) return true; else return false; } console.log(isEven(10))
Koristeće prethodno naučeno napisati lambda funkciju koja ispisuje prosledjenu reč N puta. Ako N nije prosedjeno funkciji onda datu reč ispisati samo jednom.
