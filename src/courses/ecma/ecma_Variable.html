Promenljive

Promenljiva, po definiciji, je "imenovani prostor u memoriji" koji čuva vrednosti. Promenljiva se mora deklarisati pre nego što se upotrebi. ES5 sintaksa koristila je ključnu reč var da bi postigla isto. ES5 sintaksa za proglašavanje promenljive je sledeća:
var promenljiva_ime
ES6 uvodi sledeću sintaksu deklaracije promenljivih:
  • Koristeći let
  • Koristeći const
Da bismo razumeli koja je razlika, prvo se traba upoznati sa pojmom opsega promenljivih.

Opseg promenljivih

Opseg promenljive je regija vašeg programa u kojoj je ona definisana. Tradicionalno, JavaScript definiše samo dva opsega - globalni i lokalni.
  • Globalni opseg - Promenljivom opsegu globalnog opsega može se pristupiti iz bilo kojeg dela JavaScript koda.
  • Lokalni opseg - promenljivoj sa lokalnim opsegom može se pristupiti iz funkcije u kojoj je deklarisana.
Primer: Globalna ili promenljiva lokalna Sledeći primer deklariše dve promenljive po imenu num - jedna izvan funkcije (globalni opseg), a druga unutar funkcije (lokalni opseg). var num = 10 test funkcije() { var num = 100 console.log ("vrednost num u testu ()" + broj) } console.log ("vrednost num izvan testa ()" + broj) test()

Let


Promenljivoj koja je deklarisana pomoću let ili const dodeljuje se opseg bloka(block scope). Opseg bloka je oblast unutar if, switch naredba ili for i while petlja. Uopšteno govoreći, svaki put kada vidite {vitičaste zagrade}, to je blok. Dakle promenljiva deklarisana unutar bloka neće biti vidljiva van njega.
//Promenljiva se može inicijalizirati bilo u vreme njene deklaracije ili kasnije. <!DOCTYPE html> <html> <body> <script> function run() { { const ime = "Felix"; let god = 30; document.write(ime +" ima "+god+" godina <br>"); } document.write(ime +" ima "+god+" godina <br>"); // greska } run(); </script> </body> </html>
Promenljive deklarirane pomoću ključne reči var nemaju opseg bloka. Ključna reč var daje promenljivoj opseg funkcije. <!DOCTYPE html> <html> <body> <script> "use strict" function test() { var num = 100; document.write("vrednost num u testu () je " + num + "<br>"); { document.write("Unutrašnji blok počinje<br>"); let num = 200; document.write("vrednost num:" + num + "<br>"); } document.write("Van bloka <br>"); document.write("vrednost num:" + num + "<br>"); } test(); </script> </body> </html>
Let omogućava skripti da ograniči pristup promenljivoj do najbližeg ograđenog bloka. Skripta proglašava promenljivu num u okviru lokalnog opsega funkcije i ponovo je proglašava unutar bloka koristeći ključnu reč "let".Vrednost lokalno vidljive promenljive se štampa kada se promenljivoj pristupi izvan unutrašnjeg bloka, dok se u unutrašnjem bloku upućuje promenljiva opsega bloka.

Konstanta

Izjava const stvara referencu na vrednost samo za čitanje. To ne znači da je vrednost koju drži nepromenljiva, već samo što promenljivi identifikator ne može biti dodeljen. Konstante su obuhvaćene blokom, slično kao varijable koje su definisane pomoću izraza let. Vrednost konstante ne može se promeniti ponovnim dodeljivanjem i ne može se ponovo deklarisati.
Sledeća pravila važe za promenljivu deklarisanu pomoću ključne reči const :
  • Konstante ne može biti dodeljeno vrednosti
  • Konstanta se ne može ponovo deklarisati
  • Konstanta zahteva inicijalizator. To znači da se konstante moraju inicijalizirati tokom deklaracije
  • Vrijednost dodijeljena const varijabli je promjenjiva
Primer: const k = 10; k = 12; // rezultiraće greškom !! Za razliku od promenljivih deklarisanih pomoću ključne reči let, konstante su nepromenljive. To znači da se njegova vrednost ne može menjati.
