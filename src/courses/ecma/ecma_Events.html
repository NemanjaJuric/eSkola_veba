ES6 - Events

Javascript je napravljen da dodaje interaktivnost u vašoj strani.Javascript radi ovo uz pomoć mehanizma događaja. Događaji su deo Document Object Model (DOM) nivoa 3 i svaki HTML element sadrži skup događaja koji mogu biti pokrenuti Javascript kod.

Događaj je akcija ili pojava prepoznata od softvera. Ona može biti pokrenuta od korisnika ili sistema. Neki cesti primeri događaja su kada korisnik pritisne na dugme, ucitavanje veb strane, pritiskom na hiper link i tako dalje. Slede neki češće korišćeni HTML događaji. Svaki HTML element ima svoj skup događaja povezanih sa njim. Mi mozemo definisati kako će događaj biti izvršen u Javascriptu koristeci manipulatore događaja.

Manipulatori Događaja

Nakon pojave događaja, aplikacija izvršava grupu povezanih zadataka. Blok koda koji ima ovu svrhu naziva se manipulator događaja.

Događaj tipa: Klik

Ovaj događaj je jedan od najkorišćenijih koji se izvršava kada pritisnemo levi klik na mišu. Možeti staviti potvrdu, upozorenje, i slično.

Primer

<html> <head> <script type = "text/javascript"> const sayHello = () => { document.write("Zdravo svete!"); } </script> </head> <body> <p> Pritisnite na sledeći rezultat </p> <input type = "button" onclick = "sayHello()" value = "Say Hello" /> </body> </html>

OnMouseOver & OnMouseOut

Ova dva događaja će vam pomoći da napravite lepe efekte sa slikama ili čak tekstom. OnMouseOver događaj započinje kada stavite vaš miš preko bilo kog elementa koji ima tu komandu i OnMouseOut započinje kada pomerite miš od elementa na kome ste stavili komandu.

Primer

<html> <head> <script type = "text/javascript"> const over = () =>{ document.getElementById("div1").style.color = "blue"; } const out = () =>{ document.getElementById("div1").style.color = "red"; } </script> </head> <body> <p>Stavite vaš miš unutar polja da vidite rezultat:</p> <div id="div1" onmouseover = "over()" onmouseout = "out()"> <h2> ovo je unutar polja </h2> </div> </body> </html>
