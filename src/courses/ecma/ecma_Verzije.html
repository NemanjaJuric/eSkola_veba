Ostale verzije EcmaScript-a

EcmaScript 2015(ES6) je najveće i najvažnije poboljšanje EcmaScript standarda. Svake godine,počev od 2015. izlaze nove verzije, dodaju se nove karakteristike. Ova strana će biti posvećana tim karakteristikama.

EcmaScript 2016 (ES7)

ES7 uvodi novi matematički operator pod nazivom Eksponencijalni operator. Ovaj operator je sličan metodu Math.pow() .
Eksponencijalni operator je predstavljen dvostrukom zvezdicom **. Operator se može koristiti samo sa numeričkim vrednostima.
Sintaksa za korišćenje eksponencijalni operator: let base = 2; let exponent = 3; document.write("Eksponencijalni operator: 2**3= " +base**exponent);
Metod Array.includes() prihvata parametar, proverava da li vrednost prosleđena kao parametar postoji u nizu. Ovaj metod daje "true" ako je vrednost pronađena ili daje "false" ako vrednost ne postoji. let niz = [1,2,3,5,7,11,13,17] //proverimo da li vrednost 7 postoji u nizu. if(niz.includes(50)){ document.write("Pronađen!"); }else{ document.write("Nije pronađen!"); }

EcmaScript 2017 (ES8)

ES8 uvodi sledeće nove metode ugrađene u objekte.
  • Object.entries
  • Object.values()
  • Object.getOwnPropertyDescriptors()

Metod Objekat.entries() se koristi za pristup svim svojstvima objekta. Objekat.values() se koristi za pristup vrednostima svih svojstava. <script> const student ={ firstName:'Felix', lastName:'Kjellberg' } document.write(Object.entries(student)); document.write("<br>----------------<br>"); document.write(Object.values(student)); </script>
