Operator sabiranja

Ovaj operator funkcioniše na različite načine u zavisnosti koje vrednosti sabira.

Kada sabiramo dva stringa, operator sabiranja spaja levi i desni argument, dok kada sabiramo dva broja, rezultat je njihov zbir.

Primer:
var broj1 = 1; var string1 = "Ja sam broj"; var broj2 = 999; var string2 = "99"; var rezultat1 = string1 + broj 1; //Rezultat će biti "Ja sam broj 1" var rezultat2 = broj 1 + broj2; //Rezultat će biti 1000 var rezultat3 = broj1 + string2; //Rezultat će biti 199

Korišćenjem metode Number() konvertujemo string u broj. Npr:
var broj1 = 1; var string2 = Number("99"); var rezultat3 = broj1 + string2; //Rezultat će biti 100

Nizovi


Nizovi se koriste za smeštanje više vrednosti u jednu promenljivu.

Pošto su nizovi objekti moguće je imati istovremeno različite tipove podataka kao elemente niza.

Niz možemo da kreiramo na dva načina.

Prvi način je da koristimo eksplicitni konstruktor Array():
var predmeti = new Array(); predmeti[0]= "Analiza"; predmeti[1]= "Geometrija"; predmeti[2]= "Algebra"; predmeti[3]= "Programiranje";

Drugi način je da koristimo implicitni konstruktor niza:
var predmeti = ["Analiza", "Geometrija", "Algebra", "Programiranje"];

Elementima niza pristupamo pomoću imena niza i indeksa tog elementa u nizu.

Svojstvo length

Ovo svojstvo vraća dužinu niza, tj. broj elemenata niza.

var predmeti = ["Analiza", "Geometrija", "Algebra", "Programiranje"]; var brojPredmeta = predmeti.length; //Rezultat je 4

String metode
  • toString() - konvertuje niz u string, tj. redja elemente niza razdvojene zarezima
  • join() - konvertuje niz u string, ali separator možemo da biramo sami. Npr: imeNiza.join("*")
  • pop() - uklanja poslednji element iz niza i vraća taj element
  • push() - dodaje novi element na kraj niza i vraća novu dužinu
  • shift() - uklanja prvi element iz niza, ostale pomera za jedno mesto napred i vraća taj element
  • unshift() - dodaje novi element na početak niza i vraća novu dužinu niza
  • sort() - sortira niz
       Ne može se direktno koristiti za sortiranje numeričkih vrednosti jer ih posmatra kao stringove, pa moramo koristiti funkciju sort(function(a, b){ return a-b })
  • reverse() - obrće redosled elemenata niza
  • concat() - spaja nizove
  • slice() - izdvaja podniz


Kretanje kroz nizove

Kako možemo da se krećemo kroz nizove?

Možemo pomoću metoda for() na sledeći nacin:
var predmeti = ["Analiza", "Geometrija", "Programiranje"]; <br> var brojPredmeta = predmeti.length; for(var i = 0; i < brPredmeta; i++){ document.write(predmeti[i] + "<br>"); }
U primeru prolazimo kroz niz i ispisujemo sve njegove vrednosti.

Zadatak 2: Izmeni program tako da kreiraš; niz godišnjih doba i redom ispiseš elemente tog niza.
