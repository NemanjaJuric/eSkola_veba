JavaScript brojevi i datumi

Brojeve u JavaScriptu možemo pisati sa ili bez decimala!

Primer 1: var x = 34.00; // A number with decimals var y = 34; // A number without decimals

Mnogo veliki ili mnogo mali brojevi mogu biti napisani u eksponencijalnom (naučnom) obliku:

Primer 2: var x = 123e5; // 12300000 var y = 123e-5; // 0.00123

Celi brojevi se pišu sa tačnošću do 15 cifara.

Primer 3: <!DOCTYPE html> <html> <body> <p> Celi brojevi se pišu sa tačnošću do 15 cifara.</p> <button onclick="myFunction()"> Try it </button> <p id="demo"></p> <script> function myFunction() { var x = 999999999999999; var y = 9999999999999999; document.getElementById("demo").innerHTML = x + "<br>" + y; } </script> </body> </html>

Maksimalan broj decimala je 17, ali tada tačnost nije uvek 100%.

Primer 4: <!DOCTYPE html> <html> <body> <p>Floating point arithmetic is not always 100% accurate.</p> <button onclick="myFunction()">Try it</button> <p id="demo"></p> <script> function myFunction() { var x = 0.2 + 0.1; document.getElementById("demo").innerHTML = "0.2 + 0.1 = " + x; } </script> </body> </html>

Gore istaknuti problem, pomaže pri množenju i deljenju brojeva:

Primer 5: <!DOCTYPE html> <html> <body> <p>Floating point arithmetic is not always 100% accurate.</p> <p>But it helps to multiply and divide.</p> <button onclick="myFunction()">Try it</button> <p id="demo"></p> <script> function myFunction() { var x = (0.2 * 10 + 0.1 * 10) / 10; document.getElementById("demo").innerHTML = "0.2 + 0.1 = " + x; } </script> </body> </html>

Deljenje nulom odredjuje beskonačnost:

Primer 6: <!DOCTYPE html> <html> <body> <p> Deljenje nulom određuje beskonačnost.</p> <button onclick="myFunction()">Try it</button> <p id="demo"></p> <script> function myFunction() { var x = 2 / 0; var y = -2 / 0; document.getElementById("demo").innerHTML = x + "<br>" + y; } </script> </body> </html>

Operacija tipeof Infinity vraća broj.

Primer 7: <!DOCTYPE html> <html> <body> <p> Beskonačno je broj.</p> <button onclick="myFunction()">Try it</button> <p id="demo"></p> <script> function myFunction() { document.getElementById("demo").innerHTML = typeof Infinity; } </script> </body> </html>

Operacije nad brojevima

Operacija ToString () vraća broj kao string (niz znakova).

Primer 8: <!DOCTYPE html1> <html> <body> <h2>The JavaScript toString() Method</h2> <p>Operacija toString() pretvara broj u niz znakova:</p> <p id="demo"></p> <script> var x = 123; document.getElementById("demo").innerHTML = x.toString() + "<br>" + (123).toString() + "<br>" + (100 + 23).toString(); </script> </body> </html>

Operacija toPrecision () vraća broj kao niz znakova zadate dužine, uz poštovanje pravila zaokrugljivanja decimalnih brojeva.

<!DOCTYPE html> <html> <body> <p>Operacija toPrecision() pretvara broj u niz znakova zadate preciznosti :</p> <p id="demo"></p> <script> var x = 9.656; document.getElementById("demo").innerHTML = x.toPrecision() + "<br>" + x.toPrecision(2) + "<br>" + x.toPrecision(4) + "<br>" + x.toPrecision(6); </script> </body> </html>

Operacija valueOf () vraća broj kao broj.

Primer 10: <!DOCTYPE html> <html> <body> <p id="demo"></p> <script> var x = 123; document.getElementById("demo").innerHTML = x.valueOf() + "<br>" + (123).valueOf() + "<br>" + (100 + 23).valueOf(); </script> </body> </html>

Operacija Number () može se koristiti za pretvaranje JavaScript promenljivih u brojeve. Ako se broj ne može konvertovati, vratiće se poruka NaN (nije broj).

Primer 11: <!DOCTYPE html> <html> <body> <p>Ova operacija pretvara promenljive u brojeve, ili vraća NaN ako konverzija nije moguća:</p> <p id="demo"></p> <script> document.getElementById("demo").innerHTML = Number(true) + "<br>" + Number(false) + "<br>" + Number(new Date()) + "<br>" + Number(" 10") + "<br>" + Number("10 ") + "<br>" + Number("10 6"); </script> </body> </html>

Operacija parseFloat() pretvara stringove u brojeve. Ako konverzija nije moguća, vratiće se NaN.

Primer 12: <!DOCTYPE html> <html> <body> <p>Ova funkcija pretvara stringove u brojeve, ili vraća NaN ako konverzija nije moguća:</p> <p id="demo"></p> <script> document.getElementById("demo").innerHTML = parseFloat("10") + "<br>" + parseFloat("10.33") + "<br>" + parseFloat("10 6") + "<br>" + parseFloat("10 years") + "<br>" + parseFloat("years 10"); </script> </body> </html>

JavaScript Datumi

Brojeve u JavaScriptu možemo pisati sa ili bez decimala!

Primer 1: <!DOCTYPE html> <html> <body> <p id="demo"></p> <script> document.getElementById("demo").innerHTML = Date(); </script> </body> </html>

Korišćenjem operacije new Date(), prikazuje se trenutni datum i vreme:

Primer 2: <!DOCTYPE html> <html> <body> <p id="demo"></p> <script> var d = new Date(); document.getElementById("demo").innerHTML = d; </script> </body> </html>

Kada prikažete datum objekta u HTML, on se automatski konvertuje u string operacijom toString ().

Primer 3: <!DOCTYPE html> <html> <body> <p id="demo"></p> <script> var d = new Date(); document.getElementById("demo").innerHTML = d.toString(); </script> </body> </html>

Operacija toDateString () prikazuje datum u uobičajenom formatu:

Primer 4: <!DOCTYPE html> <html> <body> <p id="demo"></p> <script> var d = new Date(); document.getElementById("demo").innerHTML = d.toDateString(); </script> </body> </html>

Operacija toUTCString() pretvara datum u niz znakova (string) standardnog oblika.

Primer 5: <!DOCTYPE html> <html> <body> <p> The toUTCString() method converts a date to a UTC string (date display standard). </p> <p id="demo"></p> <script> var d = new Date(); document.getElementById("demo").innerHTML = d.toUTCString(); </script> </body> </html>

JavaScript Formati Datuma

Datum u ISO formatu može biti napisan bez navođenja dana (GGGG-MM):

Primer 6: var d = new Date("2017-03"); <!DOCTYPE html> <html> <body> <h1>JavaScript ISO Dates</h1> <p id="demo"></p> <script> document.getElementById("demo").innerHTML = new Date("2017-03"); </script> </body> </html>

Datum u ISO formatu može se napisati bez meseca i dana (samo godina):

Primer 7: var d = new Date("2017"); <!DOCTYPE html> <html> <body> <h1>JavaScript ISO Dates</h1> <p id="demo"></p> <script> document.getElementById("demo").innerHTML = new Date("2017"); </script> </body> </html>

Datum u ISO formatu mogu biti pisani sa dodatim satima, minutima i sekundama (GGGG-MM-DDTčč: mm: SSZ):

Primer 8: var d = new Date("2015-03-25T12:00:00Z"); <!DOCTYPE html> <html> <body> <h1>JavaScript ISO Dates</h1> <p> Datum i vreme su odvojeni velikim T.</p> <p> UTC-vreme je definisano velikim slovom Z.</p> <p id="demo"></p> <script> document.getElementById("demo").innerHTML = new Date("2015-03-25T12:00:00Z"); </script> </body> </html>

Ako želite da izmenite vreme u odnosu na UTC, uklonite Z i dodajte + HH: MM ili -HH: MM umesto:

Primer 9: var d = new Date("2015-03-25T12:00:00-06:30"); <!DOCTYPE html> <html> <body> <h1>JavaScript ISO Dates</h1> <p>Modify the time relative to UTC by adding +HH:MM or subtraction -HH:MM to the time.</p> <p id="demo"></p> <script> document.getElementById("demo").innerHTML = new Date("2015-03-25T12:00:00-06:00"); </script> </body> </html>

JavaScript Operacije nad datumima


Unutrašnji sat u Java Scrupt-u računa vreme od ponoći 1. januara 1970. godine.


Operacija getTime () prikazuje koliko je milisekundi prošlo od 1. januara 1970. godine do sada:

Primer 10: <script> var d = new Date(); document.getElementById("demo").innerHTML = d.getTime(); </script> <!DOCTYPE html> <html> <body> <p>Unutrašnji sat u Java Scrupt-u racuna vreme od ponoci 1. januara 1970. godine.</p> <p>Ova operacija prikazuje broj milisekundi od tada do sada:</p> <p id="demo"></p> <script> var d = new Date(); document.getElementById("demo").innerHTML = d.getTime(); </script> </body> </html>

Operacija getFullyear () prikazuje tekuću godinu u obliku četvorocifrenog broja:

Primer 11: <script> var d = new Date(); document.getElementById("demo").innerHTML = d.getFullYear(); </script> <!DOCTYPE html> <html> <body> <p>Prikazati tekuću godinu u obliku četvorocifrenog broja:</p> <p id="demo"></p> <script> var d = new Date(); document.getElementById("demo").innerHTML = d.getFullYear(); </script> </body> </html>

Operacija getDay () prikazuje redni broj dana u tekućoj nedelji (0-6):

Primer 12: <script> var d = new Date(); document.getElementById("demo").innerHTML = d.getDay(); </script> <!DOCTYPE html> <html> <body> <p>Ovo je redni broj današnjeg dana u tekućoj nedelji:</p> <p id="demo"></p> <script> var d = new Date(); document.getElementById("demo").innerHTML = d.getDay(); </script> </body> </html>

Može se koristiti i niz imena dana u nedelji. Tada će operacija getDay () prikazati naziv današnjeg dana:

Primer 13: <script> var d = new Date(); var days = ["Sunday", "Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday"]; document.getElementById("demo").innerHTML = days[d.getDay()]; </script> <!DOCTYPE html> <html> <body> <p>Today is:</p> <p id="demo"></p> <script> var d = new Date(); var days = ["Sunday", "Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday"]; document.getElementById("demo").innerHTML = days[d.getDay()]; </script> </body> </html>

Operacija setFullyear () postavlja željeni datum.

Primer 14: <script> var d = new Date(); d.setFullYear(2020, 0, 14); document.getElementById("demo").innerHTML = d; </script> <!DOCTYPE html> <html> <body> <p>Ova operacija postavlja datum po želji.</p> <p>JavaScript numeriše mesece od 0 do 11 (jul je na primer 6. mesec u godini). </p> <p id="demo"></p> <script> var d = new Date(); d.setFullYear(2020, 0, 14); document.getElementById("demo").innerHTML = d; </script> </body> </html>
