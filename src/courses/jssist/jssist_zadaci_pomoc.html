Rešenje zadatka 1:
<html> <head> <title>Rešenje</title> </head> <body> <h3> Porez na imovinu </h3> <script> var imovina = { povrsinaImovine: 0, godinePosedovanjaImovine: 0, porez: function () { Porez = this.povrsinaImovine * 30000.00 - 0.08 * this.godinePosedovanjaImovine * 30000.00; return Porez; } } imovina.povrsinaImovine = prompt("Povrsina vaše imovine za koju želite da izračunate visinu poreza je: "); document.write("Površina vaše imovine je: " + imovina.povrsinaImovine + " m<sup>2</sup><br>"); imovina.godinePosedovanjaImovine = prompt("Koliko godina posedujete navedenu imovinu?"); document.write("Imovinu posedujete " + imovina.godinePosedovanjaImovine + " godina. <br>"); document.write("Visina poreza na vašu imovinu je " + imovina.porez() + " dinara."); </script> </body> </html>
Po uslovima zadatka prvo je definisan objekat imovina, koji ima osobine:
  • povrsinaImovine u kojoj je sačuvana numerička vrednost površine imovine za koju korisnik želi da izračuna visinu poreza,
  • godinePosedovanjaImovine u kojoj je sačuvana numerička vrednost godina koliko je korisnik vlasnik te imovine,
  • porez koja je funkcijska osobina i izračuvana visinu poreza po formuli koja je data u tekstu zadatka.
Potom, na internet stranici se funkcijom prompt zahteva unos neophodnih podataka, koji se redom ispisuju funkcijom write. Na kraju, ispisuje se i vrednost funkcijskog svojstva objekta imovina.

<html> <head> <title>Rešenje</title> </head> <body> <script> var datum = new Date(); function oboji() { if (datum.getHours() < 20) document.body.style.backgroundColor = "yellow"; else document.body.style.backgroundColor = "blue"; } oboji(); </script> </body> </html>
Definisana je nova promenljiva datum koja poziva funkciju Date i tako su u njemu sačuvani podaci o datumu i vremenu u trenutku pokretanja dokumenta. Funkcijom oboj menja se boja pozdine na žutu ili plavu. Da li će boja pozadine biti plave ili žute boje odruje naredba if koja ispituje koliko je sati u promenljivoj datum.

Rešenje zadatka 3:
<html> <head> <title>Rešenje</title> </head> <body> <h3> Prorocica Druzila </h3> <h5> Dobrodosli </h5> <p> Vase trenutno raspolozenje je: </p> <script> var raspolozenje = ["Tuzan/Tuzna", "Uplakan/Uplakana", "Srecan/Srecna", "Radostan/Radosna", "Veseo/Vesela", "Isfrustriran/Isfrustrirana", "Zaljubljen/Zaljubljena", "Namrgodjen/Namrgodjena", "Ljubomoran/Ljubomorna", "Nasmejan/Nasmejana"]; function pogodi() { var rnd = Math.random(); return Math.floor(rnd * 10); }; document.write(raspolozenje[pogodi()]); </script> </body> </html>
Definisan je niz stringova pod nazivom raspolozenje u kome su sačuvana različita raspoloženja koja osoba može imati. Funkcija pogodi slučajno bira celi broj iz intervala [0,9], a potom se na internet stranici ispisuje string iz niza raspolozenje, koji stoji na slučajno odabranom mestu.

Rešenje zadatka 4:
<html> <head> <title>Rešenje</title> </head> <body> <h3> Loto </h3> <h5> Ko igra, taj i dobija </h5> <script> function izvuciBroj() { var rnd = Math.random(); return Math.ceil(rnd * 39); }; function svihSedamBrojeva() { var izvuceni = []; var novBroj = 0; var ind = 0; for (i = 0; i < 7; i++) { ind = 0; novBroj = izvuciBroj(); for (j = 0; j < i; j++) { if (novBroj == izvuceni[j]) { ind = 1; } } if (ind == 0) { izvuceni[i] = novBroj; } else if (ind == 1) { i--; } } return izvuceni; } document.write(svihSedamBrojeva()); </script> </body> </html>
Funkcija izvuciBroj() pri pozivu određuje jedan slučajno odabrani celi broj u intervalu brojeva [1,39].
Nakon toga funkcija svihSedamBrojeva() najmanje sedam puta poziva funkciju izvuciBroj() u prvoj for petlji. Dok druga for petlja služi tome da ukoliko se neki broj ponovi pri izvlačenju, on se preskače i izvlačenje se ponavlja. Na kraju, rezultat se ispisuje na internet stranici.

Rešenje zadatka 5:
<html> <head> <title>Rešenje</title> </head> <body> <h3> (Ne)prijatelj </h3> <h5> Da li da ti verujem? </h5> <script> function ime() { return prompt("Unesite ime osobe:"); } function prezime() { return prompt("Unesite prezime osobe:"); } function proracunPrijateljstva(i, p) { if (i.localeCompare(p) > 0) { return "Ne"; } else if (i.localeCompare(p) == 0) { return "Reci ove osobe uzimajte sa rezervom"; } else { return "Da, potpuno"; } } var imeOsobe = ime(); var prezimeOsobe = prezime(); document.write(proracunPrijateljstva(imeOsobe, prezimeOsobe)); </script> </body> </html>
Definisane su tri funkcije:
  • ime - koja od korisnika traži da se unese ime osobe,
  • prezime - koja od korisnika traži da se unese prezime osobe,
  • proracunPrijateljstva - koja poredi prosleđeno ime i prezime i u zavisnosti od rezultata vraća string Ne, Reci ove osobe uzimajte sa rezervom ili Da, potpuno.
Ime i prezime se porede string metodom localCompare, koja ispituje da li prvi string pri porenju stoji pre drugog (u tom slučaju povratna vrednost ove funkcije je -1), nakon drugog (u tom slučaju povratna vrednost ove funkcije je 1), ili su stringovi jednaki (u tom slučaju povratna vrednost je 0).
