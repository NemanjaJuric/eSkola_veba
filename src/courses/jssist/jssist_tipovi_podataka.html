Tipovi podataka


JavaScript definiše od tri do šest tipova podataka. To su:
  • brojevi
  • znakovni nizovi - stringovi
  • bulove vrednosti
  • null vrednisti
  • nedefinisana vrednost
  • objekti

JavaScript ima i nekoliko referentnih tipova podataka uključujući tipove Array, Date, RegExp.

BROJEVNI TIP

U JavaScriptu celi brojevi i brojevi sa pokretnim zarezom nemaju specijalne ili odvojene tipove.

Primeri brojeva u JavaScriptu:
   14
   8,4
   -124
   0xa

Primer:
<script type="text/javascript"> var a = 0xa; var b = 0xe; var c = a * b; alert(c); </script>
Ovaj kod definiše dve promenljive, a i b i dodeljuje im heksadekadne brojeve. Potom se kreira nova promenljiva c, i dodeljuje joj se proizvod prethodne dve promenljive. Dobijena promenljiva se prosledjuje funkciji alert(). A kao rezultat se dobija broj u decimalnom formatu.

NUMERIČKE FUNKCIJE

JavaScript ima neke ugradjene funkcije za rad sa numeričkim vrednostima.
Jedna od tih funkcija je IsNaN() (is Not a Number).
Ova funkcija se koristi da bismo mogli da proverimo da li je neki broj dopušten ili ispravan u skladu sa ECMA-262 specifikacijom.
Na primer: "4" jeste broj, dok deljenje nulom je nedopušten broj isto kao i vrednosti znakovnog niza(npr. "četiri").

Primer koji testira da li je broj dopušten ili ne.
<script type="text/javascript"> document.write("Is Not a Number:" + isNaN("123")); </script>
Kao rezultat ćemo dobiti false jer celobrojna vrednost 123 jeste broj.

Funkcija isNaN() se često koristi prilikom provere ispravnosti ulaznih podataka, kako bi se utvrdilo da li je promenljiva uneta kao broj ili kao tekst.

OBJEKAT Math

Math objekat je sprecijalan ugrađen objekat koji se koristi za rad sa brojevima u JavaScriptu.

Ovaj objekat daje vrednost PI, kvadratni koren broja, apsolutnu vrednost, pseudoslučajni broj...

Neka svojstva funkcije objekta Math:
Svojstvo Definicija
Math.PI Daje vrednost broja pi
Math.random() Daje pseudoslucajan broj
Math.abs(x) Daje apsolutnu vrednost x
Math.sqrt(x) Daje koren broja x
Math.pow(x,y) Daje x stepenovan na y
Math.round(x) Daje x zaokružen na najbliži ceo broj


Primer korišćenja Math objekta:
<script> var a = 0xa; var b = Math.pow(4,3); var c = a * b; document.write("Dobijena vrednost je: " + c + "<br>"); </script>
U primeru računamo proizvod jedne promenljive koje ima heksadekadnu vrednost, i promenljive koju smo dobili korišćenjem Math objekta.

Znakovni nizovi - Stringovi


Stringovi u JavaScriptu sastoje se od jednog ili više znakova pod navodnicima. Navodnici mogu biti jednostruki ili dvostruki.


Primeri sa dvostrukim navodnicima Primeri sa jednostrukim navodnicima
"Zdravo svete!" 'Zdravo!'
"Rekla nam je 'zdravo'." 'Rekla je: "Napolju pada kiša".'


Unutar jednog tipa navodnika može se koristiti drugi tip kao što smo imali u primeru iz tabele.

Ukoliko nam je potrebno da koristimo isti stil navodnika unutar stringa, moramo poništiti specijalno značenje navodnika, da ga JavaScript ne bi interpretirao.

Kako poništiti specijalno značenje navodnika?
To se radi korišćenjem znaka za obrnutu kosu crtu (\). Dosta nam je korisno za ono što se piše na engleskom jeziku. Npr: Umesto 'I am' - 'I\'m' ili 'You are' - 'You\'re' itd.

U JavaScriptu postoje drugi znakovi sa sprecijalnim značenjem.

Znakovi sa specijalnim značenjem Vrednost znaka
\b Backspace
\t Tabulator
\n Novi red
\v Vertikalni tabulator
\f Prelazak na sledeću stranicu
\r Znak za početak reda
\\ Obrnuta kosa crta


Takodje postoje metode i svojstva stringova.
Neke od metoda i svojstava su: length, substr(), substring(), slice(), concat(), toUpperCase(), toLowerCase(), charAt(), indexOf(), trim(), valueOf().

Svojstvo length

Ovo svojstvo se koristi za odredjivanje dužine stringa.

Možemo ga koristiti na dva nacina. Direktno na stringu, ili prilikom poziva na promenljivoj.

Primer kada se length poziva direktno na stringu.
document.write("Ovo je string.".length); //Ispisuje "14"

Primer kada se length poziva na promenljivoj:
var MojString = "Ovo je string."; document.write(MojString.length); //Ispisuje "14"
Oba ova primera daju isti rezultat.

Metode slice, substring, substr

Metode subtring() i slice() prihvataju dva argumenta, tako da je prvi argument početna pozicija od koje se računa, dok je drugi argument pozicija za zaustavljanje, što nije slučaj sa metodom substr() gde je drugi argument broj znakova od početne pozicije koji se računaju.

Primer korišćenja ovih metoda:
var MojString = "Ovo je string."; document.write(MojString.substring(3)); //Ispisuje " je string." document.write(MojString.substring(3,9)); //Ispisuje " je st" document.write(MojString.substr(3)); //Ispisuje " je string." document.write(MojString.substr(3,9)); //Ispisuje " je strin"

Metod concat

Ovaj metod spaja dva niza.

Primer nadovezivanja sa concat():
var PrviString = "Zdravo "; var DrugiString = "svete"; document.write(PrviString.concat(DrugiString)); //Ispisuje "Zdravo svete"

Umesto concat() mozemo koristiti znak '+' za nadovezivanje.

Primer nadovezivanja sa +:
var PrviString = "Zdravo "; var DrugiString = "svete"; document.write(PrviString + DrugiString); //Ispisuje "Zdravo svete"

Primer:
<script> var a = "Dobar "; var b = "dan!"; var c = a + b; var l=c.length; document.write("Nadovezivanjem a i b dobili smo: " +c + "<br>" + "Dužina dobijenog stringa je: " + l + "<br>"); </script>
U primeru spajamo dva stringa pomoću znaka za nadovezivanje ( + ), potom ispisujemo string koji smo dobili i njegovu dužinu koju smo prethodno izračunali.


Zadatak 2: Izmeni program na sledeći način: Za vrednosti promenljivih a i b izaberi bilo koja dva stringa, potom izračunaj i ispiši dužinu svakog stringa ponaosob, a potom spoj ta dva stringa pomoću metode concat(), i izračunaj dužinu dobijene vrednosti, i ispiši sve to.


Bulove vrednosti


Bulove vrednosti mogu biti true (tačno) i false (netačno).

Najčešće se koriste Bulovi izrazi unutar testova, kao što je if/then/else iskaz.

Primer iskaza u kom je korišćen Bulov izraz:

if(broj < 10){ &nbsp&nbsp//uradi nešto }
Ovaj primer služi da pokaže da li će se kod unutar zagrada izvršiti ili ne.
Ukoliko je broj manji od 10, Bulov izraz će imati vrednost true i kod će se izvršiti, u suprotnom neće.

Null vrednosti


Ovo je specijalan tip podataka u JavaScriptu.
To je ništa. Predstavlja se kao vrednost false.

Ne treba mešati null sa praznom promenljivom. Praznina i null vrednost se razlikuju. Prazna promenljiva je i dalje puna - puna je praznine.

Npr. ukoliko bismo napisali ovako nešto:
var mojaPromenljiva='';
to znači da je promenljiva moja Promenljiva prazna, ali ona nema vrednost null.

Nedefinisane vrednosti


Ovo je zapravo stanje koje se koristi kao vrednost.
Koristimo ga onda kada promenljivoj nije dodeljena nikakva vrednost.
Razlikuje se od null vrednosti.

Primer:
if (number.value < 100) { ispis = "Proizvod je: " + number.value; } else { ispis = 'Rezultat je veći od 100!'; }
U primeru proveravamo da li je proizvod dva uneta broja veći ili manji od 100, ako je manji ispisujemo rezultat, a ako je veći ispisujemo: 'Rezultat je veći od 100!'.

Zadatak 3: Promeni program tako da uneseš jedan broj, proveri da li je njegov kvadrat manji od 100, ako jeste ispiši rezultat, a ako nije ispiši da je kvadrat prvog broja veći od 100.
