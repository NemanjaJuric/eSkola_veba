Uvod u JavaScript, promenljive i tipovi podataka



Šta je JavaScript?


JavaScript je najpopularniji programski jezik koji programira dinamično ponašanje naših veb stranica. JavaScript je lagan, ali izuzetno moćan programski jezik. Najčešće se sa njim susrećemo u našim pregledačima. JavaScript se može naći u osnovnim aplikacijama za PDF i u elektronskim knjigama. Čak i veb serveri mogu biti pokrenuti od strane JavaScript-a. Kao dinamičan programski jezik JavaScript ne mora biti pokrenut kroz kompajler koji čita kod u nešto što naš pregledač razume. Pregledač efektivno čita kod na isti način kao i mi, zatim ga interpretira. JavaScript je takođe slabo tipiziran programski jezik. Nije neophodno reći JavaScript-u koje je vrste promenljiva. Ako promenljivoj dodelimo vrednost 5, ne moramo programski da odredimo tu promenljivu kao broj. JavaScript prepoznaje 5 kao broj.
JavaScript razlikuje velika i mala slova. Promenljiva koja se naziva "mojaPromenljiva", ona koja se naziva "mojapromenljiva" i promenljiva sa nazivom "MOJAPromenljiva" se tretiraju kao tri različita objekta. Takođe, u nazivima promenljivih beli prostori kao što su tabovi i razmaci se ignorišu, osim ako nisu deo niza teksta i pod znacima navoda.
Skript je sastavljen od serija iskaza. Iskaz je komanda koja kaže pregledaču šta da radi. Ovde ćemo dati primer jednostavnog iskaza koja čini da pregledač prikaže upozorenje (obaveštenje) "Hvala Vam".

<!DOCTYPE html> <html> <body> <button type="button" onclick="myFunction()">Obaveštenje</button> <script> function myFunction() { alert("Hvala Vam."); } </script> </body> </html>

Takođe, bitno je reći da u JavaScript-u postoje dva metoda za korišćenje komentara. Za jednolinijske komentare koristimo karaktere // koje stavljamo na početku linije.

// Ovo je jednolinijski komentar

Višelinijski komentari se nalaze između karaktera /* */ . Sve između ovih karaktera biće ignorisano od strane pregledača.

/* Ovo je višelinijski komentar
Bilo šta između ovih karaktera biće kompletno
ignorisano od strane pregledača */

Šta može JavaScript?


  • JavaScript može promeniti HTML sadržaj.
  • JavaScript može promeniti HTML atribute.
  • JavaScript može promeniti HTML stilove (CSS).
  • JavaScript se često koristiti za proveru validnosti unosa na našoj veb stranici.


Šta nije JavaScript?


Samo ime programskog jezika JavaScript je zbunjujuće. Nevezano za ime, važno je naglasiti da je jezik JavaScript potpuno drugačiji od jezika Java, razlikuju se u planu i dizajnu.

JavaScript je kreiran od strane kompanije Netscape 1995. godine i njegovo orginalno ime je bilo LiveScript, što je ukazivalo na pravljenje živih (dinamičnih) veb stranica. U to doba, u kompaniji Sun Microsystems pojavio se jezik Java i veoma brzo stekao popularnost na internetu pošto je omogućio kreiranje apleta — aplikacija koje se preko veba distribuiraju korisnicima. Netscape ubacuje podršku za aplete u korisnički pregledač. Novi Netscape-ov jezik se između ostalog mogao koristiti za kreiranje neke interakcije izmedu Java apleta i veb stranice u koju je aplet ubačen. Kako bi iskoristio popularnost jezika Jave, uz dozvolu kompanije Sun, Netscape se odlučuje na marketinški potez i svoj jezik preimunuje u JavaScript. Skraćeno ime programskog jezika JavaScript-a je JS. JavaScript je imao lošu reputaciju. Programeri su iskoristili JavaScript za neželjena ponašanja naših veb stranica i JavaScript je bio razlog za nebrojane sigurnosne propuste. Moderni pregledači nisu dozvoljavali ovakvo ponašanje programskog jezika JavaScript. JavaScript je uvek dozvoljavao programerima da imaju ogromnu količinu moći u vezi sa ponašanjem naših veb stranica i ponašanjem pregledača, a na nama je da iskoristimo tu moć na odgovoran način.
Na razvoju JavaScript-a danas radi Mozilla Foundation, koja se smatra naslednikom Netscape-a.

Ugrađivanje skripta na stranicu


Skript ugrađujemo na našu stranicu tako što JavaScript kod dodamo kao sadržaj script elementa. Element script može biti smešten u telu i zaglavlju HTML stranice.

<script>
JavaScript kod se nalazi ovde.
</script>

Možemo navesti JavaScript kod u spoljašnjoj datoteci na koju se ukazuje korišćenjem src atributa.

<script src=”moj_skript.js"> </script>

Primer našeg prvog JavaScript programa



<!DOCTYPE html> <html> <body> <h1>Moj prvi JavaScript program!</h1> <p id="demo"></p> <button type="button" onclick="myFunction()">Datum i vreme</button> <script> function myFunction() { document.getElementById("demo").innerHTML = Date(); } </script> </body> </html>


Zadatak 1: U primeru kada kliknemo na dugme prikazuje se trenutni datum i vreme. Izmeni program tako da kada se klikne na dugme izađe tekst "Zdravo JavaScript!"

Ovo je samo upoznavanje sa terminima koje ćete čuti često dok budete učili JavaScript, i onda će oni imati sve više smisla kako budete nastavljali.

Promenljive


Promenljive se koriste za čuvanje vrednosti podataka i za izraze.

U JavaScriptu promenljive se deklarišu ključnom rečju var.

Nakon ključne reči var sledi ime promenljive.

Imena promenljivih mogu da sadrže velika i mala slova, kao i brojeve, ali ne mogu da počinju brojem.

Kao imena promenljivih se ne mogu koristiti rezervisane reči (class, enum, const, var, ...).

Osim znaka za donju crtu ( _ ) promenljive ne mogu da sadrže razmak, kao ni druge znakove interpunkcije.

Ispravno deklarisane promenljive Neispravno deklarisane promenljive
var x; var 1x;
var broj1; var broj 1;
var imePromenljive; var ime.Promenljive;
var myVar; var var;


Da bi se inicijalizovala promenljiva ili da bi joj se dodelila odredjena vrednost koristi se sledeca sintaksa:
var imePromenljive = vrednost;

Promenljiva koja je deklarisana bez dodele vrednosti ima vrednost undefined.

Vrednost dodeljena promenljivoj može biti znakovna ili numerička.

Promenljive u JavaScriptu nisu strogo tipizarane. To znači da nije potrebno da se deklariše da li promenljiva sadrži ceo broj, broj sa pokretnim zarezom ili znakovni tip.

Tip podataka koji će biti sadržan unutar neke promenljive može da se promeni jednostavnim naknadnim dodeljivanjem.

Područje važenja


Područje važenja se odnosi na lokacije sa kojih može da se pristupi vrednosti.

Promenljive mogu biti globalno vidljive i lokalno vidljive.

Globalno vidljive promenljive su one koje se koriste izvan funkcije. Njima može da se pristupi bilo gde u okviru JavaScript programa.

Lokalno vidljive promenljive su definisane samo unutar funkcije i vidljive su samo unutar funkcije. Vrednostima ovih promenljivih može da se pristupi samo u toj funkciji.

Primer:
<script type="text/javascript"> var a = "Ja sam globalna promenljiva"; function ispisi(x){ alert(a); alert(x); } ispisi("Argument"); </script>
Ovaj kod definiše dve promenljive, jednu globalnu a i jednu lokalnu promenljivu x, koja je lokalna za funkciju uradiNesto().
Potom se unutar funkcije uradiNesto() i lokalna i globalna promenljiva ravnopravno koriste kao argumenti funkcije alert().

Primer gde promenljiva a sadrži ceo broj, a potom pomoću druge dodele ona menja znakovni sadržaj:
<script> var a = 10; alert(a); a = "Ovo je broj deset"; alert(a); </script>
U primeru ispisujemo obe vrednosti promenljive a.

Zadatak 2: Izmeni program tako da zadatoj promenljivoj a dodeliš neku drugu vrednost, i ispišeš samo izmenjenu vrednost promenljive.
