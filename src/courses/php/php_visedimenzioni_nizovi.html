Višedimenzioni nizovi

Ranije smo u tutorijalu opisali jednodimenzione nizove.
Višedimenzioni niz je niz u kome svaki element moze takodje da bude niz. Vrednostima u višedimenzionom nizu pristupamo pomoću vise indeksa. Dimenzija niza odredjuje broj indeksa koji su potrebni da bi pristupili elementima niza.

Dvodimenzioni niz izgleda ovako: <?php $ocene = array( "Marko" => array("Matematika" => 5, "Fizika" => 4, "Informatika" => 5), "Janko" => array("Matematika" => 4, "Fizika" => 3, "Informatika" => 5), "Marija" => array("Matematika" => 5, "Fizika" => 5, "Informatika" => 4) ); ?>
  • Za dvodimenzioni niz su nam potrebna 2 indeksa da bi pristupili proizvoljnom elementu niza.
  • Za trodimenzionii niz su nam potrebna 3 indeksa da bi pristupili proizvoljnom elementu niza.

Primer kako pristupamo elementima niza: echo "Marko je iz fizike dobio ".$ocene['Marko']['Fizika']."<br>"; echo "Janko je iz informatike dobio ".$ocene['Janko']['Informatika']."<br>"; echo "Marija je iz matematike dobla ".$ocene['Marija']['Matematika']."<br>";

Napisati dvodimenzioni niz auti, u kom za svaki element čuvamo marku, ukupan broj auta te marke, i ukupan broj prodatih auta te marke. Za prvi u nizu ispisati marku, ukupan broj, i broj prodatih auta te marke.
