Petlje


Petlje izvršavaju zadate naredbe određeni broj puta. Ponavljanje bloka naredbi se naziva iteracija. PHP podržava četiri petlje:
  • while
  • do while
  • for
  • foreach

While petlja


While petlja je najjednostavnija. Izvršava naredbe sve dok je zadati uslov ispunjen (true). Može se desiti da se naredbe ne izvrše uopšte, a može se desiti da je uslov uvek ispunjen i da se naredbe izvršavaju beskonačan broj puta (beskonačna petlja).
while ( uslov ) { naredbe }
Ako želimo da ispišemo cifre broja 1234, možemo to da uradimo pomoću while petlje. <?php $broj = 1234; while ($broj > 1) { $cifra = $broj % 10; echo $cifra; echo "<br>"; $broj /= 10; } ?>

Sve dok je broj veći od jedan, cifru dobijamo tako što uzimamo ostatak pri deljenju broja sa 10 ($cifra = $broj % 10). Nju ćemo da ispišemo sa naredbom echo i da broj podelimo sa 10 ($broj /= 10).

Do while petlja


Ova petlje će uvek jednom izvršiti blok naredbi, zatim će ispitati uslov i opet izvršiti naredbe ako je uslov ispunjen.
do { naredbe } while (uslov);
Treba da ispišemo brojeve od 1 do 5. To ćemo lako uraditi pomoću ove petlje. Prvo ćemo ispisati broj 1, a zatim ga uvećati ($broj++). Uvećani broj ispisujemo sve dok je manji ili jednak od 5 (echo $broj). <?php $broj = 1; do { echo $broj; echo "<br>"; $broj++; } while ($broj <= 5); ?>

For petlja


For petlja ponavlja određene naredbe više puta (obično su vrednosti promenljivih menjaju u svakoj iteraciji) i koristimo je kada unapred znamo koliko puta će se izvršiti zadate naredbe. Unutar for naredbe zadaju se tri izraza.
  • Prvi izraz se izvršava samo jednom, pre početka petlje. Služi za inicijalizaciju brojača
  • Drugi izraz se izvršava pre svake iteracije. Proverava se brojač i ako je vrednost izraza true, petlja se nastavlja, a U suprotnom se prekida.
  • Treći izraz se izvršava posle svake iteracije i menja se brojač.

for (početni_izraz; uslov; izraz_posle_iteracije) { naredbe }
Brojeve od 1 do 5 možemo da ispišemo i pomoću for petlje: <?php for ($i = 1; $i <= 5; $i++) { echo $i; echo "<br>"; } ?>

Pomoću for petlje možemo da prođemo kroz numerisani niz i ispišemo njegove elemente: <?php $kursevi = array("HTML", "CSS", "PHP", "Laravel", "Spring"); $duzina_niza = count($kursevi); for ($i = 0; $i<$duzina_niza; $i++) { echo $kursevi[$i]; echo "<br>"; } ?>

Foeach petlja


Ova petlja služi samo za prolazak kroz asocijativne nizove jer imaju specifičnu ključ-vrednost strukturu. Možemo je koristiti u dva oblika.
U prvom obliku promenljiva dobija vrednost jednog elementa. foreach ($niz as $vrednost) { deo programa koji radi sa elementom koji ima vrednost $vrednost }
Kada želimo da menjamo elemente niza ili da znamo pod kojim ključem se nalazi određena vrednost, koristimo sledeći oblik: foreach ($niz as $kljuc => $vrednost) { naredbe }
Setimo se našeg niza košarkaša. Sada ćemo da ispišemo njegove elemente: <?php $kosarkasi = array("Teodosić"=>"1", "Bogdanović"=>"2", "Jokić"=>"3"); foreach ($kosarkasi as $kljuc => $vrednost) { echo "Kljuc = " . $kljuc . ", Vrednost = " . $vrednost; echo "<br>"; } ?>

Napisati program koji ispisuje parne brojeve između 1 i 10 pomoću for petlje.
