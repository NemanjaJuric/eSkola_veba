Operatori


Operatori su oznake operacija koje se izvode (sabiranje, deljenje, upoređivanje...), a operandi su vrednosti sa kojima se radi (konstante, promenljive, pozivi funkcija). Operatori se dele u više grupa:
  • Aritmetički oparatori
  • Operatori dodele
  • Operatori poređenja
  • Operatori uvećanja i umanjenja
  • Logički operatori
  • Opratori nad niskama
  • Operatori nad nizovima
  • Bitovski operatori
  • Operator izbora

Aritmetički operatori


Ovi operatori deluju na numeričke vrednosti i koriste se za izvođenje aritmetičkih operacija.

Operator Naziv
+ Sabiranje
- Oduzimanje
* Množenje
/ Deljenje
% Ostatak pri celobrojnom deljenju
** Stepenovanje

Operatori dodele


Operatori dodele deluju na numeričke vrednosti. Osnovni operator dodele je "=".

Operator Značenje
= $x=$y
+= $x=$x+$y
-= $x=$x-$y
*= $x=$x*$y
/= $x=$x/$y
%= $x=$x%$y

Operatori poređenja


Ovi operatori se koriste za poređenje dve vrednosti.

Operator Rezultat
== True ako su vrednosti koje se porede jednake
=== True ako su vrednosti koje se porede jednake i istog tipa
!= True ako su vrednosti koje se porede različite
<> True ako su vrednosti koje se porede različite
!== True ako su vrednosti koje se porede različite ili različitog tipa
> True ako je desni operand veći od levog
< True ako je desni operand manji od levog
>= True ako je desni operand veći ili jednak od levog
<= True ako je desni operand manji ili jednak od levog

Operatori uvećanja i umanjenja


Operatori uvećanja (inkrementacije) i umanjenja (dekrementacije) su unarni operatori. Uvećavaju (umanjuju) vrednost operanda za jedan.

Primer Značenje
++$x Prefiksno uvećanje $x
--$x Prefiksno umanjenje $x
$x++ Postfiksno uvećanje $x
$x-- Postfiksno umanjenje $x

Kod prefiksnog uvećanja (umanjenja) se vrednost promenljive prvo uveća (umanji), a onda takva vrednost učestvuje u izrazu. Kod postfiksnog uvećanja (umanjenja) se u izrazu koristi originalna vrednost promenljive, a nakon izračunavanja izraza se vrednost promenljive menja.

Logički operatori


Logičke operatore najčešće koristimo kada ispitujemo više uslova odjednom.

Operator Značenje
! Vraća true ako je operand netačan (negacija)
&& Vraća true ako su oba operanda tačna (konjunkcija)
|| Vraća true ako je barem jedan operand tačan (disjunkcija)
and Vraća true ako su oba operanda tačna (konjunkcija), ima slab prioritet
or Vraća true ako je barem jedan operand tačan (disjunkcija), ima slab prioritet
xor Vraća true ako je tačan samo jedan operand (ekskluzivna disjunkcija), ima slab prioritet

Opratori nad niskama


Postoje dva operatora koja deluju na niske.

Operator Značenje
. Spaja dve niske (konkatenacija)
.= Nadovezuje desni operand na levi ($x.=$y ili $x=$x.$y)

Operatori nad nizovima


Ovi operatori se koriste za rad sa nizovima.

Operator Značenje
+ Unija dva niza
== Vraća true ako nizovi imaju iste ključ/vrednost parove
=== Vraća true ako nizovi imaju iste ključ/vrednost parove na istim pozicija koji su istog tipa
!= Vraća true ako su nizovi različiti
<> Vraća true ako su nizovi različiti
!== Vraća true ako nizovi nisu identični

Bitovski operatori


Bitovski operatori se koriste u radu sa bitovima i tretiraju 1 kao true, a 0 kao false.

Operator Značenje
~ Negacija
& Konjunkcija
| Disjunkcija
<< Šiftovanje ulevo
>> Šiftovanje udesno

Operator izbora


Ovo je jedini ternarni operator (povezuje tri operanda). $x ? $y : $z Ako je $x tačno (true), izraz ima vrednost $y, a u suprotnom ima vrednost $z.

Primer: <?php $a=3; $b=11; echo $a++; echo "<br>"; echo --$b; ?>

Izmeniti prethodni primer tako da prvo ispisuje zbir promenljivih $a i $b, a zatim umanjuje vrednost promenljive $b i ispisuje njenu novu vrednost. Koristiti operator dekrementacije.
