Izmene i brisanje podataka u tabeli

Do sada smo videli kako unosimo nove podatke u tabelu i kako pretražujemo tabele. Moguće je promeniti određene podatke u već unetom redu u tabeli.

Naredba UPDATE

Sintaksa naredbe UPDATE: UPDATE ime_tabele SET ime_kolone1 = nova_vrednost1, ime_kolone2 = nova_vrednost2, ... , ime_koloneN = nova_vrednostN WHERE uslov; Vrednosti menjamo tako što koloni "dodeljujemo" novu vrednost. Iza ključne reči WHERE nam je potreban uslov da bismo promenili podatke tačno određenim redovima u tabeli, a ne da upišemo novu vrednost u svakom redu označene kolone.

Hajde da radnici Ivoni Jankovic povećamo platu na 41000. UPDATE radnik SET plata = 41000 WHERE ime = 'Ivona' AND prezime = 'Jankovic';
Da biste videli da li je upit uspešno izvršen, morate izlistati ceo sadržaj tabele pomoću upita: SELECT * FROM radnik;.
Sledećim upitom menjamo podatke u više redova. Potrebno je svim šefovima prodavnica promeniti položaj u rukovodilac prodavnice. UPDATE radnik SET polozaj = 'rukovodilac prodavnice' WHERE polozaj = 'sef prodavnice';
Svim vozačima postaviti istu platu od 65000.

Naredba DELETE

Ako je potrebno izbrisati jedan ili više redova iz tabele koristimo naredbu DELETE.
Sintaksa naredbe DELETE: DELETE FROM ime_tabele WHERE uslov; Sledećim upitom brišemo sve radnike koji rade kao vozači. DELETE FROM radnik WHERE polozaj = 'vozac';
