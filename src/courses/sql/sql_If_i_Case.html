Naredbe IF i CASE

Najčešće korišćene naredbe za upravljanje tokom izvršavanja komandi su IF i CASE, koje su veoma slične naredbama if i switch iz većine programskih jezika.

Naredba IF

Sintaksa naredbe IF:
IF(uslov, rez1, rez2): Ako je logička vrednost izraza uslov tačna, onda se izvršava izraz rez1, u suprotnom izvršava se izraz rez2.
Kako uraditi naredni zadatak?
Ispisati imena i prezimena svih radnika i, ako radnik ima platu veću od 55000 ispisati "ispod proseka", u suprotnom ispisati "iznad proseka".
Za rešavanje ovih zadataka, korisna je IF naredba. Analizirajmo naredni upit. SELECT ime, prezime, IF(plata < 55000, 'ispod proseka', 'iznad proseka') FROM radnik;
U sledećem primeru se ispisuje samo ime onih radnika koji imaju platu manju od 55000, a ostalim radnicima se ispisuje samo prezime. Dodatno, za svakog radnika ispisati radno mesto gde radi. SELECT IF(plata < 55000, ime, prezime), radno_mesto FROM radnik;

Naredba CASE

Ova naredba je slična naredbi IF, samo što kod nje imamo više rezultujućih mogućunosti.
Sintaksa naredbe CASE: CASE WHEN uslov1 THEN rezultat1 WHEN uslov2 THEN rezultat2 . . . WHEN uslovN THEN rezultatN ELSE rezultatN+1 END
Radi preglednosti, naredba CASE je pisana u više redova, ali kao što smo već rekli, svi upiti koji se pišu na ovoj platformi moraju biti u jednom redu. Svi naredni upiti koje budemo pisali će biti u jednom redu.
Naredba CASE je povoljna za rešavanje problema sličnih narednom zadatku.
Ispisati imena i prezimena svih radnika i, ako rade u Novom Sadu ispisati "Vojvodina", ako rade u Beogradu - "Glavni grad" i ako rede u Nišu - "Jug Srbije". SELECT ime, prezime, CASE WHEN radno_mesto = 'Novi Sad' THEN 'Vojvodina' WHEN radno_mesto = 'Beograd' THEN 'Glavni grad' ELSE 'Jug Srbije' END AS region FROM radnik;
Napisati upit kojim se pored imena i prezimena radnika ispisuje: "ispod proseka" ako ima platu manju od 50000; "prosek" ako ima platu od 50000 do 60000; "iznad proseka" ako mu je plata između 60000 i 110000 i "visoko iznad proseka" ako ima platu veću od 110000.
