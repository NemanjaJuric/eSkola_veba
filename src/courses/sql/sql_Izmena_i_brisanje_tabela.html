Upiti za izmenu i brisanje tabela

Ako smo napravili neku tabelu u bazi, pa smo naknadno primetili da moramo da je malo promenimo, to možemo učiniti pomoću sledećih upita.

Upit za dodavanje kolone u tabelu:
ALTER TABLE ime_postojece_tabele ADD ime_kolone tip_podatka;

U prethodnoj lekciji smo napravili tabelu osoba sa kolonoma jmbg, ime, prezime, godine. Sledeći upit toj tabeli dodaje kolonu nadimak:
ALTER TABLE osoba ADD nadimak VARCHAR(20);

Upit za brisanje kolone iz tabele:
ALTER TABLE ime_postojece_tabele DROP ime_kolone;

Iskoristimo opet tabelu osoba iz prethodne lekcije. Sledećim upitom brišemo kolonu prezime iz tabele:
ALTER TABLE osoba DROP prezime;

Upit za promenu tipa podatka u koloni tabele:
ALTER TABLE ime_postojece_tabele MODIFY COLUMN ime_kolone tip_podatka;

U tabeli osoba iz prethodne lekcije, tip podatka kolone godine iz INT promenimo u YEAR:
ALTER TABLE osoba MODIFY COLUMN godine YEAR;

Upit za brisanje tabele iz baze:
DROP TABLE ime_tabele;
