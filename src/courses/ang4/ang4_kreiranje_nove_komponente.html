Angular Komponente

Kreiranje nove komponente





Selektor druge komponente se ne dodaje u datoteci "index.html", već u datoteci "glavna.komponenta.html". Napraviti novi direktorijum i nazvati ga "zdravo-svete". U direktorijumu kreirati datoteke: "zdravo-svete.komponenta.ts", "zdravo-svete.komponenta.html" i "zdravo-svete.komponenta.css".
U datoteci "zdravo-svete.komponenta.ts" definisati klasu export class ZdravoSveteKomponenta { } Da bi radni okvir razumeo da se radi o komponenti, a ne običnoj klasi, dodati dekorator @Component({}). Pošto TypeScript ne sadrži ovaj dekorator, potrebno je uvesti ga iz biblioteke @angular/core.
Sada treba konfigurisati komponentu pomoću parametara "selector" i "template": import { Component } from "@angular/core"; @Component({ selector: 'zdravo-svete', template: `<h1>Zdravo svete iz Angular komponente!</h1>` }) export class ZdravoSveteKomponenta { } Da bi komponenta bila vidljiva, treba je uvesti u datoteku app.module.ts i deklarisati je.
Napomena: Pomoću Angular CLI kreiranje komponenti je jednostavno, kucajući u konzoli sledeći kod
ng generate component ime-komponente (ili skraćeno ng g c ime-komponente)
Zadatak:Kreirati 5 Angular komponenti: ZaglavljeKomponenta, LetoviKomponenta, ListaLetovaKomponenta, DetaljiOLetuKomponenta i UpravljanjeListomLetovaKomponenta.
Zatim kreirati direktorijume zaglavlje i letovi koji treba da se nalaze u app direktorijumu.
U direktorijumu letovi kreirati dva direktorijuma: lista-letova i detalji-o-letu, a u direktorijumu lista-letova kreirati direktorijum upravljanje-listom-letova.
Napomena:Deklarisati i uvesti sve komponente u glavni modul.
