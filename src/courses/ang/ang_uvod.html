Uvod u Angular i ucitavanje biblioteke sa CDN



AngularJS je JavaScript framework. Napisan je kao biblioteka JavaScript-a.
AngularJS se distribuiše kao JavaScript fajl, i dodaje se na web stranu putem taga script:



AngularJS proširuje HTML sa ng-direktivom
ng-app direktiva definiše AngularJS aplikaciju.
ng-model vezuje vrednosti HTML kontrola (nput, select, textarea) sa podacima aplikacije.
ng-bind vezuje podatke aplikacije sa HTML prikazom.
<!DOCTYPE html> <html> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script> <body> <div ng-app=""> <p>Name: <input type="text" ng-model="name"></p> <p ng-bind="name"></p> </div> </body> </html>
Primetimo da na početku svakog dokumenta u okviru taga script moramo ukljuciti biblioteku.
