AngularJS rutiranje



ngRoute modul pomaže vašoj aplikaciji da postane jednostranična aplikacija.

Šta je rutiranje u angularJS



Ako želite da se krećete na različitim stranama u vašoj aplikaciji, ali takođe želite da aplikacija bude SPA (jednostranična aplikacija), bez učitavanja stranice, možete koristiti ngRoute modul.

ngRoute modul vodi vašu aplikaciju u različite stranice bez ponovnog učitavanja cele aplikacije.

Primer:
Prebaciti na "red.htm", "green.htm" i "blue.htm".
<!DOCTYPE html> <html> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script> <script src="//ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular-route.js"></script> <body ng-app="myApp"> <p><a href="#/">Glavna</a></p> <a href="#red">Crvena</a> <a href="#green">Zelena</a> <a href="#blue">Plava</a> <div ng-view></div> <script> var app = angular.module("myApp", ["ngRoute"]); app.config(function($routeProvider) { $routeProvider .when("/", { templateUrl : "courses/ang/main.htm" }) .when("/red", { templateUrl : "courses/ang/red.htm" }) .when("/green", { templateUrl : "courses/ang/green.htm" }) .when("/blue", { templateUrl : "courses/ang/blue.htm" }); }); </script> <p>Kliknite na linkove da biste se prebacili na "red.htm", "green.htm", "blue.htm", ili povratak na "main.htm"</p> </body> </html>

Šta vam je potrebno?



Da bi vase aplikacije bile spremne spremne za ruting, morate ukljuciti AngularJS Route modul:
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular-route.js"></script>
Onda morate dodati ngRoute kao zavisnost u aplikaciji modula:
var app = angular.module("myApp", ["ngRoute"]);
Sada vaša aplikacija ima pristup route modulu, koji obezbeđuje $routeProvider.
Koristite $routeProvider da podesite različite rute u svojoj aplikaciji:
app.config(function($routeProvider) { $routeProvider .when("/", { templateUrl : "courses/ang/main.htm" }) .when("/red", { templateUrl : "courses/ang/red.htm" }) .when("/green", { templateUrl : "courses/ang/green.htm" }) .when("/blue", { templateUrl : "courses/ang/blue.htm" }); });

Gde to ide?



Vaša aplikacija treba u kontejner da stavi sadržaj prosleđen preko rutinga.
Ovaj kontejner je ng-view direktiva.
Postoje tri različita načina da se uključi ng-view direktiva u vašu aplikaciju:
Primer 1:
<div ng-view></div>
Primer 2:
<ng-view></ng-view>
Primer 3:
<<div class="ng-view"></div>
Aplikacije mogu imati samo jednu ng-view direktivu i to će uvek biti čuvar za sve prikaze obezbđene putanjom.

$routeProvider



Sa $routeProvider možete definisati šta strana da prikazuje kada korisnik klikne na link.
Primer:
Definišite $routeProvider:
<!DOCTYPE html> <html> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular-route.js"></script> <body ng-app="myApp"> <p><a href="#/">Glavna</a></p> <a href="#london">Grad 1</a> <a href="#paris">Grad 2</a> <p>Kliknite na linkove da pročitate nešto o Londonu i Parizu.</p> <div ng-view></div> <script> var app = angular.module("myApp", ["ngRoute"]); app.config(function($routeProvider) { $routeProvider .when("/", { templateUrl : "courses/ang/main.htm" }) .when("/london", { templateUrl : "courses/ang/london.htm" }) .when("/paris", { templateUrl : "courses/ang/paris.htm" }); }); </script> </body> </html>

Definišite $routeProvider koristeći config metodu vase aplikacije. Rad registrovan u config metodi obaviće se kada se aplikacija bude učitavala.

Kontroleri



Sa $routeProvider možete takođe definisati kontrolera za svaki "pogled".
Primer:
Dodaj kontrolere:
<!DOCTYPE html> <html> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular-route.js"></script> <body ng-app="myApp"> <p><a href="#/">Glavna</a></p> <a href="#london">Grad 1</a> <a href="#paris">Grad 2</a> <p>Kliknite na linkove.</p> <p>Imajte na umu da svaki "pogled" ima svog kontrolera koji svakom daje "msg" promenljivu vrednost.</p> <div ng-view></div> <script> var app = angular.module("myApp", ["ngRoute"]); app.config(function($routeProvider) { $routeProvider .when("/", { templateUrl : "courses/ang/main.htm", }) .when("/london", { templateUrl : "courses/ang/london.htm", controller : "londonCtrl" }) .when("/paris", { templateUrl : "courses/ang/paris.htm", controller : "parisCtrl" }); }); app.controller("londonCtrl", function ($scope) { $scope.msg = "Ja volim London"; }); app.controller("parisCtrl", function ($scope) { $scope.msg = "Ja volim Paris"; }); </script> </body> </html>

"london.htm" i "paris.htm" su normalne HTML datoteke, koje možete dodati AngularJS izrazu kao što bi sa bilo kojoj drugoj HTML sekciji vaše aplikacije AngularJS.
london.htm
<h1>London</h1> <h3>London je glavni grad Engleske.</h3> <p>To je najnaseljeniji grad u Velikoj Britaniji, sa gradskim područjem ima više od 13 miliona stanovnika.</p> <p>{{msg}}</p>
paris.htm
<h1>Paris</h1> <h3>Paris je glavni grad Francuske.</h3> <p>Područje Pariz je jedano od najvećih centara u Evropi, sa više od 12 miliona stanovnika.</p> <p>{{msg}}</p>

Šablon



U prethodnim primerima smo koristili templateUrl imovinu u $routeProvider.when metodi.
Takodje možete koristiti template imovinu, koja vam omogucava da napišete HTML direktno u vrednost imovine i ne odnosi se na stranicu.
Primer:
Napišite šablone:
<!DOCTYPE html> <html> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular-route.js"></script> <body ng-app="myApp"> <p><a href="#/">Glavna</a></p> <a href="#banana">Banana</a> <a href="#tomato">Paradajz</a> <p>Kliknite na linkove da promenite sadržaj.</p> <p>HTML prikazan u direktivi ng-view je napisan u šablon imovine metodom $routeProvider.vhen.</p> <div ng-view></div> <script> var app = angular.module("myApp", ["ngRoute"]); app.config(function($routeProvider) { $routeProvider .when("/", { template : "<h1>Glavna</h1><p>Kliknite na linkove da promenite ovaj sadržaj.</p>" }) .when("/banana", { template : "<h1>Banana</h1><p>Banane sadrže oko 75% vode.</p>" }) .when("/tomato", { template : "<h1>Paradajz</h1><p>Paradajz sadrži oko 95% vode.</p>" }); }); </script> </body> </html>

Otherwise metod



U prethodnim primerima smo koristili when metod od $routeProvider.
Takođe možete da koristite otherwise metod, koji je podrazumevana ruta kada niko drugi ne dobije meč.
Primer:
Ako "banana" kao ni "Paradajz" link nije kliknut, pokazati da se zna:
<!DOCTYPE html> <html> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular-route.js"></script> <body ng-app="myApp"> <p><a href="#/">Glavna</a></p> <a href="#banana">Banana</a> <a href="#tomato">Paradajz</a> <p>Kliknite na linkove da promenite ovaj sadržaj.</p> <p>Koristite "otherwise" metod za definisanje šta će se prikazivati kada ne kliknete na nijedan od linkova.</p> <div ng-view></div> <script> var app = angular.module("myApp", ["ngRoute"]); app.config(function($routeProvider) { $routeProvider .when("/banana", { template : "<h1>Banana</h1><p>Banane sadrže oko 75% vode.</p>" }) .when("/tomato", { template : "<h1>Paradajz</h1><p>Paradajz sadrži oko 95% vode.</p>" }) .otherwise({ template : "<h1>Ništa</h1><p>Ništa nije selktovano.</p>" }); }); </script> </body> </html>

Probajte sami:
Napisati program koji prilikom klika na neki od linkova gradova prikazuje opis tog grada,prilikom čega je boja pozadine teksta za svaki grad različita.
