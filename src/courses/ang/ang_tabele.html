Tabele

Direktiva ng-repeat je savršena za prikazivanje tabela.

Prikazivanje podataka u tabeli

Prikazivanje tabela u Angularu je veoma jednostavno:
<div ng-app="mojaAplikacija" ng-controller="mojKontroler"> <table> <tr ng-repeat="x in imena"> <td>{{ x.Ime }}</td> <td>{{ x.Zemlja}}</td> </tr> </table> </div> <script> var app = angular.module('mojaAplikacija', []); app.controller('mojKontroler', function($scope) { $scope.imena = [ { "Ime" : "Pera Perić", "Grad" : "Beograd", "Zemlja" : "Srbija" }, { "Ime" : "John Doe", "Grad" : "New York", "Zemlja" : "USA" }, { "Ime" : "Koniglich", "Grad" : "Barcelona", "Zemlja" : "Španija" }, { "Ime" : "Wolski", "Grad" : "Arhus", "Zemlja" : "Danska" } ]; }); </script>

Prikazivanje u CSS stilu

Da bi ucinili tabelu lepsom dodacemo malo CSS-a: <style> table, th , td { border: 1px solid grey; border-collapse: collapse; padding: 5px; } table tr:nth-child(odd) { background-color: #f1f1f1; } table tr:nth-child(even) { background-color: #ffffff; } </style>

Sortiranje tabele sa OrderBy filterom

Da bi sortitali tabelu dodajemo OrderBy filter. <table> <tr ng-repeat="x in imena| orderBy : 'Country'"> <td>{{ x.Ime}}</td> <td>{{ x.Zemlja }}</td> </tr> </table>

Prikazivanje sa Uppercase filterom

Da bi smo prikazali sa svim velikim slovima, koristimo Uppercase filter: <table> <tr ng-repeat="x in imena"> <td>{{ x.Ime }}</td> <td>{{ x.Zemlja| uppercase }}</td> </tr> </table>

Prikazivanje indeksa tabele

Da bi prikazali indeks tabele, dodajemo sa $index: <table> <tr ng-repeat="x in imena"> <td>{{ $index + 1 }}</td> <td>{{ x.Ime }}</td> <td>{{ x.Zemlja }}</td> </tr> </table>

Korišcenje $even i $odd

<table> <tr ng-repeat="x in imena"> <td ng-if="$odd" style="background-color:#f1f1f1">{{ x.Name }}</td> <td ng-if="$even">{{ x.Ime }}</td> <td ng-if="$odd" style="background-color:#f1f1f1">{{ x.Country }}</td> <td ng-if="$even">{{ x.Zemlja }}</td> </tr> </table>
