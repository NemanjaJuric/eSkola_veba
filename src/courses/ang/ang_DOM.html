AngularJS HTML DOM

AngularJS ima direktive za povezivanje podataka aplikacije u atribute HTML DOM elemenata.

ng-disabled direktiva

ng-disabled direktiva povezuje podatke AngularJS aplikacije sa iskljucenim atributima HTML elemenata.
<div ng-app="" ng-init="mySwitch=true"> <p> <button ng-disabled="mojPrekidac">klikni me!</button> </p> <p> <input type="checkbox" ng-model="mojPrekidac">Dugme </p> <p> {{ mojPrekidac }} </p> </div> Objasnjenje aplikacije:
ng-disabled direktiva povezuje podatak aplikacije MySwitch sa onemogucenim atributom HTML dugmeta.
ng-model direktiva povezuje vrednost elementa HTML cekboksa sa vrednoscu MySwitch.
Ako vrednost mySwitch elementa bude true, dugme se onemogucava:
<p> <button disabled>Klikni me!</button> </p> Ako vrednost bude false, dugme nece biti onemoguceno:
<p> <button>klikni me!</button> </p>

ng-show direktiva

Direktiva ng-show prikazuje ili sakriva HTML elemente.
<div ng-app=""> <p ng-show="true">Vidim se.</p> <p ng-show="false">Ne vidim se.</p> </div> ng-show direktiva pokazuje(ili skriva) HTML element na osnovu vrednosti ng-show. Mozete koristiti bilo koji izraz cija je vrednost true ili false: <div ng-app="" ng-init="sat=13"> <p ng-show="sat > 12">Vidim se.</p> </div>

ng-hide

ng-hide direktiva sakriva ili pokazuje HTML element. <div ng-app=""> <p ng-hide="true">Ne vidim se.</p> <p ng-hide="false">Vidim se.</p> </div>
