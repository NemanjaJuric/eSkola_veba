Linija <line>

Etiketa <line> se koristi kako bi se kreirala linija.
<svg height = "210" width = "500"> <line x1 = "0" y1 = "0" x2 = "200" y2 = "200" style = "stroke : rgb(0, 255, 0); stroke-width : 2" /> </svg>
Prethodni kod daje sledeću sliku:

Objašnjenje koda:
  • Atribut x1 definiše x koordinatu početka linije
  • Atribut y1 definiše y koordinatu početka linije
  • Atribut x2 definiše x koordinatu kraja linije
  • Atribut y2 definiše y koordinatu kraja linije

Zadatak: Kreirati liniju koja spaja tačke (0, 0) i (100, 50) širine 7.
