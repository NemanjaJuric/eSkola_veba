SVG efekti zamagljivanja

<defs> i <filter>

Svi SVG filteri na internetu su definisani sa etiketom <defs>. Etiketa <defs> je skraćenica od definicije i sadrži definiciju specijalne etikete (poput filtera). Etiketa <filter> se koristi da definiše SVG filter. Etiketa <filter> ima obavezni atribut id koji identifikuje filter. Grafika tada upućuje na filter koji se treba koristiti.
Etiketa <feGaussianBlur>
Etiketa <feGaussianBlur> se koristi kako bi se napravio efekat zamagljivanja. <svg height = "110" width = "110"> <defs> <filter id = "f1" x = "0" y = "0"> <feGaussianBlur in = "SourceGraphic" stdDeviation = "15" /> </filter> </defs> <rect width = "90" height = "90" stroke = "green" stroke-width = "3" fill = "purple" filter = "url(#f1)" /> </svg>

Objašnjenje koda:
  • Atribut id etikete <filter> definiše jedinstveno ime za filter
  • Zamagljeni efekat je definisan etiketom <feGaussianBlur>
  • Deo in = "SourceGraphic" definiše da je efekat kreiran za ceo element
  • Atribut stdDeviation definiše nivo zamagljenja
  • Atribut filter etikete <rect> povezuje element sa filterom f1

Zadatak: Kreirati krug prečnika 40 čija je unutrašnjost crvene boje i podesiti efekat zamagljenja na 20.
