SVG svojstvo stroke

SVG nudi širok spektar svojstva stroke. U ovoj lekciji opisaćemo sledeće:
  • stroke
  • stroke-width
  • stroke-linecap
  • stroke-dasharray
Sva stroke svojstva se mogu primeniti na bilo koju vrstu linija, teksta i spoljašnje linije drugih elemenata.

Svojstvo stroke

Svojstvo stroke definiše boju linije, teksta ili spoljašnje linije nekog elementa: <svg height = "80" width = "300"> <g fill = "none"> <path stroke = "red" d = "M5 20 l215 0" /> <path stroke = "black" d = "M5 40 l215 0" /> <path stroke = "blue" d = "M5 60 l215 0" /> </g> </svg>

Svojstvo stroke-width

Svojstvo stroke-width definiše debljinu linije, teksta ili spoljašnje linije nekog elementa: <svg height = "80" width = "300"> <g fill="none" stroke="black"> <path stroke-width = "2" d = "M5 20 l215 0" /> <path stroke-width = "4" d = "M5 40 l215 0" /> <path stroke-width = "6" d = "M5 60 l215 0" /> </g> </svg>

Svojstvo stroke-linecap

Svojstvo stroke-linecap definiše različite tipove krajeva otvorenih putanja: <svg height = "80" width = "300"> <g fill= "none" stroke = "black" stroke-width = "6"> <path stroke-linecap = "butt" d = "M5 20 l215 0" /> <path stroke-linecap = "round" d = "M5 40 l215 0" /> <path stroke-linecap = "square" d = "M5 60 l215 0" /> </g> </svg>

Svojstvo stroke-dasharray

Svojstvo stroke-dasharray se koristi za kreiranje isprekidanih linija: <svg height = "80" width = "300"> <g fill = "none" stroke = "black" stroke-width = "4"> <path stroke-dasharray = "5, 5" d = "M5 20 l215 0" /> <path stroke-dasharray = "10, 10" d = "M5 40 l215 0" /> <path stroke-dasharray = "20, 10, 5, 5, 5, 10" d = "M5 60 l215 0" /> </g> </svg>
