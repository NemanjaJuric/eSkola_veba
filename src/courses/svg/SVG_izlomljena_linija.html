Izlomljena linija <polyline>

Etiketa <polyline> se koristi kako bi se kreirao oblik koji sadrži prave linije povezane u nekoliko tačaka.
<svg height = "200" width = "500"> <polyline points = "20, 20 40, 25 60, 40 80, 120 120, 140 200, 180" style = "fill : none; stroke : purple; stroke-width : 5" /> </svg>
Prethodni kod daje sledeću sliku:

Objašnjenje koda:
  • Atribut points definiše listu tačaka (parova x i y koordinata) u kojima se spajaju prave linije kako bi se nacrtala jedna izlomljena linija

Zadatak: Kreirati izlomljenu liniju kao na slici.
