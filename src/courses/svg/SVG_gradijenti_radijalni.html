Radijalni gradijent - <radialGradient>

Etiketa <radialGradient> se koristi za definisanje radijalnog gradijenta. Etiketa <radialGradient> mora biti ugnježdena u etiketama <defs>. Etiketa <defs> je skraćenica od definicije i sadrži definiciju specijalnog elementa (poput gradijenta).
Definišemo krug sa radijalnim gradijentom od ljubičaste ka sivoj: <svg height = "200" width = "500"> <defs> <radialGradient id = "grad1" cx = "50%" cy = "50%" r = "50%" fx = "50%" fy = "50%"> <stop offset = "0%" style = "stop-color : grez; stop-opacity : 0" /> <stop offset = "100%" style = "stop-color : purple; stop-opacity : 1" /> </radialGradient> </defs> <circle cx = "200" cy = "95" r = "85" fill = "url(#grad1)" /> </svg>

Objašnjenje koda:
  • Atribut id etikete <radialGradient> definiše jedinstveno ime gradijenta
  • Atributi cx, cy, r etikete <radialGradient> definišu najudaljeniji krug, a fx i fy definišu najdublji krug
  • Opseg boja gradijenta može biti sastavljen od dve ili više boja. Svaka boja je određena sa etiketom <stop>. Atribut offset se koristi da definiše gde boja gradijenta počinje i gde se završava
  • Atribut fill povezuje krug sa gradijentom

Definišemo drugi krug sa radijalnim gradijentom od plave do bele: <svg height = "200" width = "500"> <defs> <radialGradient id = "grad2" cx = "20%" cy = "30%" r = "30%" fx = "50%" fy = "50%"> <stop offset = "0%" style = "stop-color : rgb(255, 255, 255); stop-opacity : 0" /> <stop offset = "100%" style = "stop-color : rgb(0, 0, 255); stop-opacity : 1" /> </radialGradient> </defs> <circle cx = "200" cy = "95" r = "85" fill = "url(#grad2)" /> </svg>
