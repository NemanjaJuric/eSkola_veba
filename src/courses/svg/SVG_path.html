Putanja ili trajektorija <path>

Etiketa <path> se koristi da se definiše putanja.
Sledće komande su dostupne za crtanje putanje:
  • M (moveto) = pomeriti u
  • L (lineto) = linija do
  • H (horizontal lineto) = horizontalna linija do
  • V (vertical lineto) = vertikalna linija do
  • C (curveto) = kriva do
  • S (smooth curveto) = glatka kriva do
  • Q (quadratic Bézier curve) = kvadratne Bezjeove krive
  • T (smooth quadratic Bézier curveto) = glatke kvadratne Bezjeove krive do
  • A (elliptical Arc) = eliptični luk
  • Z (closepath) = zatvorena putanja

Napomena: Sve komande iznad mogu se pozivati i sa malim slovom. Velika slova označavaju apsolutno pozicioniranje, a mala slova relativno pozicioniraje.

Primer ispod definiše putanju koja počinje u poziciji (150, 0) sa linijom do pozicije (75, 200), zatim odatle, linijom do pozicije (225, 200) i konačno se zatvara u početnoj pozociji: <svg height = "210" width = "400"> <path d = "M150 0 L75 200 L225 200 Z" /> </svg>

Bezjeove krive

Bezjeove krive se koriste za glatke krive koje se mogu skalirati u nedogled. Generalno, korisnik označi 2 krajnje tačke i jednu kontrolnu tačku. Bezjeove krive sa jednom kontrolnom tačkom se zovu kvadratne Bezjeove krive, sa dve kontrolne tačke zovu se kubne. Sledeći primer kreira kvadratnu Bezjeovu krivu, gde su A i C početna i krajnja tačka, a B je kontrolna tačka: <svg height = "400" width = "450"> <path id = "lineAB" d = "M 100 350 l 150 -300" stroke = "red" stroke-width = "3" fill = "none" /> <path id = "lineBC" d = "M 250 50 l 150 300" stroke = "red" stroke-width = "3" fill = "none" /> <path d = "M 175 200 l 150 0" stroke = "green" stroke-width = "3" fill = "none" /> <path d = "M 100 350 q 150 -300 300 0" stroke = "blue" stroke-width = "5" fill = "none" /> <!-- Označavanje početne tačke A, krajnje tačke C i kontrolne tačke B --> <g stroke="black" stroke-width="3" fill="black"> <circle id = "pointA" cx = "100" cy = "350" r = "3" /> <circle id = "pointB" cx = "250" cy = "50" r = "3" /> <circle id = "pointC" cx = "400" cy = "350" r = "3" /> </g> <!-- Zadavanje oznake tačkama --> <g font-size = "30" font-family = "sans-serif" fill = "black" stroke = "none" text-anchor = "middle"> <text x = "100" y = "350" dx = "-30"> A </text> <text x = "250" y = "50" dy = "-10"> B </text> <text x = "400" y = "350" dx = "30">C</text> </g> </svg>
