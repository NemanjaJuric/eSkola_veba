Krug <circle>

Etiketa <circle> se koristi kako bi se kreirao krug.
<svg height = "100" width = "100"> <circle cx = "50" cy = "50" r = "40" stroke = "pink" stroke-width = "6" fill = "purple" /> </svg>
Prethodni kod daje sledeću sliku:

Objašnjenje koda:
  • Atributi cx i cy definišu x i y koordinate centra kruga. Ako su cx i cy izostavljene, centar kruga je postavljen u (0,0)
  • Atribut r definiše poluprečnik kruga

Zadatak: Kreirati krug prečnika 60, čija je unutrašnjost žute boje, kružna linija zelene boje, a providnost 0.4.
