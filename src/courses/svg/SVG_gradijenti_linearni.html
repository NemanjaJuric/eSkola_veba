SVG gradijenti

Gradijent je gladak prelazak sa jedne boje na drugu. Pored toga, na isti element se može naneti više prelaza boja. Postoje dva glavna SVG gradijenta
  • Linearni
  • Radijalni

Linearni gradijent - <linearGradient>

Etiketa <linearGradient> se koristi za definisanje linearnog gradijenta. Etiketa <linearGradient> mora biti ugnježdena u etiketama <defs>. Etiketa <defs> je skraćenica od definicije i sadrži definiciju specijalnog elementa (poput gradijenta).
Linearni gradijent može biti definisan kao horizontalni, vertikalni ili ugaoni gradijent:
  • Horizontalni gradijenti se kreiraju kada su y1 i y2 jednaki i x1 i x2 različiti
  • Vertikalni gradijenti se kreiraju kada su x1 i x2 jednaki i y1 i y2 različiti
  • Ugaoni gradijenti se kreiraju kada su x1 i x2 različiti i y1 i y2 različiti
Definišemo elipsu sa horizontalnim linearnim gradijentom od plave ka sivoj: <svg height = "150" width = "400"> <defs> <linearGradient id = "grad1" x1 = "0%" y1 = "0%" x2 = "100%" y2 = "0%"> <stop offset = "0%" style = "stop-color : blue; stop-opacity : 1" /> <stop offset = "100%" style = "stop-color : grey; stop-opacity : 1" /> </linearGradient> </defs> <ellipse cx = "200" cy = "70" rx = "85" ry = "55" fill = "url(#grad1)" /> </svg>
Prethodni kod daje sledeću sliku:

  • Atribut id etikete <linearGradient> definiše jedinstveno ime gradijenta
  • Atributi x1, x2, y1, y2 etikete <linearGradient> definišu početnu i krajnju poziciju gradijenta
  • Opseg boja gradijenta može biti sastavljen od dve ili više boja. Svaka boja je određena sa etiketom <stop>. Atribut offset se koristi da definiše gde boja gradijenta počinje i gde se završava
  • Atribut fill povezuje elipsu sa gradijentom

Definišemo elipsu sa vertikalnim linearnim gradijentom od plave ka sivoj i tekstom unutar elipse: <svg height = "150" width = "400"> <defs> <linearGradient id = "grad2" x1 = "0%" y1 = "0%" x2 = "0%" y2 = "100%"> <stop offset = "0%" style = "stop-color : blue; stop-opacity : 1" /> <stop offset = "100%" style = "stop-color : grey; stop-opacity : 1" /> </linearGradient> </defs> <ellipse cx = "200" cy = "70" rx = "85" ry = "55" fill = "url(#grad2)" /> <text fill = "#ffffff" font-size = "45" font-family = "Verdana" x = "150" y = "86"> SVG </text> </svg>
