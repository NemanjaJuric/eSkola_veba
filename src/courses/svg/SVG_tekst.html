Tekst <text>

Etiketa <text> se koristi da definiše tekst. Etiketa <text> može biti raspoređena u bilo kom broju podgrupa sa etiketom <tspan>. Svaka etiketa <tspan> može sadržati formatiranje i poziciju. Tekst u nekoliko linija (sa etiketom <tspan>):
<svg height = "180" width = "200"> <text x = "10" y = "20" style = "fill : red;"> Nekoliko linija: <tspan x = "10" y = "45"> Zdravo </tspan> <tspan x = "10" y = "70"> svete! </tspan> </text> <text x = "10" y = "95" fill = "blue" transform = "rotate(30 20, 80)"> Ja volim SVG!!! </text> </svg>
