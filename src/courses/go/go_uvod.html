Uvod u programski jezik Go

Go ili Golang je programski jezik opšte namene i otvorenog koda nastao u okviru kompanije Google 2007. godine. Jezik su osmislili Robert Grisemer (Robert Griesemer), Rob Pajk (Rob Pike) i Ken Tompson (Ken Thompson). Ideja je bila osmišljavanje novog, jednostavnog i moćnog jezika. Programski jezik Go je imperativni, statički tipizirani, kompajlirani jezik. Sadrži automatsko upravljanje memorijom i funkcionalnosti za konkurentno programiranje.

Go nije objektno-orijentisani jezik, ali preuzima neke od koncepata objektno-orijentisanih jezika, kao što su metodi i interfejsi. Go je namenjen prvenstveno izradi serverskih aplikacija, kao i sistemskom programiranju.
Neke od prednosti programskog jezika Go su:
  • jednostavnost i čitkost;
  • velika brzina izvršavanja;
  • dobra standardna biblioteka, kao i veliki broj korisničkih biblioteka;
  • skalabilnost.

Instalacija

Da bi se počelo sa programiranjem, prvo je neophodno preuzeti i instalirati kompajler koji se nalazi na oficijalnom sajtu jezika, a koji prevodi izvorni kod u mašinski (izvršni) kod. Osim toga, neophodan je i terminal, bilo koji terminal za Linux i Mac ili Powershell ili cmd za Windows, kao i tekst editor. Pogodni tekst editori su Visual Studio Code i Sublime Text zato što poseduju ekstenzije za označavanje sintakse, ali ima i drugih. Fajlovi sa izvornim kodom u jeziku Go imaju ekstenziju .go. Nakon kompajliranja dobija se izvršni kod sa ekstenzijom .out na Unix-u i .exe na Windows operativnim sistemima.

Moj prvi program "Zdravo svete"

Kreiranje programa obuhvata tri glavna koraka: unošenje, prevođenje (kompajliranje) i pokretanje programa.
Unošenje programa
Svaki Go program se sastoji od paketa, koji ima pridruženu putanju i ime. Uloga paketa je da grupiše određene funkcionalnosti u jednu celinu, što omogućava lakše održavanje i korišćenje. Program počinje u paketu main.
Za ispisivanje teksta neophodno je prvo uvesti paket fmt koji sadrži neophodne funkcije za rad sa ulazom i izlazom. Kao i u C programskom jeziku i Go počinje izvršavanje od funkcije main, u okviru koje se piše logika programa.
package main import ( "fmt" ) func main() { fmt.Println("Zdravo svete") }
Prevođenje programa
Da bi se izvorni kod preveo u izvršni, sve što treba da se uradi je da se u terminal unese go build. Go će automatski kreirati izvršni fajl sa ekstenzijom .exe ili .out, zavisno od operativnog sistema i imenom koje će odgovarati imenu direktorijuma u kome se fajl nalazi.
Pokretanje programa
Da bi se novonastali program pokrenuo neophodno je u terminalu uneti imefajla.exe ili imefajla.out. Drugi način za pokretanje programa objedinjuje prevođenje i izvršavanje. U terminal treba uneti go run imefajla.go, čime će se izvorni kod prevesti i pokrenuti, a rezultat izvršavanja prikazati.
