CSS selektori


Razlika između klasnih i ID selektora je u znaku koji im prethodi, naime ukoliko je znak # (taraba) u pitanju je ID selektor, u suprotnom .(tacka) označava klasni selektor.
#top{ background-color: #ccc; padding: 20px } .intro { color: red; font-weight: bold; }
Ako želimo da primenimo navedeni CSS na HTML stranicu, koristimo atribute id i class. To izgleda ovako:
<div id="top"> <h1>Zaokruživanje decimalnih brojeva</h1> <p class="intro">Kada se decimalan broj zaokružuje, gleda se prethodna cifra. Ako je prethodna cifra veca od 5, zaokružuje se na veći broj, ako je manja, zaokružuje se na manji.</p> <p class="intro">Kada je prethodna cifra 5, gleda se da li postoji jos neka cifra pre nje. Ako postoji, broj se zaokružuje na veći, ako ne, broj se zaokružuje na manji. </p> </div>
Jos jedna razlika između klasnih i ID selektora je sto se ID korisi za prepoznavanje samo jednog elementa, dok klasni za više. Možete primeniti selektor na određeni HTML element jednostavno navodeći HTML selektor prvi , tako da npr. p.naziv_klase { ... } će biti primenjen samo na elemente paragrafa koji sadrze klasu " naziv_klase" .
