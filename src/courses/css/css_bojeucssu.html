Boje u CSS-u

Boje teksta i pozadine

CSS donosi 16 777 216 boja. One mogu biti u formi:
  • imena - CSS prepoznaje 147 definisanih naziva boja. Ove boje se zadaju jednostavnim navodjenjem naziva boje.
  • RGB (red/green/blue) vrednosti - zadaju se tri cela broja, od 0 do 255, koji predstavljaju tri kolorne komponente - crvenu, zelenu i plavu, čime se mogu postići sve nijanse.
  • heksadekadnog zapisa - boju zadajemo u obliku #RRGGBB. Heksadekadni kodovi se zadaju sa vodećom "tarabom".
Sve ove vrednosti,za zadavanje crvene boje,imaju isti rezultat:
  • red
  • rgb(255,0,0)
  • #ff0000
Imena nekih boja koje možemo da koristimo: aqua, black, blue, blanchedalmond, blueviolet, chocolate, coral, darkblue, grey, green, deeppink, gold, maroon...

CSS pravilo za podešavanje boje teksta zadaje se u obliku deklaracije koja ima sledeći oblik:
color:red
Ovaj atribut ima samo tu jednu svrhu,odnosno može se koristiti isključivo za postavljanje boje slova unutar nekog HTML dokumenta.

Boje se mogu primeniti korišćenjem color i background-color (boja pozadine) na sledeći način:

Žuta pozadina i plavi tekst bi izgledali ovako:
h1{ color: blue; background-color: yellow; }

Možemo primeniti color i background-color svojstva na većinu HTML elemenata, uključujući i body, što će promeniti boju stranice i svega na njoj.
To ćemo uraditi na sledeći način:

body{ color: green; }
Zadatak: Napisati rečenicu čiji će tekst biti crvene boje,a pozadina žute.
