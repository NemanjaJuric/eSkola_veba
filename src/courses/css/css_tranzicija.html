Tranzicije


Tranzicije (prelazi) omogućavaju da dati element postepeno promeni stil u neki drugi prelaskom kursora miša preko njega. Potrebno je obratiti pažnju na sledeća svojstva:
  • transition-property – određuje na koje CSS svojstvo ćemo primeniti prelaz (inicijalna vrednost je all)
  • transition-duration – određuje trajanje prelaza
  • transition-delay – kašnjenje tj. određuje vremenski početak vršenja tranzicije
  • transition-timing-function - opisuje kako će se vrednosti menjati tj dopušta promenu brizne tokom trajanja prelaza (ease, linear, ease-in, ease-out, ease-in-out...)

Sledeće linije koda opisuju tranziciju gde prelaskom kursora miša preko elementa on promeni svoju širinu u trajanju od 4s. div { width: 150px; height: 150px; -webkit-transition: width 4s; /* Safari */ transition: width 4s; } div:hover { width: 400px; }
Napomena: U kodu sa desne strane vidimo kako prvi element menja svoju širinu i dužinu po zadatim parametrima. Naredna četiri elementa prikazuju pomenute timing-function gde u prvom od njih primećujemo kašnjenje tranzicije za 2s.
Zadatak: Izmeniti linije koda gde će pravougaonik prelaskom kursora preko njega postati kvadrat u trajanju od 5s i kašnjenjem od 1s.
