Box model, margine, uokvirenja, ivice i zakrivljene ivice


Margine i uokvirenja su dva najčešća alata za razdvajanja teksta. Margina predstavlja razmak od ivica, a uokvirenje (padding) razmak od ostalih elemenata.
h2 { font-size: 1.5em; background-color: #ccc; margin: 20px; padding: 40px; }
Ovo znači da se ostavlja prostor za margine od 20 piksela za drugo zaglavlje (h2) i uokviruje se sa 40 piksela.
Za svaku stranu elementa (gore, dole, levo i desno) mogu se pojedinačno podesiti margine i uokvirenja ( margin-top, margin-right, margin-bottom, margin-left, padding-top, padding-right, padding-bottom i padding-left).

Box model


Margine, uokvirenja i ivice su elementi Box Modela. Box Model funkcioniše na sledeći način: u sredini se nalazi oblast sadržaja, nju okružuju još tri Box-a, najširi je margina Box, uži od njega je border Box i na kraju je padding Box. Na slici to izgleda ovako:
Margina box
Border box
Padding box
Element box

Treblo bi napomenuti da nije neophodno koristiti sve Box-ove iako su jako korisni i da se ovaj model moze primeniti na svaki element na stranici.

Ivice


Ivice mogu biti primenjene na elemente unutar HTML tela.
Da bi se napravile ivice oko elementa sve sto je potrebno je border-style. Moguće vrednosti za stil su: solid, dotted, dashed, double, groove, ridge, inset i outset.
border-width-om se podešava širina ivica, najčešće korišćenjem piksela za vrednost.Naravno, postoje i sledeće opcije: border-top-width, border-right-width, border-bottom-width i border-left-width.
Na kraju, moguće je postaviti boju pomoću border-color.
h2 { border-style: dashed;//crvena isprekidana linija border-width: 3px; border-left-width: 10px; border-right-width: 10px; border-color: red; }
Ovim će gore navedena svojstva biti primenjena na sve elemente u h2 poglavlju.

Zakivljene ivice


Zakrivljene ivice nam služe da redizajniramo izgled jednog dela teksta(ili fotografije) sa unutrašnje strane. Primera radi, ako imamo neko nabrajanje i želimo da ga oivičimo samo sa unutrašnje strane (tako da spoljna ivica ostne u pravougaonom obliku), možemo napraviti razlicite oblike!

Poluprečnik zakrivljenja


border-radius svojstvo se koristi da se napravi zakrivljenje na svakoj od četiri ivice.
#marilyn { background: #fff; width: 100px; height: 100px; border-radius: 20px; }
Naravno, ukoliko želite standardni izgled, sa dodatnom ivicom, to ćete postići na sledeći način:
#ok_a_border_then { border: 5px solid #8b2; width: 100px; height: 100px; border-radius: 5px; }

Različita zakrivljenost ivica


border-top-left-radius, border-top-right-radius, border-bottom-right-radius, i border-bottom-left-radius možemo primeniti na svaku pojedinačnu ivicu.
#monroe { background: #fff; width: 100px; height: 100px; border-radius: 6px 12px 18px 24px; }
Korišćenjem samo dve vrednosti (umesto četiri), prva će se odnositi na gornju-levu i donju-desnu, a druga na ostale dve. Opet, ako imamo tri vrdnosti, prva se odnosi na gornju-levu ivicu, druga na gornju-desnu i donju-levu, a treca na preostalu.

Elipse


Mozemo napraviti i interesantnije oblike razdvajanjem horizontalnog i vertikalnog poluprečnika (radijusa), sto činimo pomoću znaka “/”.
#nanoo { background: #fff; width: 100px; height: 150px; border-radius: 50px/100px; border-bottom-left-radius: 50px; border-bottom-right-radius: 50px; }
