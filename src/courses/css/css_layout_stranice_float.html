Float

Samo po sebi float ne predstavlja šemu pozicioniranja, ali svakako jeste jedinstven dodatak ugrađen u CSS sa interesantnim osobinama. Vrednosti koje uzima float su: left, right, none i inherit. Podrazumevana vrednost je none, primenjuje se na sve elemente i ne nasleđuje se. Najbolji način za razumevanje ovog svojstva jeste preko primera.

Napomena:Važno je uvek zadati širinu tekstualnog elementa na koji primenjujete float. Liinijski elementi sa svojstvom float se ponašaju kao blok elementi i margine se ne poklapaju. Pri korišćenju float nad blok elementima mora se zadati širina ili će u suprotnom širina biti postavljena na automatsku, a to će biti najveća dostupna, samim tim će float izgubiti smisao. Ideja pri korišćenju float jeste da sledeći element počne do njega, a ne ispod, a kad element zauzme svu dostupnu širinu naredni će biti ispod njega.

Svojstvo clear isključuje float i vraća na podrazumevani raspored. Clear ima vrednosti: left, right, none, both, inherit, primenjuje se samo na blok elemente i ne nasledjuje se, podrazumevano je none. Koristi se kada hoćete da element počne ispod elementa nad kojim je primenjen float.
Zadatak: Pokušaj da, malim izmenama u kodu, koristeći float napraviš sledeći redosled:
plavi
zeleni
narandzasti, žuti, crveni
