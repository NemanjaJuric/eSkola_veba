Animacije


Animacije imaju još nekoliko zanimljivih svojstava:
  • animation-direction - opisuje kako će se animacija kretati
    • reverse - animacija se izvršava u obrnutom smeru
    • alternative - animacija se izvršava u ciklusima
    • Napomena: U kodu sa desne strane prva dva elementa opisuju dve animacije od kojih se jedna odvija u smeru sleva na desno,a druga u obrnutom smeru.
  • animation-timing-function - opisuje kako će se vrednosti elementa menjati tj. dopušta promenu brizne promene tokom trajanja prelaza
    • easy - ovo je inicijalna vrednost, animacija kreće sporo, pa brzo, zatim ponovo usporava
    • linear - brzina animacije je konstantna
    • easy-in - animacija ima spor početak
    • easy-out - animacija ima spor kraj
    • easy-in-out - animacija ima spor i početak i kraj
    • Napomena: U kodu sa desne strane su prikazani elementi koji opisuju ove funkcije (različite brzine izvršenja animacija).

Pogledajmo kako bi izgledao kod gde bismo upotrebili sva svojstva koja opisuju jednu animaciju: div { animation-name: primer; animation-duration: 4s; animation-timing-function: easy; animation-delay: 1s; animation-iteration-count: infinite; animation-direction: alternate; } Postoji skraćeni oblik zapisa ovog koda koji obuhvata sva pomenuta svojstva animacije: div { animation: primer 4s easy 1s infinite alternate; }
Zadatak:
Na kraju ovog poglavlja pokušajte da uradite zadatak u kome se istovremeno javljaju animacije,tranzicije i transformacije:
Napisati kod u kome pravougaonik menja boje i kreće se beskonačno mnogo puta po određenoj putanji (desno, dole levo, dole desno, gore levo). Kada prisloniš kursor miša na element, on postaje kvadrat.
