2D transformacije


Skaliranje


Skaliranje je metod koji povećava ili smanjuje element, odnosno njegovu širinu i visinu. Metode su:
  • skaleX(n) – promena širine elementa (n-koliko puta po X-osi)
  • skaleY(n) – promena visine elementa (n-koliko puta po Y-osi)
  • skale(n,m) – promena i širine i visine elementa (n-koliko puta po X-osi, m-koliko puta po Y-osi)

div { -ms-transform: skale(2,3); /* IE 9 */ -webkit-transform: skale(2,3); /* Safari */ transform: skale(2,3); }

Iskošenje


Ova metoda predstavlja iskošenje elementa za dati ugao i to na sledeći način:
  • skewX(angle) - iskošenje elementa po X-osi
  • skewY(angle) - iskošenje elementa po Y-osi
  • skew(angleX,angleY) - iskošenje elementa i po X i po Y-osi

div { -ms-transform: skew(40,50); /* IE 9 */ -webkit-transform: skew(40,50); /* Safari */ transform: skew(40,50); }
Postoji metod koji je kombinacija svih 2D transformacija i ima 6 parametara:
matrix(scale-x, skew-x, skew-y, scale-y, translate-x, translate-y)
Zadatak:
a) Izmeniti deo koda tako da se samo širina pravougaonika poveća 2.5 puta.
b) Izmeniti deo koda tako da se pravougaonik iskosi za 40 stepeni duž X-ose i 10 stepeni duž Y-ose.
