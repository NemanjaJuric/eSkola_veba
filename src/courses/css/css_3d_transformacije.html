3D transformacije


Za razliku od 2D transformacija sada se nalazimo u trodimenzionom koordinatnom sistemu.
Metode su slične kao i u prethodnoj lekciji:
    Translacija:
  • translate3d(x,y,z) - 3D translacija
  • translateX(x) - 3D translacija po X-osi
  • translateY(y) - 3D translacija po Y-osi
  • translateZ(z) - 3D translacija po Z-osi
    Rotacija:
  • rotate3d(angleX,angleY,angleZ) - 3D rotacija
  • rotateX(angle) - 3D rotacija oko X-ose
  • rotateY(angle) - 3D rotacija oko Y-ose
  • rotateZ(angle) - 3D rotacija oko Z-ose
div { -webkit-transform: rotateZ(60deg); /* Safari */ transform: rotateZ(60deg); }
    Skaliranje:
  • scale3d(x,y,z) - 3D skaliranje
  • scaleX(x) - 3D skaliranje po X-osi
  • scaleY(y) - 3D skaliranje po Y-osi
  • scaleZ(z) - 3D skaliranje po Z-osi
    Kombinacija 3D metoda:
  • matrix3d (n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n) - definiše 3D transformaciju pomoću 16 parametara

Zadatak:
a) Izmeniti deo koda tako da se pravougaonik rotira oko Y-ose za 250 stepeni u trodimenzionom koordinatnom sistemu.
b) Izmeniti deo koda tako da se pravougaonik rotira oko Z-ose za 120 stepeni u trodimenzionom koordinatnom sistemu.
