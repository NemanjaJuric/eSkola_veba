JQuery-Efekti

biblioteka sadrži i metode za animiranje web stranica, to su uglavnom animacije koje se najčešće primenjuju na stranice. U ovom kursu ćemo da prođemo kroz većinu Jquery metoda za animiranje.

Efekti

  • .animate() - Aktivira standarnu animaciju nad elementom. <!DOCTYPE html> <html> <head> <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.0/jquery.min.js"></script> <script> $(document).ready(function(){ $("#desno").click(function() { $("#kutija").animate( {"left": "+=50px"}, "slow"); }); $("#levo").click(function() { $("#kutija").animate( {"left": "-=50px"}, "slow"); }); }); </script> </head> <body> <p>Primer u kom uz pomoc animate metode pomeramo element levo i desno. </p> <br> <input type="button" id="levo" value="Levo"/> <input type="button" id="desno" value="Desno"/> <div id="kutija" style="background:#98bf21;height:100px;width:100px;margin:6px;position:absolute;left:100px"></div> </body> </html>
  • .delay()- Delay metoda postavlja vreme kašnjenja za aktiviranje animacije odabranog elementa. <!DOCTYPE html> <html> <head> <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.0/jquery.min.js"></script> <script> $(document).ready(function(){ $("button").click(function(){ $("#div1").delay(800).animate({width: "300px"}); $("#div2").delay("fast").fadeIn(); }); }); </script> </head> <body> <p>Primer sa različitim brzinama delay() metoda.</p> <button>Klikni za animacije sa različitim brzinama</button> <br><br> <div id="div1" style="width:90px;height:90px;background-color:black;"></div><br> <div id="div2" style="width:90px;height:90px;display:none;background-color:green;"></div><br> <div id="div3" style="width:90px;height:90px;display:none;background-color:blue;"></div><br> </body> </html>
  • fadeOut()- Aktivira animaciju u kojoj izabrani element postepeno nestane. <!DOCTYPE html> <html> <head> <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.0/jquery.min.js"></script> <script> $(document).ready(function(){ $(".btn1").click(function(){ $("p").fadeOut() }); }); </script> </head> <body> <br> <p>Primer u kom paragraf postepeno nestane.</p> <br> <br> <p style="background-color:orange; color:white">Milutin Milanković</p> <button class="btn1">Fade out</button> </body> </html>
  • .fadeIn()- Aktivira animaciju u kojoj se nestali element postepeno pojavi. <!DOCTYPE html> <html> <head> <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.0/jquery.min.js"></script> <script> $(document).ready(function(){ $(".btn1").click(function(){ $("p").fadeIn() }); }); </script> </head> <body> <br> <br> <p>Primer u kom se nestali paragraf postepeno pojavljuje.</p> <br> <br> <p style="background-color:orange; color:white; display:none"> Jovan Jovanović Zmaj</p> <button class="btn1">Fade in</button> </body> </html>
  • .fadeTo()-Aktivira animaciju promene providnosti elementa. <!DOCTYPE html> <html> <head> <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.0/jquery.min.js"></script> <script> $(document).ready(function(){ $("button").click(function(){ $("#div1").fadeTo(1000, 0.4); }); }); </script> </head> <body> <br> <p>Primer izmene providnosti div elementa.</p> <br> <br> <button>Postepeno izmeni providnost elementa.</button> <br> <br> <div id="div1" style="width:90px;height:90px;background-color:black;"></div><br> </body> </html>
  • .fadeToggle() -Aktivira i fadeIn i fadeOut metodu nad izabranim elementom. <!DOCTYPE html> <html> <head> <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.0/jquery.min.js"></script> <script> $(document).ready(function(){ $("button").click(function(){ $("#kutija1").fadeToggle(); $("#kutija2").fadeToggle("slow"); $("#kutija3").fadeToggle(3000); }); }); </script> </head> <body> <p>Primer fadeToggle() metoda sa različitim brzinama.</p> <button>Klikni da na svaku kutiju primeniš i fadeIn i fadeOut.</button><br><br> <div id="kutija1" style="width:80px;height:80px;background-color:red;"></div> <br> <div id="kutija2" style="width:80px;height:80px;background-color:green;"></div> <br> <div id="kutija3" style="width:80px;height:80px;background-color:blue;"></div> </body> </html>
  • .hide()-Aktivira animaciju u kojoj izabrani element naglo nestane. <!DOCTYPE html> <html> <head> <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.0/jquery.min.js"></script> <script> $(document).ready(function(){ $(".btn1").click(function(){ $("p").hide() }); }); </script> </head> <body> <br> <p>Primer u kome paragraf naglo nestane.</p> <br> <br> <p style="background-color:orange; color:white;"> Mika Antić</p> <button class="btn1">Sakrij element!</button> </body> </html>
  • .show() -Aktivira animaciju u kojoj se sakriveni element naglo pojavi. <!DOCTYPE html> <html> <head> <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.0/jquery.min.js"></script> <script> $(document).ready(function(){ $(".btn1").click(function(){ $("p").show() }); }); </script> </head> <body> <br> <p>Primer u kome se skriveni paragraf naglo pojavi.</p> <br> <br> <p style="background-color:orange; color:white; display:none"> Mihajlo Pupin</p> <button class="btn1">Prikaži element!</button> </body> </html>
  • .slideDown() i slideUp()-Aktivira animacije gde se element sklizne ka dole odnosno gore. <!DOCTYPE html> <html> <head> <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.0/jquery.min.js"></script> <script> $(document).ready(function(){ $(".btn1").click(function(){ $("p").slideUp(); }); $(".btn2").click(function(){ $("p").slideDown(); }); }); </script> </head> <body> <br> <p>Primer u kom paragraf nestaje "klizeći" ka gore i pojavljuje se "klizeći" ka dole .</p> <br> <br> <p style="background-color:orange; color:white"> Nikola Tesla</p> <button class="btn1">Slide up</button> <button class="btn2">Slide down</button> </body> </html>
  • .slideToggle- Aktivira i slideUp i slideDown metodu nad izabranim elementom. <!DOCTYPE html> <html> <head> <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.0/jquery.min.js"></script> <script> $(document).ready(function(){ $("button").click(function(){ $("#kutija1").slideToggle(); $("#kutija2").slideToggle("slow"); $("#kutija3").slideToggle(3000); }); }); </script> </head> <body> <p>Primer slideToggle() metoda sa različitim brzinama.</p> <button>Klikni da na svaku kutiju primeniš i slideUp i slideDown.</button><br><br> <div id="kutija1" style="width:80px;height:80px;background-color:red;"></div> <br> <div id="kutija2" style="width:80px;height:80px;background-color:green;"></div> <br> <div id="kutija3" style="width:80px;height:80px;background-color:blue;"></div> </body> </html>
  • .stop-Zaustavlja animaciju koja se trenutno primenjuje nad izabranim elementom. <!DOCTYPE html> <html> <head> <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.0/jquery.min.js"></script> <script> $(document).ready(function(){ $("#start").click(function(){ $("div").animate({width: 300}, 3000); }); $("#stop").click(function(){ $("div").stop(); }); }); </script> </head> <body> <br> <p>Primer animiranja širine div elementa klikom na dugme "Počni animaciju" i naglog zaustavljanja animacije klikom na dugme "Zaustavi animaciju".</p> <br> <br> <p> <button id="start">Počni animaciju</button> <button id="stop">Zaustavi animaciju</button> </p> <div style="background:#98bf21;height:100px;width:100px"></div> </body> </html>
