JQuery-DOM


Jedna od najvećih prednosti Jquery biblioteke jeste manipulacija DOM drveta. DOM drvo se odnosi na objektni model dokumenta (eng. Document Object Model - DOM), koji predstavjla jezik sa nezavisnom konvencijom za rad sa objektima iz HTML, XML i XHMTL dokumenata. Uz pomoć JQuery biblioteke možemo da dodamo html element našem dokumentu, da uzmemo njegovu vrednost, da mu dodelimo vrednost ili atribut i još mnogo toga. Kroz ovaj kurs ćemo da pređemo sve najvaznije JQuery funkcije za manipulaciju DOM drveta, i te funkcije možemo da svrstamo u 4 grupe:
  • JQuery funkcije za postavljanje;
  • JQuery funkcije za dohvatanje;
  • JQuery funkcije za dodavanje;
  • JQuery funkcije za uklanjanje.


JQuery funkcije za postavljanje

  • .text() -Postavlja dati testualni sadžaj izabranom elementu. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("#btn1").click(function(){ $("#test1").text("Mrtav ladan!"); }); }); </script> </head> <body> <br> <p>U ovom primeru ćemo izmeniti teksualni sadržaj paragrafa klikom na dugme "Postavi tekst". </p> <br> <br> <p id="test1">Nabroj jedan domaci film.</p> <button id="btn1">Postavi tekst</button> </body> </html>
  • .html() -Postavlja dati html zadatom elementu. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("#dugme").click(function(){ $("#paragraf").html("<b>Ko to tamo peva!</b>"); }); }); </script> </head> <body> <br> <p>U ovom primeru ćemo dodati html boldovani tekst našem paragrafu klikom na dugme "Postavi HTML". </p> <br> <br> <p id="paragraf">Nabroj jedan domaći film.</p> <button id="dugme">Postavi HTML</button> </body> </html>
  • .val() -Postavlja datu vrednost polju forme. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("#dugme").click(function(){ $("#polje").val("Nikola Pašić"); }); }); </script> </head> <body> <br> <p>U ovom primeru ćemo postaviti novu vrednost teksualnom polju klikom na dugme "Postavi novu vrednost". </p> <br> <br> <p>Polje: <input type="text" id="polje" value="Živojin Mišić"></p> <button id="dugme">Postavi novu vrednost</button> </body> </html>
  • .attr()-Postavlja dati atribut izabranom elementu. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("button").click(function(){ $("p").attr("title", "Sve znam"); }); }); </script> </head> <body> <br> <p>U ovom primeru ćemo izmeniti naslov paragrafa klikom na dugme "Promeni vrednost naslova". </p> <br> <br> <p title="Nista ne znam"> Prevuci kursor miša preko ovog teksta da vidiš naslov.</p> <button>Promeni vrednost naslova</button> </body> </html>

JQuery funkcije za uzimanje

  • .text() -Vraca sadržaj izabranog elementa. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("#dugme").click(function(){ alert("Text: " + $("#test").text()); }); }); </script> </head> <body> <p id="test">Mihajlo Pupin je bio najpoznatiji po <b>Pupinovim kalemovima</b>.</p> <button id="dugme">Prikaži tekst elementa.</button> </body> </html>
  • .html() -Vraca sadržaj izabranog elementa ukljucujući html oznake. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("#dugme").click(function(){ alert("HTML: " + $("#test").html()); }); }); </script> </head> <body> <p id="test"> Mihajlo Pupin je bio najpoznatiji po <b>Pupinovim kalemovima</b>.</p> <button id="dugme">Prikaži HTML elementa.</button> </body> </html>
  • .val() -Vraća vrednost izabranog polja. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("button").click(function(){ alert("Value: " + $("#polje").val()); }); }); </script> </head> <body> <p>Name: <input type="text" id="polje" value="Nikola Tesla"></p> <button>Prikaži vrednost polja.</button> </body> </html>
  • .attr() -Vraca vrednost navedenog atributa izabranog elementa. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("button").click(function(){ alert($("#paragraf").attr("title")); }); }); </script> </head> <body> <br> <p>U ovom primeru ćemo uzeti vrednost atributa title našeg paragrafa klikom na dugme "Klikni da saznaš naslov knjige ovog citata?" </p> <br> <br> <p title="Zločin i kazna" id="paragraf"> "Do vlasti dolazi samo onaj ko se osmjeli da sagne i uzme je. Tu se traži jedno, samo jedno: treba se odvažiti!"</p> <button>Klikni da saznaš naslov knjige ovog citata?</button> </body> </html>

JQuery funkcije za dodavanje

  • .append() -Ubacuje sadržaj na kraj izabranog elementa. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("#dugme").click(function(){ $("ol").append("<li>Pivo</li>"); }); }); </script> </head> <body> <br> <p>U ovom primeru ćemo ubaciti još jednu stavku na kraj liste klikom na dugme "Dodaj namirnicu na kraj liste". </p> <br> <br> <ol> <li>Hleb</li> <li>Mleko</li> <li>Šunka</li> </ol> <button id="dugme">Dodaj namirnicu na kraj liste.</button> </body> </html>
  • .prepend() -Ubacuje sadržaj na početak izabranog elementa. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("#dugme").click(function(){ $("ol").prepend("<li>Pivo</li>"); }); }); </script> </head> <body> <br> <p>U ovom primeru ćemo ubaciti još jednu stavku na početak liste klikom na dugme "Dodaj namirnicu na početak liste".</p> <br> <br> <ol> <li>Hleb</li> <li>Mleko</li> <li>Šunka</li> </ol> <button id="dugme">Dodaj namirnicu na početak liste.</button> </body> </html>
  • .after()-Ubacuje sadržaj posle izabranog elementa. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("#dugme").click(function(){ $("ul").after("<li>Kaladont</li>"); }); }); </script> </head> <body> <br> <p>U ovom primeru ćemo ubaciti tekst posle date liste klikom na dugme "Ubaci tekst ispod liste".</p> <br> <br> <ul> <li>Meso</li> <li>Lubenica</li> <li>Banene</li> </ul><br><br> <button id="dugme">Ubaci tekst ispod liste.</button> </body> </html>
  • .before()-Ubacuje sadržaj pre izabranog elementa. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("#dugme").click(function(){ $("ul").before("<li>Kaladont</li>"); }); }); </script> </head> <body> <br> <p>U ovom primeru ćemo ubaciti tekst iznad date liste klikom na dugme "Ubaci tekst iznad liste".</p> <br> <br> <ul> <li>Meso</li> <li>Lubenica</li> <li>Banene</li> </ul><br><br> <button id="dugme">Ubaci tekst iznad liste.</button> </body> </html>

JQuery funkcije za uklanjanje

  • .remove() -Uklanja izabrani element i njegovu decu. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("button").click(function(){ $("p").remove(".crveno"); }); }); </script> <style> .crveno { color: red; font-size: 20px; } </style> </head> <body> <br> <p>U ovom primeru ćemo ukloniti elemente crvene boje klikom na dugme "Ukloni netačne odgovore".</p> <br> <br> <p>Glavni grad Kanade je Otava.</p> <p class="crveno">Glavni grad Kanade je Alberta.</p> <p class="crveno">Glavni grad Kanade je Ontario.</p> <button>Ukloni netačne odgovore.</button> </body> </html>
  • .empty() -Uklanja svu decu izabranog elementa <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("button").click(function(){ $("#roditelj").empty(); }); }); </script> <style> #roditelj { background-color:yellow; height:120px; width:300px; border:1px solid black; } </style> </head> <body> <br/> <div id="roditelj"> <p>Glavni grad Australije je Kanbera.</p> <p>Glavni grad Brazila je Brazilija.</p> <p>Glavni grad Svajcarske je Bern.</p> </div> <br/> <button>Ukloni sve iz div elementa.</button> </body> </html>
