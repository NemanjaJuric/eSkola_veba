Efekti biblioteke jQuery

biblioteka sadrži i metode za animiranje veb stranica, to su uglavnom animacije koje se najčešće primenjuju na stranice. U ovom kursu ćemo da prođemo kroz većinu Jquery metoda za animiranje.

Pre nego prikažemo primere iz oblasti animaranje jQuery biblioteke, objasnimo tri najčešća parametra koja se mogu proslediti gotovo svim funkcijama za animaciju:

  • trajanje – izražava se brojem koji se odnosi na broj milisekundi i predstavlja vreme za koje će se animacija izvršiti. Postoje i tri predefinisane string vrednosti: slow, normal i fast. Slow predstavlja vreme od 600 milisekuni, normal predstavlja vreme od 400 milisekundi i, konačno, fast predstavlja vreme od 200 milisekundi.
  • stil animacije – izražava se sa dve string vrednosti i to linear i swing. Predstavlja vreme animacije u određenim momentima pa tako se swing odnosi na pomeranje koje je u startu i na kraju izvođenja animacije sporije nego na sredini, dok se linear odnosi na pomeranje koje je u svim trenucima isto.
  • funkcija sa povratnim pozivom – predstavlja funkciju koja se izvršava nakon što je animacija završena.

Efekti

  • .animate(svojstva, trajanje, stil animacije, funkcija sa povratnim pozivom) - Aktivira standarnu animaciju nad elementom sa prosleđenim svojstvima <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js "></script> <script> $(document).ready(function(){ $("#desno").click(function() { $("#kutija").animate( {"left": "+=50px"}, "slow"); }); $("#levo").click(function() { $("#kutija").animate( {"left": "-=50px"}, "slow"); }); }); </script> <style> #kutija{ background:#98bf21; height:100px; width:100px; margin:6px; position:absolute; left:100px } </style> </head> <body> <p>Primer u kom uz pomoć animate metode pomeramo element levo i desno.</p> <br/> <input type="button" id="levo" value="Levo"/> <input type="button" id="desno" value="Desno"/> <div id="kutija"></div> </body> </html>
  • .delay()- Delay metoda postavlja vreme kašnjenja za aktiviranje animacije odabranog elementa. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js "></script> <script> $(document).ready(function(){ $("button").click(function(){ $("#div1").delay(800).animate({width: "300px"}); $("#div2").delay("fast").fadeIn(); }); }); </script> </head> <body> <p>Primer sa različitim brzinama delay() metoda.</p> <button>Klikni za animacije sa različitim brzinama</button> <br><br> <div id="div1" style="width:90px;height:90px;background-color:black;"></div><br> <div id="div2" style="width:90px;height:90px;display:none;background-color:green;"></div><br> </body> </html>
  • .fadeOut(trajanje, stil animacije, funkcija sa povratnim pozivom)- Aktivira animaciju u kojoj izabrani element postepeno nestane. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js "></script> <script> $(document).ready(function(){ $(".dugme").click(function(){ $("p").fadeOut() }); }); </script> </head> <body> <br> <p>Primer u kom paragraf postepeno nestane.</p> <br> <br> <p style="background-color:orange; color:white">Milutin Milanković</p> <button class="dugme">Fade out</button> </body> </html>
