JQuery-Događaji


Najvažnije JQuery događaje možemo da podelimo u sledeće grupe:
  • Događaji pregledača
  • Događaji formulara
  • Događaji tastature
  • Događaji miša
  • Događaji učitavanja dokumenta

Događaji pregledača

  • .error() - Aktivira događaj greške <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("div").error(function(){ alert("Greska!!"); }); $("button").click(function(){ $("p").error(); }); }); </script> </head> <body> <div style ="background-color: green"> <p> Prouzrokuj grešku!</p> </div> <button>Klikni i prouzrokuj grešku!</button> </body> </html>
  • .resize() - Aktivira događaj prilikom promene veličine. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> x = 0; $(document).ready(function(){ $(window).resize(function(){ $("span").text(x += 1); }); }); </script> </head> <body> <p>Veličina prozora promenjena: <span>0</span> puta.</p> <p>Smanji ili povećaj veličinu prozora!</p> </body> </html>
  • .scroll() - Aktivira događaj prilikom skrolovanja <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> x = 0; $(document).ready(function(){ $("div").scroll(function(){ $("span").text( x+= 1); }); }); </script> </head> <body> <p>Skrolujete naredni tekst.</p> <div style="border:1px solid black;width:200px;height:100px;overflow:scroll;">Pupin je u jednoj knjizi iz koje je učio na Kembridžu na marginama napisao „Preći pet strana dnevno“. Ova beleška me je usmerila kako raditi – svakog dana malim koracima, ali bez prekida. Čime god da se bavimo, to daje sigurne rezultate. Nikad za deset godina nisam napravila pauzu, nisam ostavila Pupina pa mu se vratila. Intenzitet nije bio uvek isti, ali moja misija je uvek bila prisutna. Od Pupina možemo da naučimo da je vredno biti istrajan i da istrajnost uvek daje rezultat. <br><br> Ali najveća lekcija je da najveći rezultat i najveće zadovoljstvo dolaze iz rada za opšte dobro. Stalni kapacitet da proizvodi dobro, da radi za dobro, da radi za druge – to je najveća pouka. </div> <p>Skrolovano <span>0</span> puta.</p> </body> </html>

Događaji formulara.

  • .change()- Aktivira dodgađaj nakon sto se napravi izmena nad elementom. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> x=0; $(document).ready(function(){ $("input").change(function(){ $("span").text( x+= 1); }); }); </script> </head> <body> <input type="text"> <p>Unesi tekst i klikni van tekstualnog polja.</p> <br> <p>Tekst je izmenjen <span>0</span> puta.</p> </body> </html>
  • .select()-Aktivira događaj kad je tekst elementa selektovan. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("input").select(function(){ $( "p" ).text( "Deo teksa je selektovan!" ).show().fadeOut( 1000 ); }); }); </script> </head> <body> <input type="text" value="Mihajlo Idvorski Pupin"> <p>Selektuj neki deo teksta iz tekstualnog polja.</p> </body> </html>
  • .focus()-Aktivira događaj kada element dobije fokus. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("input").focus(function(){ $("span").css("display", "inline").fadeOut(2000); }); }); </script> <style> span { display: none; } </style> </head> <body> <input> <span>Osmeh!?</span> <p>Klikni u tekstualno polje za fokus.</p> </body> </html>
  • .submit()- Aktivira događaj nakon sto unesemo formu. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("form").submit(function(){ alert("Uneseno!"); }); }); </script> </head> <body> <form action=""> First name: <input type="text" name="Ime" value="Mihajlo"><br> Last name: <input type="text" name="Prezime" value="Pupin"><br> <input type="submit" value="Unesi"> </form> </body> </html>
  • .blur() - Aktivira događaj nakon sto element izgubi fokus. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("input").blur(function(){ alert("Tektualno polje je izgubilo svoj fokus."); }); }); </script> </head> <body> Unesite svoje ime: <input type="text"> <p>Unesite ime u tekstualno polje, a potom kliknite van njega da izgubi fokus.</p> </body> </html>

Događaji tastature.

  • .keydown()-Aktivira događaj kada držimo pritisnut taster na tastaturi. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("input").keydown(function(){ $("input").css("background-color", "yellow"); }); $("input").keyup(function(){ $("input").css("background-color", "white"); }); }); </script> </head> <body> Unesi svoje ime: <input type="text"> <p>Unesi ime u tekstualno polje. Promeniće pozadinu na pritisnut taster.</p> </body> </html>
  • .keypress()-Aktivira događaj na pritisnut taster na tastaturi. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> i = 0; $(document).ready(function(){ $("input").keypress(function(){ $("span").text(i += 1); }); }); </script> </head> <body> Unesi tekst: <input type="text"> <p>Broj unetih slova: <span>0</span></p> </body> </html>
  • .keyup()-Aktiva događaj prilikom puštanja tastera na tastaturi. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("input").keydown(function(){ $("input").css("background-color", "white"); }); $("input").keyup(function(){ $("input").css("background-color", "pink"); }); }); </script> </head> <body> Unesi svoje ime: <input type="text"> <p>Unesite ime u tekstualno polje. Promeniće pozadinu prilikom puštanja pritisnutog tastera.</p> </body> </html>

Događaji miša.

  • .click()- Aktivira događaj nakon klika na element. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("p").click(function(){ $("div").append("<p>Majka ima 30 a ćerka 10 godina.</p>") }); }); </script> </head> <body> <div> <p>Pre pet godina majka je bila pet puta starija od ćerke, a za deset godina biće samo dva puta starija. <br> Koliko godina imaju majka i ćerka? (klikni na pitanje da dobiješ odgovor!) </p> </div> </body> </html>
  • .dblclick()-Aktivira događaj nakon dvostrukog klika tastera miša. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("p").dblclick(function(){ $("div").append("<p>Velikih je 9 a malih 81!</p>") }); }); </script> </head> <body> <div> <p>Ako u svaku veliku kutiju stavimo onoliko malih kutija koliko imamo velikih kutija ukupan broj ce biti 90. <br>Koliko imamo velikih a koliko malih kutija? (dvoklikom na pitanje da dobiješ odgovor!) </p> </div> </body> </html>
  • .hover()-Aktivira događaj prelaskom kursa miša preko elementa. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("p").hover(function(){ $("div").append("<p>Marko je čitao knjigu 24 dana.</p>") }); }); </script> </head> <body> <div> <p>Marko je knjigu iz lektire koja ima 384 stranice, čitao određen broj dana. Janko je tu istu knjigu čitao 8 dana kraće jer je svakog dana čitao 8 stranica više nego Marko. <br>Koliko je dana Marko čitao knjigu?(prelaskom kursorom miša preko pitanja dobijaš odgovor!) </p> </div> </body> </html>
  • .mousedown()-Aktivira događaj držeci pritisnut taster miša. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("input").mousedown(function(){ $("input").css("background-color", "yellow"); }); $("input").mouseup(function(){ $("input").css("background-color", "white"); }); }); </script> </head> <body> Klikni na polje: <input type="text"> <p>Pritisom tastera (i držanjem) promenićeš boju polja.</p> </body> </html>
  • .mouseup()-Aktivira događaj nakon puštanja pritisnutog tastera miša. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $("input").mousedown(function(){ $("input").css("background-color", "white"); }); $("input").mouseup(function(){ $("input").css("background-color", "pink"); }); }); </script> </head> <body> Klikni na polje: <input type="text"> <p>Puštanjem pritisnutog tastera miša promenićeš boju polja.</p> </body> </html>
  • .mousemove()-Aktivira događaj prilikom pomeranja miša. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ $(document).mousemove(function(event){ $("span").text(event.pageX + ", " + event.pageY); }); }); </script> </head> <body> <p>Koordinate miša: <span></span>.</p> </body> </html>




Događaji učitavanja dokumenta.

  • .ready()-Aktivira događaj nakon što je DOM drvo spremno za manipulaciju. <!DOCTYPE html> <html> <head> <script src="https://code.jquery.com/jquery-latest.min.js"></script> <script> $(document).ready(function(){ alert("Spremno za manipulaciju!") }); </script> </head> <body> <p>Upozorenje se prikazuje kada je DOM drvo spremno za manipulaciju!</p> </body> </html>
