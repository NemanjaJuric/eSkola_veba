JQuery-AJAX


Osnovna funkcionalno AJAX (Asynchronous JavaScript and XML) poziva jeste promena sadržaja strane asinhrono, što znači da korisnička veb strana ne mora da se učita potpuno ispočetka već se ažurira sadržaj strane na osnovu podataka koji su pristigli. Dva najčešća zahteva serveru jesu:
  • HTTP GET zahtev
  • HTTP POST zahtev

  • .get() - Funkcija za dohvatanje podataka sa određenog izvora. <!DOCTYPE html> <html> <head> <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script> <script> $(document).ready(function(){ $("button").click(function(){ $.get("https://jsonplaceholder.typicode.com/todos/1", function(data, status){ $('p').html("Naslov: " + data.title + "<br />" + "Status: " + status); }); }); }); </script> </head> <body> <button>Posalji HTTP GET zahtev i prikaži naslov iz odgovora</button> <p></p> </body> </html>
  • .post() - Funkcija za slanje novih podataka serveru zarad kreiranja novog objekta ili ažuriranje postojećeg. <!DOCTYPE html> <html> <head> <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script> <script> $(document).ready(function(){ $("button").click(function(){ $.post("https://jsonplaceholder.typicode.com/posts", { title: 'foo', body: 'bar', userId: 1 }, function(data,status){ alert("Id: " + data.id + "\nTitle: "+ data.title + "\nUserID: " + data.userId + "\nStatus: " + status); }); }); }); </script> </head> <body> <button>Pošalji HTTP POST zahtev i prikaži rezultat kreiranog objekta</button> </body> </html>
