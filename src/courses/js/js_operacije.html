Operator sabiranja

Ovaj operator funkcioniše na različite načine u zavisnosti koje vrednosti sabira.

Kada sabiramo dva stringa, operator sabiranja spaja levi i desni argument, dok kada sabiramo dva broja, rezultat je njihov zbir.

Primer:
var broj1 = 1; var string1 = "Ja sam broj"; var broj2 = 999; var string2 = "99"; var rezultat1 = string1 + broj 1; //Rezultat &#263;e biti "Ja sam broj 1" var rezultat2 = broj 1 + broj2; //Rezultat &#263;e biti 1000 var rezultat3 = broj1 + string2; //Rezultat &#263;e biti 199

Korisćenjem metode Number() konvertujemo string u broj. Npr:
var broj1 = 1; var string2 = Number("99"); var rezultat3 = broj1 + string2; //Rezultat &#263;e biti 100

Operatori jednakosti

Ovi operatori koriste se da proverimo da li su neka dva izraza ista ili se razlikuju.

Uvek daju vrednosti true ili false.

Operator Značenje
= = Jednako
!= Različito
= = = Jednako korišćenjem strožije metode
!= = Različito korišćenjem strožije metode

Operator in

Ovaj operator se najčešće koristi za procenjivanje da li je neko svojstvo sadržano unutar objekta ili ne.

Takodje se koristi i za kretanje kroz objekat.

Operator instanceof
