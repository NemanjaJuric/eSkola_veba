Veza sa HTML elementima


Kad programirate u JS, pronalaženje elemenata predstavlja osnovni način korišćenja DOM-a. U ovom odeljku ćemo ispitati dve glavne metode koje koristite za pronalaženje elemenata, a to su metode: getElementById() i getElementByTagName() .

Pronalaženje elemenata koristeći ID


Metoda getElementById() predstavlja glavnu metodu DOM-a. Ova metoda pronalazi određeni element HTML dokumenta i vraća ga kao referencu na taj element. Da biste pronašli određeni element potreban je id atribut.
Na primer, iz primera 1 možete modifikovati HTML dokument kako biste elementu <a> dodelili atribut id , što bi izgledalo ovako:
<!doctype html> <html> <head> <title> Zdravo </title></head> <body> <p> Tekst </p> <p> Tekst </p> <p> Link <a id="googlelink" href="http://www.google.com"> Google </a></p> </body> </html>
Pošto element a ima id atribut, možete ga pronaći koristeći metodu getElementById()
var a1 = document.getElementById("googlelink"); Referenca elementa sa ID-om googlelink biće smeštena unutar JavaScript promenljive a1.
Zadatak1: Modifikovati HTML iz primera 1, tako da svi elementi < p > dobiju ID, tako da se oni mogu pronaći upotrebom metode getElementById(). Zatim, napisati deo koda koji prikazuje kako pronalazimo elemente.

Pronalaženje pomoću imena oznake


Metoda getElementById() funkcioniše dobro kada želite da pronađete samo jedan ili nekoliko elemenata. Međutim, kada istovremeno želite da pronađete više od jednog elementa, preporučuje se korišćenje metode getElementByTagName().
Metoda getElementByTagName() vraća sve elemente određenog tipa oznake i to u obliku niza ili liste. Na primer, da biste pronašli sve slike oznake <img> u dokumentu, ukucajte sledeći kod:
var images = document.getElementsByTagName("img");
Zatim, možete ispitati osobine <img> elementa, koje se nalazi unutar promenljive images, tako što ćete proći petljom kroz njih.
Navešćemo jedan primer koji modifikuje tabelu. Ovaj kod menja boju pozadine svakog <td> elementa unutar tabele , svaki put kada korisnik klikne na link Klikni da izmeniš boju.
Primer 2: <!doctype html> <html> <head> <title> Tag name </title> <script type = "text/javascript"> function promeni_boju(){ var a2 = document.getElementsByTagName("td"); var a2length= a2.length; for (var i = 0; i < a2length; i++){ a2[i].style.background = "#aaabba"; } } </script> </head> <body> <table id = "moja_tabela" border = "1"> <tr> <td id = "levo0"> Leva kolona </td> <td id = "desno0"> Desna kolona </td> </tr> <tr> <td id = "levo1"> Leva kolona </td> <td id = "desno1"> Desna kolona </td> </tr> <tr> <td id = "levo2"> Leva kolona </td> <td id = "desno2"> Desna kolona </td> </tr> </table> <a href = "#" onclick = "promeni_boju()"> Klikni da promeniš boju </a> </body> </html>
Ova funkcija pronalazi sve <td> elemente korišćenjem metode getElementByTagName() i smešta ih u niz a2:
var a2= document.getElementByTagName("td");

Kreiranje elemenata


Nismo ograničeni samo na interakciju sa elementima koji postoje na stranici, već slobodno možemo dodavati elemente upotrebom DOM-a. Sada ćemo objasniti način na koji se to i radi:

Dodavanje teksta


U najosnovnijem obliku metoda createElement() kreira ili dodaje element dokumentu. Navodimo jedan primer koda: var novielement = document.createElement("p");
Promenljiva novielement sada ima referencu ka novom elementu. Da bi naš elemenat bio vidljiv, moramo ga pridružiti elementu. Element dodajemo dokumentu koristeći metodu appendChild(), kao što je prikazano:
document.body.appendChild(novielement);
Međutim, kakvu korist imamo od elementa <p> ako on ne sadrži tekst? Ovde nam može pomoći element appendChild() sa metodom createTextNode(). Možemo dodati čvor teksta elementu

i to ovako:
novielement.appendChild(document.createTextNode("Zdravo svima"));
Ova tri reda koda možemo koristiti u telu dokumenta i imati ovako neki primer:
Primer 3:

<!doctype html> <html> <head> <title> Create </title> </head> <body> <script type = "text/javascript"> var novielement = document.createElement("p"); document.body.appendChild(novielement); novielement.appendChild(document.createTextNode("Zdravo svima")); </script> </body> </html>

Dodavanje elementa i podešavanje ID-a

Prethodni primer nam je pokazao kako se dodaje element. Pored dodavanja, nekad je potrebno dodati i neki atribut kao sto je ID za neki element.
Naredni kod se vezuje za prethodni primer kako bi se dodao atribut id:
<!doctype html> <html> <head> <title> Create </title> </head> <body> <script type = "text/javascript"> var novielement = document.createElement("p"); novielement.setAttribute("id","novielement"); document.body.appendChild(novielement); novielement.appendChild(document.createTextNode("Zdravo svima")); </script> </body> </html>

Brisanje elemenata


Možete ukloniti čvorove iz dokumenta koristeći metodu removeChild().
Nadovežimo se na prethodni primer i dodajmo još nekoliko elemenata <p>.
<!doctype html> <html> <head> <title> Remove </title> </head> <body> <script type="text/javascript"> for(var i = 0; i < 3; i++){ var element = document.createElement("p"); element.setAttribute("id","element" + i); document.body.appendChild(element); element.appendChild(document.createTextNode("Zdravo svima, ja sam element" + i + "."));} </script> </body> </html>
Zatim, želimo da ukonimo neki od novokreiranih elemenata koristeći metodu removeChild(). Ovo su dva reda koja su dodata kodu:
var removeel = document.getElementById("element1"); document.body.removeChild(removeel);
Celokupan primer dodavanja i uklanjanja elemenata izgleda ovako:
<!doctype html> <html> <head> <title> Remove </title> </head> <body> <script type="text/javascript"> for(var i = 0; i < 3; i++){ var element = document.createElement("p"); element.setAttribute("id","element" + i); document.body.appendChild(element); element.appendChild(document.createTextNode("Zdravo svima, ja sam element" + i + ".")); } var removeel = document.getElementById("element1"); document.body.removeChild(removeel); </script> </body> </html>
Zadatak2: Kreirati dokument u koji ćete dodati neki tekst,a zatim ukloniti tekst iz jednog reda pasusa.
