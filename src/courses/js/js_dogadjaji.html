Događaji


Rukovanje događajima koristimo kako bi reagovali na akcije korisnika ili događaje dokumenta.
DOM model događaja sadrži nekoliko događaja koje HTML oznake aktiviraju kao odgovor na akcije korisnika ili odgovor na promenu stanja.
Kroz tabelu ćemo opisati svaki dogadjaj.
Tabela 1- DOM događaji
Naziv dogadjaja Opis
onblur() element gubi fokus.(tj.više nije izabran od strane korisnika)
onchange() element je izmenjen (na primer, korisnik je uneo neki podatak u polje za tekst ) i nije više u fokusu
onclick() element je izabran pritiskom na taster miša
ondblclick() element je izabran dvostrukim pritiskom na taster miša
onfocus() element je dobio focus
onkeydown() taster na tastaturi je pritisnut dok je element u fokusu
onkeypress() dok je element u fokusu pritisnut je (i pušten) taster na tastaturi
onkeyup() taster na tastaturi je otpušten dok je element u fokusu
onload() element se učitava(dokument, slika)
onmousedown() taster miša je pritisnut dok je element u fokusu
onmousemove() miš je pomeren dok se pokazivač nalazio iznad elementa
onmouseout() pokazivač miša je pomeren sa elementa
onmouseover() pokazivač miša se nalazi iznad elementa
onmouseup() taster miša je otpušten
onreset() element obrasca je resetovan, kao na primer kada korisnik pritisne dugme za resetovanje formulara
onresize() veličina prozora je promenjena
onselect() tekst u elementu obrasca je izabran (za file, password, textarea)
onsubmit() formular se šalje
onunload() Dokument je zatvoren ili se otišlo na drugu stranicu. Ovo se desi kada se prozor pretraživača isključi ili kada se ode na drugu stranicu, tj.kad stranica treba da se "uništi". Onda npr.može da se pita "da li želis da zatvoriš stranicu, a nisi poslao poruku."

Prikažimo neke od događaja:

Događaji za učitavanje stranice

  • onload() Aktivira događaj kada se stranica učita.

Događaji za učitavanje stranice

  • onunload() Aktivira događaj kada se stranica ne učita ili ako se prekine učitavanje stranice

Događaji vezani za formu

  • onfocus() Aktivira događaj kada element dobije focus.
  • onblur() Aktivira događaj kada element izgubi focus.
  • onchange() Aktivira događaj kada je neki element izmenjen.
  • onreset() Aktivira događaj kada je neki element resetovan
  • onsubmit() Aktivira događaj za slanje formulara.

Događaji vezani za tastaturu

  • onkeydown() Aktivira događaj kada se pritisne taster na tastaturi
  • onkeyup() Aktivira događaj kada se pusti taster na tastaturi
  • onkeypress() Aktivira događaj kada se pritisne (pusti) taster na tastaturi

Događaji vezani za taster miša

  • onclick() Aktivira događaj kada element izabran pritiskom tastera miša.
  • ondblclick() Aktivira događaj kada element izabran dostrukim pritiskom tastera miša.
  • onmousedown() Aktivira događaj pritiskom tastera miša dok je element u fokusu
  • onmouseup() Aktivira događaj kada je otpušten taster miša
  • onmousemove() Aktivira događaj prilikom pomeranja kursora miša
  • onmouseout() Aktivira događaj prilikom pomeranja kursora miša izvan elementa
  • onmouseover() Aktivira događaj prelaskom tastera miša preko elementa
