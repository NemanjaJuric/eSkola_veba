JavaScript Stringovi

Šta su stringovi?

JavaScript stringovi se koriste za čuvanje i rad sa tekstom

String može biti bilo koji tekst (niz znakova) između jednostrukih ili dvostrukih navodnika. Na primer, ''e-skola weba'' ili 'grupa 5ML'.

Primer 1: <!DOCTYPE html> <html> <body> <p id="demo"></p> <script> var carName1 = "Nikon D3200"; var carName2 = 'Nikon D3200'; document.getElementById("demo").innerHTML = carName1 + "<br>" + carName2; </script> </body> </html>

U okviru stringa mogu se koristiti i navodnici koji se razlikuju od navodnika koji taj niz znakova okružuju.

Primer 2: <!DOCTYPE html> <html> <body> <p id="demo"></p> <script> var answer1 = "Sve je u redu!"; var answer2 = "Zvali smo Ivana!"; var answer3 = 'Zvali smo Ivana!'; document.getElementById("demo").innerHTML = answer1 + "<br>" + answer2 + "<br>" + answer3; </script> </body> </html>

Dužina stringa

Dužina stringa može se proveriti pomoću ugrađene funkcije LENGTH.

Primer 3: <!DOCTYPE html> <html> <body> <p id="demo"></p> <script> var txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"; document.getElementById("demo").innerHTML = txt.length; </script> </body> </html>

Specijalni znaci

Stringovi moraju biti napisani između navodnika, jer JavaSkript može pogrešno shvatiti na primer sledeću konstrukciju:

Primer 4: var y = "We are the so-called "Vikings" from the north."

String će biti iseckan na "We are the so-caled". Da bi se izbegao ovaj problem, rešenje je da se koristi znak \ (escape ili backslash). On pretvara specijalni znak u deo stringa.

Primer 5: <!DOCTYPE html> <html> <body> <p id="demo"></p> <script> var x = 'It\'s alright'; var y = "We are the so-called \"Vikings\" from the north."; document.getElementById("demo").innerHTML = x + "<br>" + y; </script> </body> </html>
We are the so-called "Vikings" from the north.

Stringovi kao objekti

JavaScript stringovi su primitivne vrednosti, sastavljene od znakova ( var custorginn = "Jovan"). Ali stringovi takođe mogu biti definisani i kao objekti sa ključnim rečima ( var FirstName = novi string ( "Jovan") )

Primer 6: <!DOCTYPE html> <html> <body> <p id="demo"></p> <script> var x = "Jovan"; // x is a string var y = new String("Jovan"); // y is an object document.getElementById("demo").innerHTML = typeof x + "<br>" + typeof y; </script> </body> </html> Primer 7: <!DOCTYPE html> <html> <body> <h2>Never Create Strings as objects.</h2> <p>JavaScript objekti ne mogu se porediti.</p> <p id="demo"></p> <script> var x = new String("Jovan"); // x is an object var y = new String("Jovan"); // y is an object document.getElementById("demo").innerHTML = (x === y); </script> </body> </html>

Operacije nad stringovima

Nad primitivnim vrednostima, kao što su ''grupa 5ML'', ne mogu se izvršavati operacije (zato što nisu objekti).

Ali u JavaScript-u, operacije se mogu primenjivati i nad primitivnim vrednostima, jer JavaSkript tretira primitivne vrednosti kao objekte pri izvršavanju operacija.

Primer 8: <!DOCTYPE html> <html> <body> <p id="demo"></p> <script> var txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"; document.getElementById("demo").innerHTML = txt.length; </script> </body> </html>

Operacija indexOf () vraća indeks (položaj) prvog pojavljivanju određenog teksta u stringu:

Primer 9: <!DOCTYPE html> <html> <body> <p id="p1">Please locate where 'locate' occurs!.</p> <button onclick="myFunction()">proba</button> <p id="demo"></p> <script> function myFunction() { var str = document.getElementById("p1").innerHTML; var pos = str.indexOf("locate"); document.getElementById("demo").innerHTML = pos; } </script> </body> </html>

Operacija search() traži u stringu zadatu vrednost i vraća poziciju poklapanja:

Primer 10: <!DOCTYPE html> <html> <body> <p id="p1">Please locate where 'locate' occurs!.</p> <button onclick="myFunction()">proba</button> <p id="demo"></p> <script> function myFunction() { var str = document.getElementById("p1").innerHTML; var pos = str.search("locate"); document.getElementById("demo").innerHTML = pos; } </script> </body> </html>

Ako izostavite drugi parametar, pretraga će se nastaviti do kraja niza.

Operacija The slice() izdvaja deo stringa i vraca izdvojeni deo kao novi niz.

Primer 11: <!DOCTYPE html> <html> <body> <p> Operacija The slice() izdvaja deo stringa i vraca izdvojeni deo kao novi niz: </p> <p id="demo"></p> <script> var str = "Apple, Banana, Kiwi"; document.getElementById("demo").innerHTML = str.slice(7); </script> </body> </html>

Operacija Substring () je slična operaciji slice(). Razlika je jedino u tome što ne može da prihvati negativne indekse.

Primer 12: <!DOCTYPE html> <html> <body> <p> Operacija The substr() izdvaja deo stringa i vraca izdvojeni deo kao novi niz: </p> <p id="demo"></p> <script> var str = "Apple, Banana, Kiwi"; document.getElementById("demo").innerHTML = str.substring(7, 13); </script> </body> </html>

Zadatak:
Izmeniti program koji za niz znakova (string) ''Isprobaj i ti.'' Određuje njegovu dužinu, kao i indeks (položaj) prvog pojavljivanja reči "proba".
