Bulove vrednosti


Bulove vrednosti mogu biti true (tačno) i false (netačno).

Najčešće se koriste Bulovi izrazi unutar testova, kao što je if/then/else iskaz.

Primer iskaza u kom je korišćen Bulov izraz: 

if(broj < 10){
  //uradi nešto
}

Ovaj primer služi da pokaže da li će se kod unutar zagrada izvršiti ili ne.
Ukoliko je broj manji od 10, Bulov izraz će imati vrednost true i kod će se izvršiti, u suprotnom neće.

Null vrednosti


Ovo je specijalan tip podataka u JavaScriptu.
To je ništa. Predstavlja se kao vrednost false.

Ne treba mešati null sa praznom promenljivom. Praznina i null vrednost se razlikuju. Prazna promenljiva je i dalje puna - puna je praznine.

Npr. ukoliko bismo napisali ovako nešto:
var mojaPromenljiva='';
to znači da je promenljiva moja Promenljiva prazna, ali ona nema vrednost null.

Nedefinisane vrednosti


Ovo je zapravo stanje koje se koristi kao vrednost.
Koristimo ga onda kada promenljivoj nije dodeljena nikakva vrednost.
Razlikuje se od null vrednosti.

Primer: 
if (number.value < 100) { 
	ispis = "Proizvod je: " + number.value;
}
else { 
	ispis = 'Rezultat je veći od 100!';
}

U primeru proveravamo da li je proizvod dva uneta broja veći ili manji od 100, ako je manji ispisujemo rezultat, a ako je veći ispisujemo: 'Rezultat je veći od 100!'.

Zadatak: Promeni program tako da uneseš jedan broj, proveri da li je njegov kvadrat manji od 100, ako jeste ispiši rezultat, a ako nije ispiši da je kvadrat prvog broja veći od 100.
