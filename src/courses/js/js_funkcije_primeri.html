Nakon pokretanja primera funkcija prompt() pokreće iskačući prozor, sa tekstualnim poljem u kome je moguće uneti dužinu ivice kvadrata.
Pri validnom i nevalidnom unosu, program radi. Ukoliko korisnik ne unese podatak, program nastavlja sa radom, a to nije očekivano. Poželjno je da ukoliko ne bude unosa, program prekine sa radom.

Palindrom

function is_Palindrome(str1) { var rev = str1.split("").reverse().join(""); return str1 == rev; } function longest_palindrome(str1){ var max_length = 0, maxp = ''; for(var i=0; i < str1.length; i++){ var subs=str1.substr(i, str1.length); for(var j=subs.length; j>=0; j--){ var sub_subs_str = subs.substr(0, j); if (sub_subs_str.length <= 1) continue; if (is_Palindrome(sub_subs_str)){ if (sub_subs_str.length> max_length){ max_length = sub_subs_str.length; maxp = sub_subs_str; } } } } return maxp; }
Ovaj primer prikazuje funkciju, t.j. kombinaciju funkcija koje pronalaze najduži palindrom u prosleđenoj reči.

Funkcija is_Palindrome proverava da li je prosleđena reč palindrom. Prvo se string metodom split("") pretvori u niz karaktera, zatim metodom reverse() se tako dobijen niz preokrene, a na kraju se metodom join("") preokrenut niz pretvara u string. Odnosno, dobije se preokrenuti string u odnosu na početni i sačuva se u promenljivoj rev. Funkcija na kraju proverava da li su jednaki početni string i novonastali preokrenuti string i ukoliko je odgovor potvrdan vraća vrednost true, u suprotnom vraća vrednost false.

Funkcija longest_palindrome prosleđeni string razlaže na podstringove i proverava svaki da li je palindrom koristeći se funkcijom is_Palindrome. Prva for petlja od napred prolazi kroz string, dok druga for petlja od pozadi skraćuje podstringove koji se proveravaju.

Na primer, ukoliko se proverava reč banana, provera će ići sledećim tokom: banana (i=0, j=6) banan (i=0, j=5) bana (i=0, j=4) ban (i=0, j=3) ba (i=0, j=2) b (i=0, j=1) anana (i=1, j=6) anan (i=1, j=5) ana (i=1, j=5) ...
Provera se dalje nastavlja istim tokom, ne prekida se iako je pronađena reč koja jeste najduži palindrom. Prvi pronađeni palindrom je slovo b i tada promenljiva max_length uzima vrednost 1. Sledeća reč anana je takođe palindrom, dužine 5, a promenljiva max_length uzima vrednost 5, jer je 5>1. Sledeći palindrom na koji funkcija naiđe je ana, ali on je kraće dužine, pa se vrednost promenljive max_length neće promeniti. Funkcija će još jednom naići na reč ana koja je palindrom, koja takođe neće promeniti vrednost promenljive max_length.

Kada funkcija izađe iz obe for petlje vraća se vrednost sačuvana u promenljivoj max_length, koja bi u navedenom primeru bila broj 5.

Najmanji zajednički sadržalac

function NZS(x, y){ return Math.abs((x * y) / NZD(x, y)); } function NZD(x, y){ x = Math.abs(x); y = Math.abs(y); while(y){ var t = y; y = x % y; x = t; } return x; }
Funkcije date u ovom primeru pronalaze najmanji zajednički sadržalac i najveći zajednički delilac dva broja.

Funkcija NZS pronalazi najmanji zajednički sadržalac prosleđenih brojeva tako što absolutnu vrednost proizvoda tih brojeva podeli sa njihovim najvećim zajedničkim deliocem.

Funkcija NZD pronalazi najveći zajednički delilac prosleđenih brojeva. Prvo se oba broja pretvore u njihovu apsolutnu vrednost, odnosno ukoliko je neki broj bio pozitivan, njegova vrednost se neće promeniti. Zatim se while petljom primenjuje Euklidov algoritam za pronalaženje najvećeg zajedničkog delioca sve dok je druga promenljiva veća od nule (vrednosti prve i druge promenljive se menjaju prolaskom kroz petlju). Kada se izađe iz while petlje, odnosno kada vrednost druge promenljive postane 0, tada se vraća vrednost prve promenljive, koja predstavlja najveći zajednički delilac brojeva.
