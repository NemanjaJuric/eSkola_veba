If



Uslovne naredbe u JavaScript-u se koriste da bi se izvršile različite akcije u zavisnosti od nekog uslova.
Tako na početku imamo naredbu If. Ona služi da proverimo da li je neki uslov ispunjen i u zavisnosti od ispunjenosti uslova izvršimo zadatu naredbu.
Naredba if ima sledeću strukturu:

if ( uslov ) {
naredba
}

Ovo nam govori da ukoliko uslov u zagradi bude tačan izvršice se naredba napisana u vitičastoj zagradi.
Primer:

if (number.value==5) { alert (“Uneli ste broj 5!”) }

U ovom primeru proveravamo da li je vrednost broja koji korisnik unese 5, ako jeste izbacuje se poruka “Uneli ste broj 5!”.

Kako mi postavljamo pitanje o ispunjenosti nekog uslova odgovor koji nam Javascript vraća može da bude TAČNO ili NETAČNO. U našem prvom primeru pitamo se da li je vrednost unetog broja 5 i odgovor koji želimo da dobijemo da bi se ispisala poruka je TAČNO. Međutim može se desiti da korisnik ne unese broj 5 i tada nam Javascript vraća odgovor NETAČNO.
U našem prvom primeru nemamo alternativu za tako nešto, pa ćemo uvesti naredbu if else. Ona nam pruža mogućnost da proverimo da li je uslov ispunjen i ukoliko jeste da se izvrši jedna naredba, a ako nije da se izvrši alternativna naredba.
Naredba if else ima sledeću strukturu:

if (uslov) {naredba}
else {naredba}

Dakle ako je uslov iz zagrade ispunjen izvršava se naredba iz prve vitičaste zagrade, a ukoliko uslov nije ispunjen naredba iz druge vitičaste zagrade.
Sada naš primer sa brojem 5 možemo proširiti:

if (number.value==5) { alert (“Uneli ste broj 5!”) } else { alert (“Uneli ste broj koji nije 5!”) }

U ovom proširenom primeru smo dodali da ukoliko uslov iz zagrade nije ispunjen, tj ako nije unet broj 5, pojavljuje se poruka “Uneli ste broj koji nije 5!”.

Zadatak: Promeni program tako da proverava da li je uneti broj 4 i ispisuje poruku o tome.
