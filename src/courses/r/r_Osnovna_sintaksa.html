Osnovna sintaksa

Kao što nepisana konvencija među programerima nalaže,učenje novog programskog jezika započinje pisanjem programa "Hello World". U zavisnosti od potrebe, program se može pisati direktno u R-komandnoj liniji ili u R script fajlu koji je prethodno potrebno otvoriti.

R - komandna linija

Jednom kada se podesi okruženje, tada je lako pokrenuti R komandnu liniju, potrebno je samo početi pisati program.
Sledeća sekvenca koda napisana je direktno u komandnoj liniji i izvršena pritiskom na dugme enter.
> uvod <- "Hello, World!" > print ( uvod) [1] "Hello, World!" Prva naredba definiše promenjivu pod nazivom uvod i dodeljuje joj nisku "Hello World", a zatim se naredba print() koristi za ispis vrednosti pohranjene u promenjivoj uvod.

R Script File

U većini slučajeva, programer se odlučuje za pisanje programa u ovom fajlu, a zatim se on izvršava red po red pomoću interpretatora pod nazivom Rscript.
Prethodni delovi koda napisani u R Script fajlu izgledaju ovako:
uvod <- "Hello, World!" print ( uvod) Kada se pokrene program dobija se sledeći rezultat:
[1] "Hello, World!"
Za kraj ove uvodne lekcije, potrebno je još napomenuti da se komentari u R-u obeležavaju pomoću znaka #, što će u narednim lekcijama biti upotrebljeno mnogo puta
