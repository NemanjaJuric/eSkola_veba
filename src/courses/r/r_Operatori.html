Operatori


Operator je simbol koji govori kompajleru da izvrši određene matematičke ili logičke manipulacije. Programski jezik R je bogat ugrađenim operatorima i oni mogu da budu različitih tipova.

Tipovi operatora

U programskom jeziku R postoje sledeći tipovi operatora:
  • Aritmetički operatori
  • Relacioni operatori
  • Logički operatori
  • Operatori dodele

Aritmetički operatori

Sledeća tabela prikazuje sve aritmetičke operatore koji su podržani u R-u. Operatori deluju na svaki element vektora pojedinačno.
Operatori Opis Primer
+ sabira dva vektora x <- c(2,5.5,6) y <- c(8,3,4) print(x+y) Rezultat: [1] 10.0 8.5 10.0
- oduzima drugi vektor od prvog x <- c(2,5.5,6) y <- c(8,3,4) print(x-y) Rezultat: [1] -6.0 2.5 2.0
* množi dva vektora x <- c(2,5.5,6) y <- c(8,3,4) print(x*y) Rezultat: [1] 16.0 16.5 24.0
/ deli prvi vektor sa drugim x <- c(2,5.5,6) y <- c(8,3,4) print(x/y) Rezultat: [1] 0.250000 1.833333 1.500000
%% ostatak pri deljenju prvog vektora sa drugim x <- c(2,5.5,6) y <- c(8,3,4) print(x%%y) Rezultat: [1] 2.0 2.5 2.0
%/% celobrojno deljenje prvog vektora sa drugim x <- c(2,5.5,6) y <- c(8,3,4) print(x%/%y) Rezultat: [1] 0 1 1
^ prvi vektor podignut na eksponent drugog vektora x <- c(2,5.5,6) y <- c(8,3,4) print(x^y) Rezultat: [1] 256.000 166.375 1296.000


Relacioni operatori

Kao i u slučaju rada sa aritmetičkim operatorima, svaki element prvog vektora poredi se sa odgovarajućim elementom drugog vektora. Rezultat poređenja je tipa Boolean.


Operatori Opis Primer
> proverava da li je svaki element prvog vektora veći od odgovarajućeg elementa drugog vektora x <- c(2,5.5,6,9) y <- c(8,2.5,14,9) print(x>y) Rezultat: [1] FALSE TRUE FALSE FALSE
< proverava da li je svaki element prvog vektora manji od odgovarajućeg elementa drugog vektora x <- c(2,5.5,6,9) y <- c(8,2.5,14,9) print(x<y) Rezultat: [1] TRUE FALSE TRUE FALSE
= = proverava da li je svaki element prvog vektora jednak odgovarajućem elementu drugog vektora x <- c(2,5.5,6,9) y <- c(8,2.5,14,9) print(x==y) Rezultat: [1] FALSE FALSE FALSE TRUE
<= proverava da li je svaki element prvog vektora manji ili jednak odgovarajućem elementu drugog vektora x <- c(2,5.5,6,9) y <- c(8,2.5,14,9) print(x<=y) Rezultat: [1] TRUE FALSE TRUE TRUE
>= proverava da li je svaki element prvog vektora veći ili jednak odgovarajućem elementu drugog vektora x <- c(2,5.5,6,9) y <- c(8,2.5,14,9) print(x>=y) Rezultat: [1] TRUE FALSE TRUE TRUE
!= proverava da li je svaki element prvog vektora različit od odgovarajućeg elementa drugog vektora x <- c(2,5.5,6,9) y <- c(8,2.5,14,9) print(x>=y) Rezultat: [1] TRUE TRUE TRUE FALSE

Logički operatori

Ovaj tip operatora moguće je primeniti samo na vektore logičkog, numeričkog i kompleksnog tipa. Brojevi veći od 1 se smatraju kao logička vrednost TRUE. Svaki element prvog vektora poredi se sa odgovarajućim elementom drugog vektora. Rezultat poređenja je tipa Boolean.

Operatori Opis Primer
& Operator poznat pod nazivom "elementarno logičko i". Poredi elemente prvog vektora sa odgovarajućim elementima drugog vektora i vraća vrednost TRUE, ukoliko su obe vrednosti bile TRUE. Inače vraća vrednost FALSE x <- c(3,1,TRUE,2+3i) y <- c(4,1,FALSE,2+3i) print(x & y) Rezultat: [1]TRUE TRUE FALSE TRUE
| Naziv ovog operatora je "elementarno logičko ili". Vraća TRUE ukoliko je jedna od vrednosti koje se porede TRUE. Jedini slučaj kada vraća FALSE je kada su obe vrednosti FALSE x <- c(3,1,TRUE,2+3i) y <- c(4,1,FALSE,2+3i) print(x|y) Rezultat: [1]TRUE FALSE TRUE TRUE
! Poznat pod nazivom "elementarno logičko ne". Uzima vrednost svakog elementa vektora i vraća njegovu suprotnu vrednost x <- c(3,1,TRUE,2+3i) print(x!y) Rezultat: [1]FALSE TRUE FALSE FALSE

Logički operatori && i || porede prvi element prvog vektora sa prvim elementom drugog vektora i vraćaju odgovarajuću vrednost. Ukoliko je potrebno porediti cele vektore onda se koriste & i |. Ovi operatori porede svaki element prvog vektora sa odgovarajućim elementom drugog vektora i za svako poređenje vraćaju vrednost TRUE/FALSE

Operatori Opis Primer
&& Operator poznat pod nazivom "logičko i". Poredi samo prve elemente oba vektora i vraća TRUE u slučaju da su obe vrednosti bile TRUE. x <- c(3,0,TRUE,2+2i) y <- c(1,3,TRUE,2+3i) print(x && y) Rezultat: [1]TRUE
|| Naziv ovog operatora je "logičko ili". Poredi prve elemente oba vektora i vraća TRUE ukoliko je jedan od njih imao vrednost TRUE x <- c(0,0,TRUE,2+2i) y <- c(0,3,TRUE,2+3i) print(x||y) Rezultat: [1] FALSE

Operatori dodele

Namena ovih operatora je da dodele vrednosti vektorima.


Operatori Opis Primer
<− ili = ili <<− Levi operatori dodele x <- c(3,1,TRUE,2+3i) y <<- c(3,1,TRUE,2+3i) z = c(3,1,TRUE,2+3i) print(x) print(y) print(z) Rezultat: [1] 3+0i 1+0i 1+0i 2+3i [1] 3+0i 1+0i 1+0i 2+3i [1] 3+0i 1+0i 1+0i 2+3i
-> ili ->> Desni operator dodele c(3,1,TRUE,2+3i)-> x c(3,1,TRUE,2+3i) ->> y print(x) print(y) Rezultat: [1] 3+0i 1+0i 1+0i 2+3i [1] 3+0i 1+0i 1+0i 2+3i
