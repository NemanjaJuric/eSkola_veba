Prikaz XML dokumenta

Ne treba očekivati da prikaz XML dokumenta liči na prikaz HTML dokumenta. Zapravo, XML se suštinski razlikuje od HTML standarda. HTML je namenjen prikazu podataka, i od velikog je značaja kako prikazani podaci izgledaju. XML ima fokus na samim podacima, odnosno njihovoj strukturi i označavanju i opisivanju istih, i ne sadrži informacije o prikazu.

Većina pretraživača veba će XML dokument prikazati vrlo slično njegovom izvornom kodu, kao obojeni kod. Često se mogu naći znaci plus, minus ili strelice za proširivanje i sužavanje vidljivosti sadržaja elemenata. Kao i u HTML standardu, izvorni kod se uvek može videti opcijom „View page source“ ili „View source“ .

Prikaz XML dokumenta sa greškom

Ukoliko XML dokument ima sintaksnu grešku, različiti pretraživači mogu prikazati taj dokument na različite načine. Neki će prijaviti grešku, neki će ga prikazati, a neki će ga neispravno prikazati. Slika 6 sadrži prikaz XML dokumenta sa greškom kakav daje pretraživač Google Chrome.

Slika 6: XML dokument sa greškom (Google Chrome)

U XML dokumentu čiji je prikaz ilustrovan slikom 6, javlja se greška zato što je krajnja etiketa elementa „naslov“ pogrešno zapisana. Zapravo, zapisana je isto kao početna, što prikazuje primer 29.

Primer 29: Pogrešno zapisana krajnja etiketa elementa „naslov“ <?xml version="1.0" encoding="UTF-8"?> <obavestenje> <od>Autor</od> <za>Čitalac</za> <naslov>Obaveštenje<naslov> <poruka> Počinje kurs. Srećno! </poruka> </obavestenje>

Zašto XML ima ovakav prikaz

XML dokument ne sadrži nikakve informacije o prikazu sadržaja. S obzirom na to da su etikete u XML dokumentima formirane od strane autora, pretraživači neće razumeti njihovo tačno značenje. Dobar primer za to je baš problem opisan u lekciji „Prostori imena“, u kome nije jasno da li etiketa <grad> označava vremensku nepogodu ili naseljeno mesto. Stoga, kako pretraživači nemaju informacije o tome kako prikazati XML dokument, oni ga prikazuju u njegovom izvornom obliku.
Napomena: U prethodnoj lekciji pomenut je jezik XSLT kojim se XML dokument može transformisati u neki drugi format pa time dobiti drugačiju formu prikaza.
