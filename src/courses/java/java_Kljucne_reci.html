Ključne reči/h1>

Ključne reči (engl. keywords) jezika Java su rezervisane reči i ne mogu biti korišćene kao identifikatori. U ključne reči spadaju i const i goto, iako se ne koriste u jeziku Java, ali bi njihova pojava u programu mogla da zbuni programere koji su ranije koristili jezike C i C++.
