Moj prvi program "Zdravo svete"

Kreiranje programa obuhvata tri glavna koraka: unošenje, prevođenje (kompajliranje) i pokretanje programa.

Unošenje programa

Program u jeziku Java se na logičkom nivou sastoji od jedne ili više klasa. Izvorni tekst svake klase se piše u posebnoj datoteci čiji naziv mora početi imenom klase koju sadrži, a ekstenzija te datoteke mora biti .java. U nastavku je prikazan primer 1. koji se sastoji od samo jedne klase Zdravo.


Primer 1. – Prvi program „Zdravo svete!“ public class Zdravo { public static void main(String[] args) { System.out.println("Zdravo svete!"); } }

U ovom primeru, koristeći bilo koji editor, tekst klase treba sačuvati u datoteku pod nazivom Zdravo.java. Treba voditi računa i o tome da raspored velikih i malih slova u imenu klase i imenu datoteke mora biti isti. Java pravi razliku između malih i velikih slova.

Prevođenje programa

Pre izvršavanja program u programskom jeziku Java se mora prevesti u odgovarajući bajtkod. Svaka klasa u programu može se prevoditi nezavisno od drugih klasa programa. Prevođenje jedne klase obavlja Java prevodilac koji se pokreće iz komandne linije na sledeći način:

C:\> javac Zdravo.java

Pokretanje programa

Pokretanjem izvršavanja neke klase započinje se izvršavanje metode u toj klasi sa službenim imenom main.U datom primeru klase Zdravo ta metoda se sastoji od samo jedne naredbe kojom se ispisuje određena poruka na ekranu. Izvršavanje dobijenog bajtkoda neke klase obavlja se Java interpreterom koji se pokreće komandom java:


C:\> java Zdravo.java

Кada se program izvrši na ekranu se ispisuje poruka: Zdravо svete!
