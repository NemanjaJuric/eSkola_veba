Funkcija @each

Each petlja je logička struktura koja ima za cilj iteriranje kolekcija, što su liste i mape u programskom jeziku Sass. Zbog toga each petlja radi onoliko puta koliki je broj elemenata u kolekciji (elemenata u listi, parova ključeva/vrednosti u mapi).
Funkcija @each ima sledeći oblik:
  1. naredba @each
  2. ime promenljive
  3. ključna reč in
  4. kolekcija za iteraciju
Funkcija @each ima sledeći oblik:
@each $var in <list or map>
Pri čemu $var može biti bilo koje ime promenljive, a je izraz koji vraća listu ili mapu.
Primer each funkcije sa listom:
$speakers: bob-banker, patty-plume, sandra-smith; @each $speaker in $speakers { .#{$speaker}-profile{ background-image: url('/img/#{$speaker}.png') } }
Primer each funkcije sa mapama:
$font-sizes:(tiny: 8px, small: 11px, medium: 13px, large: 18px); @each $name, $size in $font-sizes{ .#{$name}{ font-size: $size; } }
