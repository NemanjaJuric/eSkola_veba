Tipovi podataka

Sve vrednosti u programskom jeziku Sass, ili tačnije SassScript-u su vezane za određeni tip podataka. Postoji sedam tipova podataka:
  • string (“Hello world”)
  • number (1.2, 16, 21px )
  • color (orange, rgb (255, 0, 0), #BADA55)
  • bool (true of false)
  • null (null)
  • list (0.5px 1px 2px )
  • map (npr. )
Možemo da izvodimo neke matematičke operacije sa brojevima i bojama, ali ne i sa nizovima. Moguće je izvrsavati neke specifične funkcije sa listama i mapama
