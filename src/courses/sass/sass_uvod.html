Uvod u Sass


Sta je Sass?

Sass je skript programski jezik koji se interpretira u CSS. Pojavio se u novembru 2006. godine. Prvobitno ga je dizajnirao Hempton Kejtlin, a razvili ga Natali Vajzenbaum i Kris Epstajn. Sass koristi dve sintakse. Prva,"uvučena sintaksa", koristi uvlačenje za odvajanje blokova koda. Nova sintaksa,SCSS, koristi zagrade za odvajanje blokova koda i znak tačka-zarez za odvajanje linija unutra bloka. Uvučena sintaksa i SCSS fajlovi imaju ekstenzije .sass i .scss.
Za potrebe ovog kursa, koristićemo SCSS sintaksu.

CSS je jezik formatiranja pomoću kog se definiše izgled veb-stranica.
Jedan od najvećih nedostataka jezika CSS je održavanje postojećeg koda kod veb-stranica koje koriste velike CSS datoteke. Takve datoteke mogu imati i po nekoliko hiljada linija koda.

Zašto koristiti Sass?

  • Sass je modularan pa se umesto u jednoj, stilovi mogu pisati u više datoteka koje mogu činiti logičke celine.
  • Koristi DRY("don't repeat yourself") princip, što znači da nema ponavljanja
  • Njegovom upotrebom sadržaj veb strana se prilagođava telefonima, tabletima i desktop računarima.
