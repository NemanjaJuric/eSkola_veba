Mixin sa argumentima

Mixin-i takođe mogu imati svoje argumente koji mu omogućavaju prilagođavanje u različitim situacijama. Argumenti se navode posle naziva mixin-a i predstavljaju listu naziva promenljivih okruženih zagradama.

Primer mixin-a sa argumentima. @mixin rounded($radius){ border-radius: $radius; }
Prilikom pozivanja mixin mora imati isti broj argumenata kao prilikom definisanja.
.warning -button{ @include rounded(6px); background-color: orange; }
Primer mixin-a sa više argumenata. @mixin (){ }
Zadatak: Definisati mixin za box koji kao argumente ima gornju, donju, levu i desnu marginu.
