Mixin

Mixin je blok koda koji omogućava da pravite grupe CSS deklaracija koje ponovo želite da koristite na svom sajtu. Možemo preneti i vrednosti kako bi mixin postao fleksibilniji.
Definišu se sa @mixin, zatim sledi naziv miksina, potencijalno argumente i blok sa sadržajem mixin-a.

Primer mixin-a koji definiše velika slova. @mixin large-text { font: { family: Arial; size: 24px; weight: bold; } color: purple; }
Primer mixina koji prikazuje HTML elemente fleksibilnije. .row { display: -webkit-flex; display: flex; }
Mixin se uključuje u dokument korišćenjem @include naredbe praćenog imenom tog miksina. .page-title { @include large-text; padding: 3px; margin-top: 20px; }
Takođe, možemo koristiti jedan mixin unutar drugog na sledeći način:
@mixin rounded{ border-radius: 6px; } @mixin box { @include rounded; border:1px solid #333; }
Zadatak: Definisati mixin za iskošena slova, crvene boje, veličine 16px.
