Null

Ovde se bavimo određenim scenarijem, jer postoji jedinstvena vrednost koja ima null kao tip podataka: null. Zaista, null je u isto vreme vrednost i tip podataka, čineći vrlo specifican element programskog jezika Sass.
Imajte na umu na moraju biti mala slova i citirana da bi bila tipa null. NULL ili bilo koja druga varijacija je tipa string.

$type: type-of(null); // null $type: type-of(NULL); // string $type: type-of('null'); // string $type: type-of('n' + 'u' + 'LL'); // string
Obično se null koristi za opisivanje prazne vrednosti koja će se popuniti kasnije ili prazno stanje koje nije ni istinito ni lažno. Zbog toga je null vrlo koristan: kada se proceni kao CSS vrednost Sass će izostaviti deklaraciju u potpunosti. Iako se može činiti čudnim ovo ponašanje zapravo vrlo korisno pri izgradnji miksina sa opcionalnim argumentima. Umesto da testiramo svaki argument i proverimo da ili ima vrednost, možemo iskoristi da null vrednost ne bude izlazna.
@mixin absolute($top: null, $right: null, $bottom: null, $left: null) { position: absolute; top: $top; right: $right; bottom: $bottom; left: $left; }
