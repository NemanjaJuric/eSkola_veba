Ugnježdavanje

Prilikom pisanja veb-stranica primećujemo da HTML ima jasnu ugnježdenu i vizuelnu hijerarhiju.
Kod CSS-a to nije slučaj, dok Sass omogućava da blokovi koda budu umetnuti jedan u drugi.
Primer
nav { ul { margin: 1px; padding: 2px; } li { display: inline-block; } a { display: block; padding: 5px 10px; text-decoration: none; } } se prevodi kao nav ul { margin: 1px; padding: 2px; } nav li { display: inline-block; } nav a { display: block; padding: 5px 10px; text-decoration: none; }
Ugnježdavanje je jako korisno kada posmatramo velike CSS fajlove, omogućava nam da lakše rastumačimo strukturu fajla bez njegovog čitanja.
