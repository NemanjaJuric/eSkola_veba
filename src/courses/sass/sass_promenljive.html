Promenljive


Najispravniji način da upotrebite SassScript je da koristite promenljive. Promenljive se koriste za skladištenje informacije koje želimo ponovo da koristimo. Možemo čuvati stvari kao što su boje,font ili bilo koje informacije koje želimo ponovo da koristimo. U Sass-u promenljive se deklarišu simbolom za dolar $. Vrednost promenljivima dodaje se pomoću znaka dvotačka(:).
Da bi se inicijalizovala promenljiva ili da bi joj se dodelila određena vrednost koristi se sledeća sintaksa:

$height: 6px;

Onda je mozemo pozvati na sledeći način: #main { height: $height; }
SassScript podržava osam tipova podataka:
  • brojeve(1.2, 16, 8px)
  • stringove("tree", 'tree', tree)
  • Bulovu algebru(true, false)
  • boje(rgba(255,0,0,0.1), red)
  • mape od jedne do druge vrednosti(key1: value1, key2: value2)
  • liste vrednosti, odvojene razmacima ili zarezima(0.5px 1px 0 2px, Helvetica, Arial, sans-serif)
  • nulls(null)
  • funkcijske reference

Područje važenja

Promenljive su dostupne samo u okviru ugnježdenih selektora tamo gde su definisane. Ako su definisane van ugnježdenih selektora, onda važe svuda.
Takodje, mogu biti definisane oznakom !global, i u tom slučaju su postojane svuda.
Primer #main { $height: 5px !global; height: $height; } #sidebar { height: $height; } prevodi se u #main { height: 5px; } #sidebar { height: 5px; }
