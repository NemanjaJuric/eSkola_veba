Radni okvir Spring

Spring je jedan od najpopularnijih radnih okvira za programski jezik Java. Spring je radni okvir otvorenog koda, potpuno je besplatan za korišćenje i poseduje veliki broj korisnika koji ga konstantno unapređuju. Izvorni kod je dostupan na platformi GitHub.

Tvorac radnog okvira Spring je Rod Jonhson, a prva verzija objavljena je u oktobru 2002. godine. Napravljen je sa ciljem da značajno olakša razvoj Java aplikacija. Od tada se konstantno razvija, čini ga veliki broj radnih okvira koji daju rešenja za sve poznate probleme u razvoju softvera. Bez obzira na tip aplikacije koju je potrebno napraviti, postoji Spring projekat koji može olakšati razvoj takve aplikacije. Spring je modularan radni okvir, a zavisnost između modula je minimalna, što omogućava korišćenje samo onog dela radnog okvira koji odgovara potrebama projekta koji se razvija. Spring je takođe neinvazivan radni okvir, osim Spring anotacija ne postoji mnogo indikacija da je u pitanju Spring aplikacija.

Na zvaničnom sajtu može se pronaći kompletna Spring dokumentacija, linkovi za preuzimanje Spring biblioteka, kao i razni tutorijali i primeri aplikacija pisanih pomoću radnog okvira Spring.

Na sledećoj slici dat je prikaz svih Spring projekata.



Spring projekti

Radni okvir Spring Framework

Radni okvir Spring Framework predstavlja osnovu familije radnih okvira Spring. Ovaj radni okvir sadrži osnovne biblioteke na kojima se zasnivaju svi drugi radni okviri. Sastoji se od 20 različitih modula čiji šematski prikaz je dat na slici sledećoj slici.



  • Core Spring Container - Centralni deo radnog okvira Spring Framework jeste Spring kontejner. Njegov zadatak jeste kreiranje i povezivanje objekata koji čine jednu aplikaciju, kao i upravljanje kompletnim životnim ciklusom tih objekata. U osnovi kontejnera jeste princip umetanja zavisnosti.
  • AOP Module - Spring daje podršku za aspektno orijentisano programiranje, što omogućava da se funkcionalnosti koje prožimaju aplikaciju odvoje od objekata na koje se primenjuju.
  • Data Access and Integration - Rad sa bibliotekom JDBC dovodi do pisanja većeg broja linija koda, kao i koda koji se ponavlja. Ovaj modul olakšava pristup podacima, kao i komunikaciju između biznis sloja aplikacije i sloja prezistencije. Takođe, nudi sloj apstrakcije oko Java biblioteka koje se koriste za asinhronu integraciju sa drugim aplikacijama.
  • Web and Remoting - Za razvoj veb aplikacija najčešće se koristi Model View Controller (MVC) arhitektura, koja razdvaja deo aplikacije koji se odnosi na korisnički interfejs od dela koji sadrži logiku aplikacije. U okviru ovog modula nalazi se podrška za takvu arhitekturu.
  • Instrumentation - Ovaj modul daje podršku za dodavanje agenata u JVM.
  • Testing - Modul za testiranje Spring aplikacija, koji nudi podršku za pisanje jediničnih i integracionih testova.

Radni okvir Spring Boot

Kao što radni okvir Spring Framework ima cilj da olakša razvoj Java aplikacija, tako Spring Boot projekat ima cilj da olakša razvoj aplikacija pisanih pomoću Spring radnih okvira. Spring Boot omogućava kreiranje Spring aplikacije koju je moguće brzo pokrenuti. Spring Boot automatski konfiguriše kako Spring biblioteke, tako i sve druge biblioteke koje je moguće integrisati sa radnim okvirom Spring. Spring Boot takođe pruža početne zavisnosti (biblioteke) prilagođene potrebama aplikacije.

Drugim rečima, Spring Boot sadrži uobičajene konfiguracije svega što je programeru Spring aplikacije potrebno, pružajući potpuno funkcionalnu aplikaciju, koju je samo potrebno nadograditi i prilagoditi svojim potrebama.

Radni okvir Spring Data

Spring Data je radni okvir koji olakšava rad sa relacionim i nerelacionim bazama podataka. Spring Data olakšava integraciju sa drugim radnim okvirima za perzistenciju, od kojih je najpopularniji Hibernate. Jedna od mnogih zanimljivih karakteristika projekta jeste i automatsko generisanje upita na osnovu imena metoda.

Radni okvir Spring Cloud

Spring Cloud je projekat koji se nadograđuje na Spring Boot i omogućava brz razvoj distribuiranih sistema, omogućujući korisnicima lak razvoj i podizanje servisa/aplikacija u distribuiranim sistemima.

Radni okvir Spring Cloud Data Flow

Spring Cloud Data Flow je radni okvir za procesiranje serije ili toka podataka u arhitekturama baziranim na mikroservisima.

Radni okvir Spring Security

Spring Security predstavlja radni okvir za autentifikaciju i kontrolu pristupa, koji značajno olakšava implementaciju sigurnosnih aspekata Java aplikacija. Pored raznovrsne podrške za autentifikaciju i autorizaciju, Spring Security pruža i podršku za zaštitu od uobičajenih načina za zlonameran hakerski napad, kao i implementaciju sigurnosnih protokola i standarda. Spring Security se bazira na principu aspektno orijentisanog programiranja.

Radni okvir Spring Integration

Spring Integration projekat ima za cilj olakšavanje integracije poslovnih aplikacija pružajući podršku za implementaciju ustaljenih obrazaca u ovom domenu. Ovaj radni okvir omogućava lagani sistem za razmenu poruka između Spring aplikacija, ali i integraciju sa spoljnim sistemima koristeći koncept deklarativnih adaptera.

Radni okvir Spring Web Flow

Spring Web Flow se nadograđuje na Spring MVC (koji je deo radnog okvira Spring Framework) i pruža podršku za lak razvoj veb aplikacija koje imaju kontrolisan tok, odnosno niz određenih koraka koji koji vode ka izvršavanju nekog poslovnog zadatka. Primer ovakve aplikacije može biti aplikacija za kupovinu putem interneta ili prijavljivanje na let, a zajednička im je potreba za održavanjem stanja.

Radni okvir Spring Session

Spring Session projekat pruža API i implementaciju upravljanja korisničkim sesijama, sa ciljem da prevaziđe ograničenja HTTP sesija. Prednost ovog projekta jeste to da upravljanje sesijom nije vezano za konkretnu implementaciju kontejnera.
