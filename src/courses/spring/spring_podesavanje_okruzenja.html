Podešavanje okruženja

Da bi se koristio radni okvir Spring Framework, potrebno je preuzeti Spring JAR fajlove, a zatim ih dodati u Eclipse projekat.

Potrebno je pristupiti stranici https://repo.spring.io/release/org/springframework/spring/ koja predstavlja Spring repozitorijum, u okviru koga se nalaze sve dosadašnje verzije. Nakon odabira poslednje verzije otvara se sledeća stranica:


Potrebno je preuzeti datoteku spring-framework-5.0.9.RELEASE-dist.zip. Preuzetu zip datoteku je potrebno raspakovati i JAR fajlove iz datoteke libs kopirati u Eclipse projekat. U okviru Eclipse projekta napraviti folder lib i u njega kopirati JAR fajlove.


Desnim klikom na ime projekta potrebno je odabrati opciju Properties, a zatim u okviru Java Build Path/Libraries kliknuti na AddJARs i dodati JAR fajlove iz datoteke lib.
