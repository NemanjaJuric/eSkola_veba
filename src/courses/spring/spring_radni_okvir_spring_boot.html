Radni okvir Spring Boot

Spring Boot predstavlja nadogradnju radnog okvira Spring Framework. Napravljen je sa ciljem da značajno olakša i ubrza razvoj Spring aplikacija. Spring Boot vodi računa o svim tehničkim detaljima projekta i omogućava programerima da se na početku rada na projektu fokusiraju na logiku aplikacije koju razvijaju. Svaki Spring Boot projekat ima kao osnovnu zavisnost spring-boot-starter-parent projekat. Projekat spring-boot-starter-parent nudi početne zavisnosti koje su potrebne svima, bez obzira na tip projekta koji se razvija. Takođe vodi računa i o kompatibilnosti između verzija zavisnosti. Još jedna njegova prednost je i automatska konfiguracija. Spring Boot će za svaku zavisnost automatski pokušati da kreira i poveže odgovarajuća zrna, čime se značajno smanjuje eksplicitna xml i Java konfiguracija.



Spring Boot nudi veliki broj početnih projekata namenjenih za različite vrste aplikacija. Na početku rada na projektu potrebno je odabrati početne projekte koji odgovaraju namenama aplikacije koja se razvija, što omogućava brzo započinjanje projekta. Za kreiranje veb aplikacija Spring Boot nudi spring-boot-starter-web projekat koji koristi Spring MVC i Tomcat kao ugrađeni veb server.
