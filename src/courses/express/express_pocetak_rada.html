Početak rada u Express.js

Pokretanje servera

Korišćenjem Express.js okruženja, server se pokreće tako što nakon kreiranja JavaScript fajla index.js sa kodom: var express = require('express'); var app = express(); app.get('/', function(req, res){ res.send("Zdravo svete!"); }); app.listen(2020); isti se pokrene iz CMD komandom:
> node index.js

Da je server pokrenut kako treba, može se proveriti u nekom internet pretraživaču (Mozilla Firefox, Google Chrome ...) tako što pozivom adrese:
http://localhost:2020

odgovor treba da bude poruka "Zdravo svete!" kao na slici:


Aplikacija radi tako što se u prvoj linij koda pristupi modulu express funkcijom require a zatim se kreira instanca modula i čuva se u promenljivoj var app na kojoj se zatim definiše šta će se dogoditi kada server dobije get zahtev.

Prvi argument funkcije get jeste stirng kojim se zadaje ruta '/' (u sledećem delu biće više reči o rutama) dok je drugi argument funkcija (callback function) koja će biti pozvana kada server bude dobio get zahtev sa zadatom rutom.
Ta funkcija ima argumente req (skraćeno od request - zahtev) i res (skraćeno od response - odgovor).
Objekat req predstavlja Http zahtev sa raznim propertima koji nose informacije o zahtevu.
Objekat res predstavlja Http odgovor koji će server poslati kada dobije Http zahtev, a funkcijom res.send() u ovom slučaju klijentu će biti poslat objekat koji nosi poruku "Zdravo svete!".

Funkcija app.listen definisana sa:
app.listen(port, [host], [backlog], [callback]);
pokreće server na portu (broj koji je zadat jedinim obaveznim parametrom port) i domenu koji je zadat parametrom host.
backlog je maksimalan broj Http zahteva koje server može da stavi na red za izvršavanje. Podrazumevana vrednost je 511.
callback je funkcija koja će se pozvati kada se server pokrene tj. kada počne sa prihvatanjem Http zahteva.
