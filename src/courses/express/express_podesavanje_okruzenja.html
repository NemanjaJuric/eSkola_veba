Podešavanje okruženja za rad

Uvod

Express.js je framework (programski okvir) tj. standardni serverski okvir za Node.js. Razvio ga je TJ Holowaychuk, a održavan je od strane Node.js fondacije. Za razumevanje istog, potrebno je osnovno poznavanje HTML, JavaScript kao i klijent-server arhitekture.

Osnovno

Potrebno je instalirati tekstualni editor (npr. Notepad++ za Windows, Vim za LINUX) ili WebStorm, Visual Studio itd. za kucanje programskog koda. Takođe treba instalirati Node (za odgovarajući operativni sistem) koji se može pronaći na:
https://nodejs.org/en/download/

Testirati zatim, da li je instalacija uspešno završena tj. kreirati main.js fajl sa sledećim kodom: /*Test program u node.js*/ console.log("Zdravo svete!"); a zatim iz Command Prompt (CMD) pokrenuti program komandom:
> node main.js
Karakter '>' na početku označava samo da je u pitanju komandna linija, ne treba ga unositi tj. nije deo komande.
Ako je sve u redu, ispis u konzoli će biti:
> Zdravo svete!

Instaliranje paketa

Za instaliranje paketa u Node najčešće se koristi Node Package Manager (npm), a postoje dva načina da se paket instalira:
1. Globalno (za instaliranje razvojnih alata) komandom:
> npm install -g <ime-paketa>
2. Lokalno (za instaliranje razvojnih okvira - framework i biblioteka) bez dela –g komandom:
> npm install <ime-paketa>
gde se <ime-paketa> zameni sa imenom paketa koji instaliramo.

Inicijalizacija projekta i instaliranje paketa express

Iz CMD kreirati novi folder, nazvati ga hello-world i ući u isti:
> mkdir hallo-world
> cd hallo-world

Zatim, kreirati package.json inicijalni fajl komandom:
> npm init

Pritiskajući enter preskočiti ili popuniti odgovarajuće informacije, nakon čega je kreiran package.json fajl a prikaz u CMD treba da izgleda ovako:

Sada ostaje da se instalira express komandom:
> npm install --save express

Express je instaliran u folderu node_modules.
--save (može se zameniti sa –S) znači da će informacija o tome da projekat zavisi od paketa express biti dodata u package.json fajl. Kada je potrebno preneti projekat na neko drugo mesto, ne treba prenositi folder node_modules već samo na novom mestu sledećom komandom instalirati sve pakete od kojih projekat zavisi:
> npm install
