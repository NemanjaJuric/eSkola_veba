Pug

Šta je pug?

Pug je šablon za Express koji treba da pojednostavi HTML kod, tj. daleko čitljiviji pug kod se renderuje u HTML kod i takav se šalje klijentu.

Pug se instalira komandom:
npm install --save pug
i nije potrebno posebno ga učitati sa require, već samo treba postaviti pug kao šablon koji će se koristiti, kao i folder gde će se nalaziti .pug fajlovi. U ovom slučaju taj folder je zadat na primer relativnom putanjom "./views" u odnosu na index.js fajl gde je i neophodno dodati kod: app.set('view engine', 'pug'); app.set('views','./views'); U folderu views kreirati fajl .pug fajl sa imenom na primer moj_prvi_view.pug i kodom: doctype html html head title = "Moj prvi pug" body p.myclass#myid Zdravo svete! Kada se get pozivom sa rutom npr. "/moj_prvi_template" app.get('/moj_prvi_template', function(req, res){ res.render('moj_prvi_view'); }); renderuje sadržaj moj_prvi_view.pug fajla, klijentu će biti zapravo poslat HTML: <!DOCTYPE html> <html> <head> <title>Moj prvi pug</title> </head> <body> <p class = "myclass" id = "myid">Zdravo Svete!</p> </body> </html>

Prosleđivanje parametara

Moguće je dinamičko korišćenje šablona tj. pri renderovanju .pug fajlova, za prikaz proslediti neke proizvoljne vrednosti na sldeći način: var express = require('express'); var app = express(); app.set('view engine', 'pug'); app.set('views','./views'); app.get('/dinamicki_template', function(req, res){ res.render('dinamicki_view', { name: "eŠkola veba", url:"http://edusoft.matf.bg.ac.rs/eskola_veba" }); }); app.listen(2020); Ovde se za get poziv sa rutom "/dinamicki_template" klijentu vraća dinamicki_view.pug fajl kojem se pri renderovanju prosleđuju vrednosti parametara name i url. dinamicki_view.pug se nalazi u folderu views i izgleda na primer ovako: html head title=name body h1=name a(href = url) URL Testiramo iz pretražovača sa:
http://localhost:2020/dinamicki_template

Ispitivanje uslova

U zavisnosti od određenih uslova, renderovani .pug fajl moze da nosi različit prikaz. To se reguliše if/else ispitivanjem na primer: html head title Primer ispitivanja uslova body if(student) h1 Zdravo, #{student.ime} else a(href = "/prijavljivanje") Uloguj se! U slučaju da je klijent (student) uspošno ulogovn tj. objekat student je različit od null, biće prikazana odgovarajuća poruka, a u suprotnom biće prikazan link ka ruti "/prijavljivanje" kojom se klijent šalje na stranu gde može da se uloguje.
objekat student se prosleđuje kao drugi argument funkcije render: res.render('/dinamicki_view',{ student: {ime: "Milan", starost: "20"} });
