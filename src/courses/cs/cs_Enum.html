Nabrojivi tip

Nabrajanje nam omogućava da kreiramo skup konstanti na koje možemo da referenciramo u programu. Nabrajanje je vrednosni tip.
Npr. Program koji treba da referencira na redosled meseci u godini.
Redosled je: Januar, Februar, Mart, April, Maj, Jun, Jul, Avgust, Septembar, Oktobar, Novembar, Decembar.

Nabrajanje se deklariše korišćenjem ključne reči.
Primer sintekse nabrojivog tipa:
enum ime_nabrajanja { lista-konstanti }

ime_nabrajanja : Ime koje se dodeljuje nabrajanju. Obično je prvo slovo imena nabrajanja je veliko.
lista_konstanti : Lista konstanti koji se nalazi u nabrajanju.
enum Godina { Januar, Februar, Mart, April, Maj, Jun, Jul, Avgust, Septembar, Oktobar, Novembar, Decembar } Ovim nabrajanjem definišemo dvanaest konstanti, pri čemu svaki mesec ima svoju poziciju.
Januar ima podrazumevanu vrednost 0, dok svaka naredna konstanta ima vrednost za 1 veću od prethodne.
Podrazumevani tip za konstante u nabrajanju je Int.

Konstante u nabrajanju možemo i inicijalizovati, kao što je prikazano u sledećem primeru: enum Godina { Januar=1, Februar=2, Mart=3, April=4, Maj=5, Jun=6, Jul=7, Avgust=8, Septembar=9, Oktobar=10, Novembar=11, Decembar=12 }

Da bismo pristupili odredjenom mesecu, koristimo sledeću sintaksu: Console.WriteLine((int)Godina.Decembar); Napomena1
Ovde se koristi operator za konverziju (int), da bi se dobio broj meseca Decembra u godini. Bez ove konvrzije Godina.Decembar vratiće string "Decembar". Više o konverziji biće u nekoj od narednih lekcija.


Napomena2
Ista konstanta se ne može koristiti u deklaracijama različitih nabrojivih tipova podataka.
Npr.

enum Radnidani { pon, uto, sre, cet, pet };
enum vikend { pet, sub, ned };
su NEKOREKTNE, jer se konstanta pet pojavljuje u definiciji oba tipa.
