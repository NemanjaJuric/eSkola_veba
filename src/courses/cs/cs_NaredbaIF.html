Naredba IF

Do sada smo se bavili programima koji su imali linijsku strukturu.
To znači da su se koraci izvršavali redom, jedan za drugim, bez preskakanja.
Osim linijske, postoji i razgranata struktura, u kojoj se svaki korak može izvršiti jednom,ali i ne mora.
To postižemo zahvaljujući uslovu "AKO".

Pitamo se AKO nešto važi, onda izvršiti jednu ili više naredbi.
Tj. da li je uslov ispunjen. Ako uslov nije ispunjen neće se izvršiti naredba ili se može izvršiti drugi blok naredbi.

Naredba if nam može vratiti vrednost true (ako je uslov ispunjen) i false (ukoliko uslov ne vazi).
If naredba može se koristiti za obavljanje jednosmernog, dvosmernog, višesmernog ili ugnježdenog testiranja.

Sintaksa naredbe if je: if(uslov) { naredba; } Ovakva naredba if smatra se jednosmernim testiranjem.
Pitamo se da li je uslov ispunjen. Ako jeste izvršiće se naredba, a ukoliko nije, samo će preskočiti ovaj korak.

Napomena 1
Iza zatvorene male zagrade u if-testu ne stavlja se znak tačka zarez.
Primer 1. Napisati program, u kom se sa tastature unosi jedan broj, i proverava da li je broj negativan.
Ako jeste, ispisati u consoli poruku "Uneli ste negativan broj." using System; namespace ConsoleApp4 { class Program { static void Main(string[] args) { double a; string b; Console.WriteLine("Unesite jedan broj"); b=Console.ReadLine(); a = Convert.ToInt32(b); if(a<0) { Console.WriteLine("Uneli ste negativan broj."); } Console.ReadKey(); } } } Kada unesemo negativan broj, ovaj program proverava i ispisuje poruku "Uneli ste negativan broj." Ukoliko nije unet negativan broj, program nam neće ništa ispisati (proverava uslov, i obzirom da ne važi, zanemariće ostale korake)

Dvosmerna testiranja imaju sledeću sintaksu:
if(uslov) { naredba1; } else { naredba2; } Primer 2. Napisati program, u kom se sa tastature unosi ceo broj, i proverava da li je broj paran.
Ako jeste, ispisati u consoli poruku "Uneli ste paran broj.",a ukoliko nije poruku "Uneli ste neparan broj." using System; namespace ConsoleApp5 { class Program { static void Main(string[] args) { int a; string b; Console.WriteLine("Unesite jedan broj"); b=Console.ReadLine(); a = Convert.ToInt32(b); if(a%2==0) { Console.WriteLine("Uneli ste paran broj."); } else { Console.WriteLine("Uneli ste neparan broj."); } Console.ReadKey(); } } }

Višesmerna testiranja ili višestruko grananje omogućava ispitivanje više uslova.
Sintaksa višestrukog grananja naredbe if izgleda:
if(uslov1) { naredba1; } else if(uslov 2) { naredba2; } else if(uslov 3) { naredba3; } else { naredba4; naredba 5; }

Ako je vrednost prvog uslova true izvršiće se naredba 1, ostali koraci se onda zanemaruju. Ako je vrednost prvog uslova false, proverava se drugi uslov. Ako je vrednost true onda tu se izvodi drugi blok naredbi. A ako je false, onda se proverava treći uslov i tako redom.

Primer 3. Napisati program, u kom se sa tastature unosi ceo broj, i proverava da li je broj jednak broju 10, manji ili veći od tog broja.
U consoli u zavisnosti od ispunjenog uslova ispisuje jednu od sledećih poruka:
"Uneti broj je jednak broju 10", ili "Uneti broj je veći od 10", ili poruku "Uneti broj je manji od 10". using System; namespace ConsoleApp6 { class Program { static void Main(string[] args) { double a; string b; Console.WriteLine("Unesite jedan broj"); b=Console.ReadLine(); a = Convert.ToInt32(b); if(a==10) { Console.WriteLine("Uneti broj je jednak broju 10"); } else if(a>10) { Console.WriteLine("Uneti broj je veci od 10"); } else { Console.WriteLine("Uneti broj je manji od 10"); } Console.ReadKey(); } } }

Naredbu if možemo staviti unutar neke druge naredbe if.
Za unutrašnju naredbu if kaže se da je ugnježdena unutar druge.

Sintaksa ugnježdene naredbe je:
if(uslov) { naredba1; } else { if(uslov 2) { naredba2; } else { naredba3; } } Primer 4. Napisati program, u kom se sa tastature unosi ceo broj, i prvo se proverava da li je broj jednak nuli, ako jeste ispisuje poruku "Uneti broj je 0.",
Ukoliko nije proverava se da li je unet paran ili neparan broj i ispisuje odgovarajuća poruka. using System; namespace ConsoleApp7 { class Program { static void Main(string[] args) { double a; string b; Console.WriteLine("Unesite jedan broj"); b=Console.ReadLine(); a = Convert.ToInt32(b); if(a==0) { Console.WriteLine("Uneti broj je 0."); } else { if (a % 2==0) { Console.WriteLine("Broj je paran."); } else { Console.WriteLine("Broj je neparan."); } } Console.ReadKey(); } } } Napomena 2
Izmedju ključnih reči else i if postoji razmak, jer su to dve zasebne ključne reči.
