For petlja

Sintaksa for naredbe je:

for ( inicijalizacija; uslov; inkrementacija ){
blok naredbi;
}

Prvo se izvrši inicijalizacija, i zatim se proverava uslov. Ukoliko je njegova vrednost tačna , ulazi se u blok naredbi koji je naveden u petlji.
Kada se izvrše sve naredbe izvršava se inkrementacija i tok programa se vraća na ponovno ispitivanje uslova. Opisani proces se ponavlja sve dok je ispunjen uslov.
Ako uslov nije ispunjen izlazi se iz petlje i nastavlja se sa izvršavanjem naredbi koje su navedene posle for petlje.

While i for petlja predstavljaju dva različita načina da uradimo istu stvar. Iako uvek možemo rešiti problem koristeći bilo koju od ove dve petlje,
nekada je zgodnije koristiti jednu, a nekada drugu. Petlja for je pravljena za slučajeve gde, kada dođemo do petlje,
znamo koliko puta ćemo je izvršiti. Bilo da je to fiksan broj ili vrednost promenljive.
Sa druge strane while petlju koristimo u slučajevima gde ne znamo unapred koliko puta će se petlja ponoviti.

Primer1. . Ispisati poruku “Zdravo svete” 9 puta.
using System; namespace ConsoleApp2 { class Program { static void Main(string[] args) { int i; for (i = 1; i <= 9; i++) { Console.WriteLine("Zdravo svete"); } } } } Nakon izvrsenja, rezultat programa ce biti:
Zdravo svete
Zdravo svete
Zdravo svete
Zdravo svete
Zdravo svete
Zdravo svete
Zdravo svete
Zdravo svete
Zdravo svete

Inicijalizacijom, promenljiva i je jednaka 1 u for petlji. Zatim, proverom uslova da je 1<=9, ulazi se u blok naredni gde se prvi put ispisuje poruka,
nakon cega se i inkrementira (i++). Ponovnom proverom uslova da je 2<=9,drugi put se ispisuje poruka, zatim ponovna inkremetacija gde I dobija vrednost 3.
For petlja ce se izvrsavati sve dok se poruka ne ispise 9 puta sto je naglaseno u uslovu petlje.

Primer2. Izračunavati zbir svih parnih brojeva u intervalu od nula, do nekog zadatog broja n. using System; namespace ConsoleApp3 { class Program { static void Main(string[] args) { int n; Console.WriteLine("Unesite prirodan broj: "); n = int.Parse(Console.ReadLine()); int i; int zbir = 0; for(i=0;i<=n;i+=2) { zbir += i; } Console.WriteLine("Zbir je jednak " + zbir); } } } Za zadati broj 8, zbir ce biti jednak 2+4+6+8 = 20.
Nakon unosa broja(primer kada unosimo broj 8), vrsi se inicijalizacija promenljive i ( i = 0 ). Nakon provere uslova da je 0<=8,
izvrsava se blok naredbi gde se zbiru dodaje 0, zatim inkrementacijom se i povecava za 2 (i+=2). Nakon toga se proverava uslov da li je 2<= 8,
I zatim se broj 2 dodaje zbiru. Inkrementacijom, i dobija vrednost 4, koji se nakon provere uslova dodaje zbiru ( prethodna vrednost zbira je 2,nakon sabiranja 2 + 4, zbir dobija vrednost 6 ).
For petlja ce se izvrsavati dokle god je ispunjen uslov da je i<=n, sto u nasem slucaju znaci da ce se sabrati brojevi 0,2,4,6,8, I zbir ce iznositi 20.

Zadatak1. Za zadati prirodan broj, sabrati sve neparne prirodne brojeve manje od njega,zatim ispisati zbir.
