Konzolni programi i programi zasnovani na prozorima

Postoje konzolni programi i programi zasnovani na prozorima. Koja je njihova razlika?
Kod konzolnih programa, mi pokrećemo program, a programski kod odredjuje sled dogadjaja.
Unosimo podatke kada se to zatraži, a program izbacuje rezultate kada to želi (najčešće je unapred odredjeno).
U svakom trenutku je poznato koji će se programski kod sledeći izvršiti.

Dok kod programa zasnovanim na prozorima rad zavisi od toga šta korisnik uradi sa grafičkim korisničkim interfjesom.
Biranjem članova menija, dugmadi ili korišćenjem tastature utičemo na dešavanja unutar programa.
Dok mi nešto ne uradimo, nije poznat programski kod koji sledeći treba da se izvrši.


Podešavanja okruženja

C# programski jezik je formalni jezik za prezentaciju računarskih programa, pomoću njega se računaru,
čoveku čitljivim tekstom, daju instrukcije i naredbe za izvršenje različitih zadataka.
Kako bi se instrukcije zadate u programskom jeziku izvršile, potrebno je prevesti ih na mašinski kod, da bi računar mogao da ih izvrši.
Ovo prevođenje se vrši pomoću kompajlera i prevodioca programa koji prevode zadate instrukcije na mašinski kod.
Upravo ovakav program koristićemo i za programiranje u C# programskom jeziku.

Da bismo napisali program u C# i upoznali se sa alatom za kreiranje softvera, potrebno nam je integrisano okruženje za razvoj programa.
Takvo okruženje u sebi sadrži uređivač/editor teksta, prevodilac, alate za otkrivanje grešaka, alat za rukovanje razvojem velikih programskih sistema i još neke druge alate koji čine rad udobnijim.
U ovom kursu ćemo koristiti Visual Studio koji postoji samo na Windows platformi.

Visual Studio Community je okruženje za razvoj softvera kompanije Microsoft koje integriše razvojni interfejs i sve potrebne alate za kompajliranje različitih programskih jezika.
On predstavlja ograničenu i besplatnu verziju okruženja Visual Studio.Na sledećoj adresi:
ovde izaberite Visual Studio -> Community 2019.
Nakon preuzimanja, potrebno je da se pokrene (.exe) fajl instalacije, a zatim da se prate instrukcije instalacionog programa.
Instalacija zahteva i posedovanje besplatnog Microsoft naloga koji se jednostavno kreira.
Ovaj nalog već imaju svi sa adresom elektronske pošte koja se završava na: hotmail.com, live.com ili outlook.com.

Posle instalacije, pokrenite Microsoft Visual Studio 2019, kako bismo napisali svoj prvi program.
Izaberite File -> New -> Project -> Create New Project
Izaberite Console Application -> Next (slika 1. i 2).



Zatim, imate opciju za davanje imena projektu, kao i lokaciju gde želite da projekat bude sačuvan.
(Ukoliko ne promenite, podrazumevano ime projekta biće ConsoleApplication1 za prvi, ConsoleApplication2 za drugi i tako dalje. Ovde je podešeno da se programi čuvaju u folderu Program na Desktopu, a to možete promeniti klikom na ...).
Kliknete Create kao što je i prikazano na sledećim slikama (slika 3. i 4).




Okuženje za pisanje projekta je prikazano na slici 5(Code editor), dok se pri prevodjenju i pokretanju ovog projekta pojavljuje consola ( slika 6).



Na sledećim slikama opisan postupak kreiranja projekta u programu zasnovan na prozoru
(GUI eng.Graphical User Interface - grafički korisnički interfejs) koji podržava upotrebu miša ili nekog drugog uređaja za prikazivanje.
(O ovakvim projektima biće više reči kasnije).
