String tip

String (niska) je uređeni niz simbola. Stringovi su, u stvari, nizovi UNICODE znakova i deklariše ih korišćenje tipa string.
String je pogodan za karakter-po-karakter obrađivanje i po tome se razlikuje od klasičnih nizova.
Na desni kraj niske se dopisuje karakter ’\0’ (tzv.terminalna nula), time se postiže da ne postoji ograničenje za dužinu niske, ali je neophodno proći kroz celu nisku kako bi se odredila dužina niske. Niske se navode između dvostrukih navodnika. string niska = "Danas je lep dan."; Console.WriteLine(niska); Stringove možemo nadovezivati uz pomoć operatora +.
Primer1. string niska1 = "Danas je lep dan"; string niska2= ",al sutra bice lepsi."; Console.WriteLine(niska1+niska2);

Svojstvo Length

Ovo svojstvo se koristi da bi se dobio broj znakova u stringu, tj. dužina niske. Svojstvo length vraća tip int.
U Primeru 3. ćemo pokazati na koji način možemo odraditi dužinu stringa "Danas je lep dan" . U consoli ćemo ispisati dužinu niske.

Primer 3. using System; namespace ConsoleApp3 { class Program { static void Main(string[] args) { string niska = "Danas je lep dan."; Console.WriteLine("Broj znakova niske: \""+niska +"\" je "+niska.Length); Console.ReadKey(); } } }

Ispis je Broj znakova niske: "Danas je lep dan." je 17

U consoli se ispisuje broj 17, jer pored slova broji i sve ostale znakove u rečenici uključujući i beline i znakove interpukcije.

Metoda Compare()

Metoda Compare() vraća vrednost tipa int koji ukazuje na to da li je prvi string veći, jednak ili manji od drugog stringa. Poređenje se vrši alfabetski nad slovima i numerički nad brojevima. Metoda Compare() koristi sledeća pravila za poređenje:
  • Ako je prvi string veći od drugog stringa, vraća 1
  • Ako je prvi string jednak od drugog stringa, vraća 0
  • Ako je prvi string manji od drugog stringa, vraća -1


Najednostavnija metoda Compare() ima sledeću sintaksu: String.Compare( niska1 , niska 2 );

Primer4. String.Compare("abc","bbc");
U primeru 4, consola će vratiti vrednost -1.
Takođe metoda Compare(), pored ova dva paramtera, prihvata i parametar tipa bool koji nam ukazuje na to da li želimo da poredimo i veličinu slova.
Ova metoda Compare() ima sledeću sintaksu: String.Compare( niska1 , niska 2 , ignorisiSlova );

Gde je ignorisiSlova promenljiva, tipa bool, kojim se označava da li poređenjem želimo da koristimo i veličinu slova niska1 и niska 2 .
Ako postavimo vrednost promenljive ignorisiSlova nа true, onda se veličina slova ne razmatra pri poređenju stringova, a ako postavimo na false, onda se pri poređenju razmatra veličina slova.

U primeru 5, consola će vratiti vrednost 0 ,dok primer broj 6 vratiće vrednost 1 (jer je "BBC" leksikografski veće od "bbc").
Primer 5. String.Compare("BBC","bbc",true);

Primer 6. String.Compare("BBC","bbc",false);

Metoda Equals()


Za proveru da li su dva stringa jednaka možemo koristiti metodu Equals(), koja nam vraća vrednost true ukoliko jesu, i false ako nisu jednaka.

Ova metoda Equals() ima sledeću sintaksu: String.Equals( niska1 , niska 2 );

Takođe, za proveru da li su dva stringa leksikografski jednaka, možemo koristiti i operator ==.

bool rezultatPoredjenja= niska1 == niska 2 ;

Metode ToUpper() и ToLower()

Ove metode služe kako bi se konvretovale veličine slova u stringu.
Metoda ToUpper() vraća string sa svim velikim slovima, dok metoda ToLower() string sa svim malim slovima.
Obe metode vraćaju novi string! Ove metode imaju sledeću sintaksu: niska1.ToUpper(); niska2.ToLower();

Primer 7. String novaNiska1=niska1.ToUpper(); String novaNiska2=niska2.ToLower();

Metoda Substring()

Metoda Substring() služi kako bi se iz nje izvukao podstring. Ova metoda vraća string. Metoda Substring ima sledeću sintaksu: niska1.Substring(index);

Gde je indeks vrednost tipa int koja određuje poziciju znaka od koje treba početi sa čitanjem stringa niska1. Druga verzija ove metode ima sintaksu: niska1.Substring(index,brojKaraktera);

Gde osim indeksa, imamo i brojKaraktera tipa int koja određuje koliko karktera od početnog indeksa uzimamo za novu nisku.
Primer 8.
Od niske "Srecan rodjendan" uz upotrebu metode Substring() dobijamo dva podstinga:
niskaNova1= "rodjendan" ,dok niskaNova2= "rodj" using System; namespace ConsoleApp3 { class Program { static void Main(string[] args) { string niska1 = "Srecan rodjendan"; string niskaNova1 = niska1.Substring(6); string niskaNova2 = niska1.Substring(6,5); Console.WriteLine(niskaNova1 +","+niskaNova2); Console.ReadKey(); } } }
