Uvod u Bootstrap4


Bootstrap 4 je najpopularnije HTML, CSS i JS radno okruženje za kreiranje veb stranica čiji je prikaz prilagodljiv na uređajima svih rezolucija.

Prilično je lak za korišćenje posebno ako ste upoznati sa HTML i CSS osnovama.
Bootstrap 4 je kompatibilan sa modernim pretraživačima (Chrome, Mozzila, Opera, Safari), međutim za razliku od prethodne verzije, Bootstrap 4 ne podržava Internet Explorer 9.

Kako doći do radnog okruženja Bootstrap 4?

Postoje dva načina pomoću kojih se može koristiti:
  • prvi način je preuzimanje ovog radnog okruženja sa adrese www.getbootstrap.com, pri čemu treba ispratiti tamo zadata uputstva za instalaciju;
  • drugi način je uključivanjem iz distribucionog sistema servera CDN (Content Delivery Network).
Za potrebe ovog kursa, primenjivaćemo drugi način.

Svi potrebni fajlovi se uključuju tako što se u okviru etikete <head> navode sledeće linije koda: <!-- Bootstrap 4 CSS --> <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css"> <!-- jQuery biblioteka --> <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script> <!-- Popper JS --> <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.12.9/umd/popper.min.js"></script> <!-- JavaScript --> <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/js/bootstrap.min.js"></script>

Kreiranje prve strane

Da biste omogućili pravilno prikazivanje stranice na svakom uredjaju u okviru etikete <head> treba navesti i : <meta name="viewport" content="width=device-width, initial-scale=1">
width=device-width postavlja da širina stranice bude jednaka širini ekrana.
initial-scale=1 postavlja inicijalni nivo uvećanja stranice, sprečava podrazumevano uvećanje i pomaže pri promeni orijentacije ekrana.

Klase omotača

Omotač je predstavljen etiketom <div> i on može imati dve klase: .container i .container-fluid. Klasa .container formatira omotač tako da bude fiksirane širine, dok u klasi .container-fluid omotač je raširen po celom ekranu.

Zadatak: Napraviti veb stranu koristeći Bootstrap 4, veb strana treba da sadrži proizvoljan tekst i da koristi klasu .container.
