Paginacija (engl. Pagination)

Radi bolje preglednosti, odnosno, da bi se izbeglo beskonačno skrolovanje veb strane uvodi se podela sadržaja na više strana, paginacija. U okruženju Bootstrap 4 to se radi tako što se elementu <ul> doda klasa .pagination, svakom <li> elementu klasa .page-item, a elementima <a> unutar svakog od elemenata liste se doda klasa .page-link. <!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Breadcrumb</title> <meta name="viewport" content="width=device-width, initial-scale=1"> <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css"> <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.12.9/umd/popper.min.js"></script> <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/js/bootstrap.min.js"></script> </head> <body> <div class="container"> <ul class="pagination"> <li class="page-item"><a class="page-link" href="#">Prethodna</a></li> <li class="page-item"><a class="page-link" href="#">1</a></li> <li class="page-item"><a class="page-link" href="#">2</a></li> <li class="page-item"><a class="page-link" href="#">3</a></li> <li class="page-item"><a class="page-link" href="#">Sledeća</a></li> </ul> </div> </body> </html>
Za potrebe ove lekcije umesto pravih linkova stoje #.
Klasom .active je moguće prikazati koja je stranica trenutno prikazana, nju treba dodati elementu <li>. <!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Breadcrumb</title> <meta name="viewport" content="width=device-width, initial-scale=1"> <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css"> <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.12.9/umd/popper.min.js"></script> <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/js/bootstrap.min.js"></script> </head> <body> <div class="container"> <ul class="pagination"> <li class="page-item"><a class="page-link" href="#">Prethodna</a></li> <li class="page-item"><a class="page-link" href="#">1</a></li> <li class="page-item active"><a class="page-link" href="#">2</a></li> <li class="page-item"><a class="page-link" href="#">3</a></li> <li class="page-item"><a class="page-link" href="#">Sledeća</a></li> </ul> </div> </body> </html>
Klasu .disabled dodajemo onom elementu liste kome korisnik trenutno ne može da pristupi. U sledećem slučaju to je link koji vodi na prethodnu stranu koja ne postoji. <!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Breadcrumb</title> <meta name="viewport" content="width=device-width, initial-scale=1"> <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css"> <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.12.9/umd/popper.min.js"></script> <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/js/bootstrap.min.js"></script> </head> <body> <div class="container"> <ul class="pagination"> <li class="page-item disabled"><a class="page-link" href="#">Prethodna</a></li> <li class="page-item active"><a class="page-link" href="#">1</a></li> <li class="page-item"><a class="page-link" href="#">2</a></li> <li class="page-item"><a class="page-link" href="#">3</a></li> <li class="page-item"><a class="page-link" href="#">Sledeća</a></li> </ul> </div> </body> </html>

Veličina i pozicija paginacije

Postoje dve klase kojima se može zadati veličina bloka sa linkovima strana, ove klase se dodaju elementu <ul>, a one su .pagination-lg za veće blokove i .pagination-sm za manje blokove. <!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Breadcrumb</title> <meta name="viewport" content="width=device-width, initial-scale=1"> <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css"> <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.12.9/umd/popper.min.js"></script> <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/js/bootstrap.min.js"></script> </head> <body> <div class="container"> <!--Veliki blok paginacije--> <ul class="pagination pagination-lg"> <li class="page-item"><a class="page-link" href="#">Prethodna</a></li> <li class="page-item"><a class="page-link" href="#">1</a></li> <li class="page-item"><a class="page-link" href="#">2</a></li> <li class="page-item"><a class="page-link" href="#">3</a></li> <li class="page-item"><a class="page-link" href="#">Sledeća</a></li> </ul> <!--Mali blok paginacije--> <ul class="pagination pagination-sm"> <li class="page-item"><a class="page-link" href="#">Prethodna</a></li> <li class="page-item"><a class="page-link" href="#">1</a></li> <li class="page-item"><a class="page-link" href="#">2</a></li> <li class="page-item"><a class="page-link" href="#">3</a></li> <li class="page-item"><a class="page-link" href="#">Sledeća</a></li> </ul> </div> </body> </html>
Takođe, blokove sa paginacijom je moguće i poravnati, postoje levo, desno i centralno poravnanje. Standardno poravnanje je levo pa za njega nije potrebna klasa za poravnanje, ukoliko hoćemo centralno da poravnamo blok sa paginacijom to se radi klasom .justify-content-center koja se dodaje elementu <ul>. Desno porvananje se omogućava dodavanjem klase .justify-content-end elementu <ul>. <!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Breadcrumb</title> <meta name="viewport" content="width=device-width, initial-scale=1"> <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css"> <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.12.9/umd/popper.min.js"></script> <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/js/bootstrap.min.js"></script> </head> <body> <div class="container"> <ul class="pagination"> <li class="page-item"><a class="page-link" href="#">1</a></li> <li class="page-item"><a class="page-link" href="#">2</a></li> <li class="page-item"><a class="page-link" href="#">3</a></li> </ul> <ul class="pagination justify-content-center"> <li class="page-item"><a class="page-link" href="#">1</a></li> <li class="page-item"><a class="page-link" href="#">2</a></li> <li class="page-item"><a class="page-link" href="#">3</a></li> </ul> <ul class="pagination justify-content-end"> <li class="page-item"><a class="page-link" href="#">1</a></li> <li class="page-item"><a class="page-link" href="#">2</a></li> <li class="page-item"><a class="page-link" href="#">3</a></li> </ul> </div> </body> </html>

Paginacija pomoću kartica

Paginaciju pomoću kartica je moguće napraviti pomoću klase .breadcrumb. Klasu .breadcrumb treba zadati elementu <ul>, dok klasu .breadcrumb-item treba dodati svakom elementu te liste. Pogledati sledeći primer. <!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Breadcrumb</title> <meta name="viewport" content="width=device-width, initial-scale=1"> <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css"> <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.12.9/umd/popper.min.js"></script> <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/js/bootstrap.min.js"></script> </head> <body> <div class="container"> <ul class="breadcrumb"> <li class="breadcrumb-item"><a href="#">Prva</a></li> <li class="breadcrumb-item"><a href="#">Druga</a></li> <li class="breadcrumb-item"><a href="#">Treca</a></li> <li class="breadcrumb-item active">Četvrta</li> </ul> </div> </body> </html>
Zadatak: Napraviti paginaciju sa 5 strana, dodati linkove za prethodnu i sledeću stranu, onemogućiti pristup jednoj strani i označiti koja je aktivna. Centrirati blok sa paginacijom i neka blok bude mali.
