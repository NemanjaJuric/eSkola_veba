Crtanje teksta u Canvas-u

Za crtanje teksta u Canvas-u najbitnija svojstva i metode su:

  • font - defini≈°e font teksta
  • fillText(text,x,y) - iscrtava 'popunjen' tekst u canvas elementu
  • strokeText(text,x,y) - iscrtava tekst u canvas elementu

Primer koji demonstrira poziv metode fillText, postavlja font na 30px "Arial" i ispisuje "Hello world":

<canvas id="myCanvasfill" width="200" height="100" style="border:1px solid #d3d3d3;"></canvas> <script> var canvas = document.getElementById("myCanvasfill"); var ctx = canvas.getContext("2d"); ctx.font = "30px Arial"; ctx.fillText("Hello World", 10, 50); </script>

Primer koji demonstrira poziv metode strokeText, postavlja font na 30px "Arial" i ispisuje "Hello world":

<canvas id="myCanvasStroke" width="200" height="100" style="border:1px solid #d3d3d3;"></canvas> <script> var canvas = document.getElementById("myCanvasStroke"); var ctx = canvas.getContext("2d"); ctx.font = "30px Arial"; ctx.strokeText("Hello World",10,50); </script>
