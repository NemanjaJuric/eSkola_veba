Dodavanje slike u HTML5 Canvas-u

Dodavanje slike pomoću elementa HTML5 Canvas se postiže pozivom metode drawImage() kojoj se prosleđuje objekat slike i odredišna tačka. Odredišna tačka definiše gornji levi ugao slike u odnosu na gornji levi ugao Canvas pravougaonika.
Obzirom da metoda drawImage() zahteva objekat slike, neophodno je najpre kreirati sliku i sačekati njeno učitavanje pre nego što se pozove metoda drawImage(). To se može postići pozivom svojstva onload za objekat slike.

<canvas id="myCanvas" width="578" height="200"></canvas> <script> var canvas = document.getElementById('myCanvas'); var context = canvas.getContext('2d'); var imageObj = new Image(); imageObj.onload = function() { context.drawImage(imageObj, 69, 50); }; imageObj.src = 'http://www.html5canvastutorials.com/demos/assets/darth-vader.jpg'; </script>
