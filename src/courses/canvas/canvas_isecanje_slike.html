Isecanje slike u HTML5 Canvas-u

Za isecanje slike koristeći element HTML5 Canvas neophodno je dodeliti vrednost sledećim argumentima metode drawImage(): sourceX, sourceY, sourceWidth, sourceHeight, destWidth, destHeight. Ovi argumenti definišu lokaciju i dimenziju pravougaonika koji se koristi prilikom isecanja slike.

<canvas id="myCanvas" width="578" height="200"></canvas> <script> var canvas = document.getElementById('myCanvas'); var context = canvas.getContext('2d'); var imageObj = new Image(); imageObj.onload = function() { // iscrtavanje isečka slike var sourceX = 150; var sourceY = 0; var sourceWidth = 150; var sourceHeight = 150; var destWidth = sourceWidth; var destHeight = sourceHeight; var destX = canvas.width / 2 - destWidth / 2; var destY = canvas.height / 2 - destHeight / 2; context.drawImage(imageObj, sourceX, sourceY, sourceWidth, sourceHeight, destX, destY, destWidth, destHeight); }; imageObj.src = 'http://www.html5canvastutorials.com/demos/assets/darth-vader.jpg'; </script>
