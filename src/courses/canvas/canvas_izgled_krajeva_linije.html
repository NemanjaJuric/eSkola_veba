Izgled krajeva linije u HTML5 Canvas-u

Da bi se definisao izgled krajeva linije potrebno je postaviti vrednost svojsta lineCap. Postoje tri definisane vrednosti svojstva lineCap:

    round - krajevi linija su zaobljeni,
    square - krajevi linije se kvadriraju dodavanjem okvira jednake širine linije i polovine debljine linije,
    butt - krajevi linije su kvadratni na krajnjim tačkama.
Podrazumevana vrednost je butt. Svojstvu je neophodno dodeliti vrednost pre poziva metode stroke().

<canvas id="myCanvas" width="578" height="200"></canvas> <script> var canvas = document.getElementById('myCanvas'); var context = canvas.getContext('2d'); context.beginPath(); context.moveTo(200, canvas.height / 2 - 50); context.lineTo(canvas.width - 200, canvas.height / 2 - 50); context.lineWidth = 20; context.strokeStyle = '#0000ff'; context.lineCap = 'butt'; context.stroke(); context.beginPath(); context.moveTo(200, canvas.height / 2); context.lineTo(canvas.width - 200, canvas.height / 2); context.lineWidth = 20; context.strokeStyle = '#0000ff'; context.lineCap = 'round'; context.stroke(); context.beginPath(); context.moveTo(200, canvas.height / 2 + 50); context.lineTo(canvas.width - 200, canvas.height / 2 + 50); context.lineWidth = 20; context.strokeStyle = '#0000ff'; context.lineCap = 'square'; context.stroke(); </script>
