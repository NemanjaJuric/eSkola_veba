Debljina linije u HTML5 Canvas-u

Da bi se definisala debljina linije potrebno je dodeliti vrednost svojstvu lineWidth. Vrednost svojstva lineWidth mora biti dodeljena pre poziva metode stroke().

<canvas id="linija" width="578" height="200"></canvas> <script> var canvas = document.getElementById('myCanvas'); var context = canvas.getContext('2d'); context.beginPath(); context.moveTo(0, 0); context.lineTo(200, 100); context.lineWidth = 15; context.stroke(); </script>
