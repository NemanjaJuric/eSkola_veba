Canvas koordinatni sistem

U 2D prostoru pozicije se određuju upotrebom x i y koordinata. x osa se prostire horizontalno, a y vertikalno. Centar ima poziciju x = 0 i y = 0, može biti i predstavljeno kao (0, 0). Ova metoda pozicioniranja objekata se u matematici naziva Dekartov koordinantni sistem. Canvasov koordinatni sistem međutim postavlja početnu tačku u gornjem levom uglu canvas-a sa x koordinatom koja se prostire u desno i y koordinatom koja sa prostire prema dole do dna canvas-a. Za razliku od standardnog Dekartovog koordinatnog sistema, Canvas prostor nema vidljivih negativnih tačaka.

Crtanje u Canvas-u pomoću JavaScript jezika

Svaki vid crtanja u HTML canvas-u zahteva određeni skript jezik, najčešće JavaScript.

Korak 1: Pre svega, treba pronaći <canvas> element.
To činimo koristeći HTML DOM metod getElementById():
var canvas = document.getElementById("myCanvas");
Korak 2: Definisanje objekta za crtanje.
To činimo koristeći HTML metod getContext():
var ctx = canvas.getContext("2d");
Korak 3:Crtanje objekta
Postavljamo property fillStyle na crvenu boju. Podrazumevana vrednost za fillStyle je crna boja.
ctx.fillStyle="#FF0000";
Metod fillRect(x,y,width,height) crta pravougaonik čija je unutrašnjost crvene boje.
ctx.fillRect(0, 0, 150, 75); <canvas id="myCanvas" width="200" height="100" style="border:1px solid #c3c3c3;"></canvas> <script> var canvas = document.getElementById("myCanvas"); var ctx = canvas.getContext("2d"); ctx.fillStyle="#FF0000"; ctx.fillRect(0, 0, 150, 75); </script>

Crtanje linije

Za iscrtavanje prave linije u Canvas-u, koristimo sledeće metode:

  • moveTo(x,y) - definiše početnu tačku linije
  • lineTo(x,y) - definiše kranju tačku linije

Da bi se ustvari povukla crta između definisane početne i krajnje tačke neophodan je poziv metode: stroke().

Definišemo početnu tačku na poziciji (0,0), i krajnju tačku na poziciji (200,100). Zatim pozovemo stroke() metod za iscrtavanje linije.

<canvas id="linija" width="200" height="100" style="border:1px solid #c3c3c3;"></canvas> <script> var canvas = document.getElementById("linija"); var ctx = canvas.getContext("2d"); ctx.moveTo(0, 0); ctx.lineTo(200, 100); ctx.stroke(); </script>

Crtanje kruga

Za crtanje kruga u HTML canvas-u je neophodno sledeće:

  • poziv metode begintPath() kojom se započinje crtanje linije.
  • poziv metode arc(x,y,r,startangle,endangle) koja kreira luk ili krivu. Za kreiranje kruga je potrebno setovati početni ugao: 0 i krajnji ugao: 2*Math.PI. Parametri x i y definišu x- i y-koordinate centra. Parametar r definiše poluprečnik kruga.
  • poziv metode stroke() kojom se iscrtava definisani krug.
<canvas id="krug" width="200" height="100" style="border:1px solid #c3c3c3;"></canvas> <script> var canvas = document.getElementById("krug"); var ctx = canvas.getContext("2d"); ctx.beginPath(); ctx.arc(95, 50, 40, 0, 2 * Math.PI); ctx.stroke(); </script>
