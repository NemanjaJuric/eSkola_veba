Transformacije u Canvas-u. Translacija

Translacija HTML5 Canvas contexta se postiže pozivom metode translate(). Translacijom postižemo kretanje unutrašnjih elemenata canvas-a pozivom samo jedne metode što je jako zgodno u slučaju složenih objekata gde bi bilo neophodno menjati vrednosti x i y-koordinate za svaku tačku posebnu. Na ovaj način vršimo transformaciju context objekata, potom iscrtavanje grafičkih objekata.

U sledećem primeru transliramo context za polovinu dužine i polovinu visine canvas elementa, zatim crtamo pravougaonik dužine 150 i visine 75.

<canvas id="myCanvas" width="578" height="200"></canvas> <script> var canvas = document.getElementById('myCanvas'); var context = canvas.getContext('2d'); var rectWidth = 150; var rectHeight = 75; context.translate(canvas.width / 2, canvas.height / 2); context.fillStyle = 'blue'; context.fillRect(rectWidth / -2, rectHeight / -2, rectWidth, rectHeight); </script>
