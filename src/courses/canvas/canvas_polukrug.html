Polukrug u HTML5 Canvas-u

Crtanje polukruga pomoću elementa HTML5 Canvas se postiže pozivom metode arc() postavljanjem vrednosti (početni ugao + PI) za krajnji ugao.

<canvas id="myCanvas" width="578" height="200"></canvas> <script> var canvas = document.getElementById('myCanvas'); var context = canvas.getContext('2d'); context.beginPath(); context.arc(120, 40, 70, 0, Math.PI, false); context.closePath(); context.lineWidth = 5; context.fillStyle = 'red'; context.fill(); context.strokeStyle = '#550000'; context.stroke(); </script>
