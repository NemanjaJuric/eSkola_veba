Shadow u Canvas-u

Dodavanje senke u HTML5 Canvas-u se postiže setovanjem sledećih context svojstava:

  • shadowColor
  • shadowBlur
  • shadowOffsetX
  • shadowOffsetY
Primer:
<canvas id="myCanvas" width="578" height="200"></canvas> <script> var canvas = document.getElementById('myCanvas'); var context = canvas.getContext('2d'); context.rect(188, 40, 200, 100); context.fillStyle = 'red'; context.shadowColor = '#999'; context.shadowBlur = 20; context.shadowOffsetX = 15; context.shadowOffsetY = 15; context.fill(); </script>
