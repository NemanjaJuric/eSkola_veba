Transformacije u Canvas-u. Rotacija

Rotacija u HTML5 Canvas-u se postiže pozivom metode rotate() koja se zadaje uglom u radijanima. Da bi se definisala tačka rotacije mora se najpre translirati context objekat tako da se gornji levi ugao contexta dovede do željene tačke rotacije.

U sledećem primeru ćemo najpre translirati context objekat do centra pravougaonika, zatim zarotirati pravougaonik oko njegovog centra za ugao od PI/4.

<canvas id="myCanvas" width="578" height="200"></canvas> <script> var canvas = document.getElementById('myCanvas'); var context = canvas.getContext('2d'); var rectWidth = 150; var rectHeight = 75; context.translate(canvas.width / 2, canvas.height / 2); context.rotate(Math.PI / 4); context.fillStyle = 'blue'; context.fillRect(rectWidth / -2, rectHeight / -2, rectWidth, rectHeight); </script>
