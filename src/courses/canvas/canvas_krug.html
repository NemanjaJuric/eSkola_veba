Krug u HTML5 Canvas-u

Crtanje kruga pomoću elementa HTML5 Canvas se postiže pozivom metode arc() postavljanjem nule za početni ugao i 2 * PI za krajnji ugao.

<canvas id="myCanvas" width="578" height="200"></canvas> <script> var canvas = document.getElementById('myCanvas'); var context = canvas.getContext('2d'); var centerX = canvas.width / 2; var centerY = canvas.height / 2; var radius = 70; context.beginPath(); context.arc(centerX, centerY, radius, 0, 2 * Math.PI, false); context.fillStyle = 'green'; context.fill(); context.lineWidth = 5; context.strokeStyle = '#003300'; context.stroke(); </script>
