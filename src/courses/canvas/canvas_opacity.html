Opacity u HTLM5 Canvas-u

Opacity se u HTML5 Canvas-u podešava setovanjem context svojstva globalAlpha na realne vrednosti između 0 i 1, pri čemu 0 predstavlja potpunu providnost, a 1 potpunu neprovidnost.

Sledeći primer predstavlja providnost kruga u odnosu na pravougaonik za vrednost 0.5:

<canvas id="myCanvas" width="578" height="200"></canvas> <script> var canvas = document.getElementById('myCanvas'); var context = canvas.getContext('2d'); // crta plavi pravougaonik context.beginPath(); context.rect(200, 20, 100, 100); context.fillStyle = 'blue'; context.fill(); // crta transparenti crveni krug context.globalAlpha = 0.5; context.beginPath(); context.arc(320, 120, 60, 0, 2 * Math.PI, false); context.fillStyle = 'red'; context.fill(); </script>
